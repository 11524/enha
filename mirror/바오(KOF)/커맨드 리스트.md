  * 만사 : 공중에서 ↓ + B

  * 사이코 볼 어택 리플렉트 : ↓↙← + B
  * 사이코 볼 어택 에어 프론트 : 공중에서 ↓↙← + A
  * 사이코 볼 어택 에어 바운드 : 공중에서 ↓↘→ + B
  * 사이코 볼 크래쉬 라이즈 : ↓↘→ + C
  * 사이코 볼 크래쉬 바운드 : ↓↘→ + D

  * 사이코 볼 크래쉬 리플렉트 : ↓↙← + D

  * 사이코 볼 크래쉬 에어 바운드 : 공중에서 ↓↘→ + D  

★ 초필살기

  * 사이코 볼 크래쉬 SP : ↓↙←↓↙← + B or D
