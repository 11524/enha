## 1 개인리그 ¶

### 1.1 [2012 HOT6 GSL Season 1](2012%20HOT6%20GSL%20Season%201.md) ¶

Code S 32강에서 조2위로 16강 진출하나 [한이석](%ED%95%9C%EC%9D%B4%EC%84%9D.md),
[정종현](%EC%A0%95%EC%A2%85%ED%98%84.md)에게 패배하여 조4위. 그러나 Code A 24강에서 최연식에게
2:0으로 이겨 다시 복귀.

### 1.2 [2012 HOT6 GSL Season 2](2012%20HOT6%20GSL%20Season%202.md) ¶

Code S 32강전에서 조3위. 이번에는 Code A 32강에서 패배하여 승격강등전을 거쳤는데 **4전 전승**으로 Code S 다시
복귀.

### 1.3 [2012 무슈제이 GSL Season3](2012%20%EB%AC%B4%EC%8A%88%EC%A0%9C%EC%9D%B4%20GSL%20Season%203.md) ¶

Code S 32강전에서 또 조3위. 그리고 김상준, 장현우 선수를 거쳐서 또 Code S 복귀.  

### 1.4 [2012 HOT6 GSL Season 4](2012%20HOT6%20GSL%20Season%204.md) ¶

Code S에서 이번에는 8강 이상까지 올라가려나 했는데 16강전에서 다시 미끄러졌다. 그런데 Code A 24강에서
[변현제](%EB%B3%80%ED%98%84%EC%A0%9C.md)를 2:0으로 잡아 차기 시즌 Code S를 잡았다(...) 더구나,
[변현제](%EB%B3%80%ED%98%84%EC%A0%9C.md)와
[이영호](%EC%9D%B4%EC%98%81%ED%98%B8.md) 중 승자와 붙는 입장이었는데 자신은 둘 다 Code A로 보인다고
했으며 실제로 둘 다 Code S 승급 실패.  
본인도 인터뷰에서 코드 S에서 좋은 플레이를 하지 못한 아쉬움이 있어 제대로 된 성적을 내고 싶어하는 듯.

### 1.5 [2012 HOT6 GSL Season 5](2012%20HOT6%20GSL%20Season%205.md) ¶

[Code S](Code%20S.md) 32강에서 [조성주](%EC%A1%B0%EC%84%B1%EC%A3%BC.md)와
[송현덕](%EC%86%A1%ED%98%84%EB%8D%95.md)을 각각 2:0, 2:1로 잡으며 16강 진출. 그에게 진 2명은
모두 Code A로 내려갔으며, 특히 Code A 1R으로 내려간 조성주에게 [Code A에서 수련이 필요하다](http://www.this
isgame.com/board/view.php?viewtype=pc&category=13439&id=1333457)는 말을 전하며 판독기의
위엄을 보여주었다. <del>오오 판독기 오오</del>  
Code S 16강에서 [참치](%EA%B9%80%EB%8F%99%EC%9B%90.md)를 잡고 승자조에 올라가 <del>판독기를
걷어차고</del> 8강에 진출하는가 싶었으나 [국가대표 대장장이](%EC%9E%A5%ED%98%84%EC%9A%B0.md)에게
2:1로 석패하고 최종전에서 김동원이 2:0으로 역습하며 [고향](Code%20A.md)으로 돌아갔다. <del>김동원과 싸우는게
아니라 자신의 운명과 싸우는듯했다더라</del> 그런데 12월 14일 3라운드에서 [약쟁이 토스](%EC%A0%95%EB%AF%BC%EC%88%98%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md)를 꺾고 다시 코드 S 귀환.

  

여담으로 조지명식에서 [이승현](%EC%9D%B4%EC%8A%B9%ED%98%84.md)을 코드 A,
[김민철](%EA%B9%80%EB%AF%BC%EC%B2%A0.md)을 코드 S라고 말했는데, 16강 D조에서 귀신같이 이승현이 탈락해서
코드 A 24강행, 김민철이 8강에 진출했다. 이쯤되면 무섭다.

## 2 팀리그 ¶

### 2.1 [2012 GSTL Season 1](2012%20GSTL%20Season%201.md) ¶

3월 10일 B조 조별 리그 3경기 [Prime](Prime.md)의
[송병학](%EC%86%A1%EB%B3%91%ED%95%99.md),
[조성주](%EC%A1%B0%EC%84%B1%EC%A3%BC.md), [이정훈](%EC%9D%B4%EC%A0%95%ED%9B%88%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8.md) 선수들을 잡아내나 마지막의
[장현우](%EC%9E%A5%ED%98%84%EC%9A%B0.md) 선수에게 패배한다.

  

3월 24일 B조 조별 리그 5경기 [NS호서](NS%ED%98%B8%EC%84%9C.md)의
[김광섭](%EA%B9%80%EA%B4%91%EC%84%AD.md) 선수에게 승리하나 [정지훈](%EC%A0%95%EC%A7%80%ED%9B%88%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md) 선수에게 패배한다.

  

3월 31일 플레이오프 4강 1경기
[MVP](MVP%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9E%84%EB%8B%A8%29.md)의
[김승철](%EA%B9%80%EC%8A%B9%EC%B2%A0.md), [정민수](%EC%A0%95%EB%AF%BC%EC%88%98%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md) 선수들을 잡아내며 팀을 결승에 올려놓는데
성공한다.

  

4월 8일 결승전 [Prime](Prime.md)의 [이정훈](%EC%9D%B4%EC%A0%95%ED%9B%88%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md) 선수에게 패배한다.

### 2.2 [2012 HOT6 GSTL Season 2](2012%20HOT6%20GSTL%20Season%202.md) ¶

6월 16일 B조 조별 리그 1경기 [NS호서](NS%ED%98%B8%EC%84%9C.md)의
[박용환](%EB%B0%95%EC%9A%A9%ED%99%98.md) 선수에게 패배한다.

[남기웅](%EB%82%A8%EA%B8%B0%EC%9B%85.md) 선수에게 패배한다.

### 2.3 [2012 HOT6 GSTL Season 3](2012%20HOT6%20GSTL%20Season%203.md) ¶

10월 6일 MVP의 [황규석](%ED%99%A9%EA%B7%9C%EC%84%9D.md) 선수에게 승리하며 팀의 마무리에 성공한다.

  

11월 24일 [FXO.KR](FXO.KR.md)의 [고병재](%EA%B3%A0%EB%B3%91%EC%9E%AC.md),
[이대진](%EC%9D%B4%EB%8C%80%EC%A7%84.md),
[남기웅](%EB%82%A8%EA%B8%B0%EC%9B%85.md) 선수들을 잡아내나 [이인수](%EC%9D%B4%EC%9D%B8%EC%88%98%28%ED%94%84%EB%A1%9C%EA%B2%8C%EC%9D%B4%EB%A8%B8%29.md) 선수에게 패배한다.
