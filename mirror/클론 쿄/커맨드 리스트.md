## 1.1. [KOF 99](KOF%2099.md) ¶

  

★ 기본잡기  

  * 팔철 : 근접해서 ← or → + C
  * 일찰 업어치기 : 근접해서 ← or → + D  

★ 특수기  

  * 흑물기 : → + A
  * 수잔 : → + B
  * 섬돌뚫기 : ↘ + C  

★ 필살기  

  * 어둠쫓기 : ↓↘→ + A or C
  * 창귀 : ↓↙← + A or C
  * 칠십오식 개 : ↓↘→ + B or D, B or D
  * 농차 : →↓↘ + B or D  

★ 초필살기  

  * 대사치 : ↓↙←↙↓↘→ + A or C (모으기 가능)

  * 금월 양 : →↘↓↙← + B or D
  * R.E.D 킥 : ←↓↙ + B or D
