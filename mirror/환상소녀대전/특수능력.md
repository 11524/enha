# 6. 기타 ¶

  * 합체 (合体)  
서로 인접해 있을 때 '합체' 커맨드를 사용할 수 있다.  
사용자 : [서니 밀크](%EC%84%9C%EB%8B%88%20%EB%B0%80%ED%81%AC.md), [루나차일드](%EB%A3%A8%EB%82%98%20%EC%B0%A8%EC%9D%BC%EB%93%9C.md), [스타사파이어](%EC%8A%A4%ED%83%80%20%EC%82%AC%ED%8C%8C%EC%9D%B4%EC%96%B4.md), [루나사 프리즘리버](%EB%A3%A8%EB%82%98%EC%82%AC%20%ED%94%84%EB%A6%AC%EC%A6%98%EB%A6%AC%EB%B2%84.md), [메를랑 프리즘리버](%EB%A9%94%EB%A5%BC%EB%9E%91%20%ED%94%84%EB%A6%AC%EC%A6%98%EB%A6%AC%EB%B2%84.md), [리리카 프리즘리버](%EB%A6%AC%EB%A6%AC%EC%B9%B4%20%ED%94%84%EB%A6%AC%EC%A6%98%EB%A6%AC%EB%B2%84.md)  

  * 카제하후리 (風祝)  
특정한 행동에 의해 신앙 게이지가 상승한다. 게이지에 따라 대미지, 명중률 등이 오르고 필살기를 사용할 수 있게 되나 소비 MP도 증가한다.  
사용자 : [코치야사나에](%EC%BD%94%EC%B9%98%EC%95%BC%20%EC%82%AC%EB%82%98%EC%97%90.md)  

  * 싱크로나이즈  
기력이 120 이상일 때, 루나사의 각 스테이터스가 세명 중 가장 높은 캐릭터의 수치가 된다.  
사용자 : [프리즘리버 삼자매](%ED%94%84%EB%A6%AC%EC%A6%98%EB%A6%AC%EB%B2%84%20%EC%86%8C%EB%A0%B9%EC%95%85%EB%8B%A8.md)  

  * 기함 (旗艦)  
자신의 마스로 이동한유닛을 탑재할 수 있다. 또한, 인접한 유닛을 회수할 수 있다.  
사용자 : 성련선  

  * 인형생성(호라이) (人形生成 (蓬萊))  
고유 커맨드 「인형생성(호라이)」를 추가. MP를 25 소비해 유닛「호라이 인형」을 생성한다.`[6]`  
사용자 : 앨리스 마가트로이드  

  * 인형생성(샹하이) (人形生成 (上海))  
고유 커맨드 「인형생성(샹하이)」를 추가. MP를 40 소비해 유닛「샹하이 인형」을 생성한다.  
사용자 : 앨리스 마가트로이드  

  * 흡혈귀  
야간 스테이지인 경우, HP, MP가 턴당 10%씩 회복. 단, 주간 스테이지에선 지형적응이 극단적으로 내려간다.  
사용자 : [레밀리아 스칼렛](%EB%A0%88%EB%B0%80%EB%A6%AC%EC%95%84%20%EC%8A%A4%EC%B9%BC%EB%A0%9B.md), [플랑드르스칼렛](%ED%94%8C%EB%9E%91%EB%93%9C%EB%A5%B4%20%EC%8A%A4%EC%B9%BC%EB%A0%9B.md)  

  * 특수내성  
파워가 100 이하로 내려가지 않는다. 또한, 위압을 포함한 몇몇 정신 커맨드와 고정스킬 등을 무효화  
사용자 : [와타츠키노 요리히메](%EC%99%80%ED%83%80%EC%B8%A0%ED%82%A4%EB%85%B8%20%EC%9A%94%EB%A6%AC%ED%9E%88%EB%A9%94.md), [레이우지 우츠호](%EB%A0%88%EC%9D%B4%EC%9A%B0%EC%A7%80%20%EC%9A%B0%EC%B8%A0%ED%98%B8.md), [후지와라노 모코우](%ED%9B%84%EC%A7%80%EC%99%80%EB%9D%BC%EB%85%B8%20%EB%AA%A8%EC%BD%94%EC%9A%B0.md), [야고코로 에이린](%EC%95%BC%EA%B3%A0%EC%BD%94%EB%A1%9C%20%EC%97%90%EC%9D%B4%EB%A6%B0.md), [호라이산카구야](%ED%98%B8%EB%9D%BC%EC%9D%B4%EC%82%B0%20%EC%B9%B4%EA%B5%AC%EC%95%BC.md)  

  * 에인션트 문  
파워가 130 이상일 때, 공격력+100, 운동성+10  
사용자 : [미마](%EB%AF%B8%EB%A7%88.md)  

  * 오러리스 선  
파워가 130 이상일 때, 공격력+100, 운동성+10  
사용자 : [키리사메마리사](%ED%82%A4%EB%A6%AC%EC%82%AC%EB%A9%94%20%EB%A7%88%EB%A6%AC%EC%82%AC.md)  

  * 정체불명  
장갑, 운동성 등의 수치를 볼 수 없다.  
사용자 : [호쥬 누에](%ED%98%B8%EC%A5%AC%20%EB%88%84%EC%97%90.md)  

  * 가희  
파워의 상승에 따라 노래의 효과가 올라간다  
사용자 : [미스티아 로렐라이](%EB%AF%B8%EC%8A%A4%ED%8B%B0%EC%95%84%20%EB%A1%9C%EB%A0%90%EB%9D%BC%EC%9D%B4.md)  

  * 백택화  
파워가 130 이상일 때, 고유커맨드 백택화를 사용하여 케이네(백택)으로 변신하는 것이 가능해진다.(스테이지 클리어가 되기 전에 변신을
해제할 수 없다.)  
사용자 : [카미시라사와 케이네](%EC%B9%B4%EB%AF%B8%EC%8B%9C%EB%9D%BC%EC%82%AC%EC%99%80%20%EC%BC%80%EC%9D%B4%EB%84%A4.md)  

  * 몽상천생  
파워가 150 이상일 때, 고유커맨드 몽상천생을 추가. 3턴간 특수능력 투명한 무녀가 추가되며, 무기 몽상천생이 사용가능해진다.  
사용자 : [하쿠레이레이무](%ED%95%98%EC%BF%A0%EB%A0%88%EC%9D%B4%20%EB%A0%88%EC%9D%B4%EB%AC%B4.md)  

  * 투명한 무녀  
파워가 150 이상일 때, 25%의 확률로 공격을 회피한다. 또한, 그레이즈 대미지를 무효화한다.  
사용자 : [하쿠레이레이무](%ED%95%98%EC%BF%A0%EB%A0%88%EC%9D%B4%20%EB%A0%88%EC%9D%B4%EB%AC%B4.md)
