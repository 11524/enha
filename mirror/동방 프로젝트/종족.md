## 1. [신](%EC%8B%A0.md) ¶

  * [용신](%EC%9A%A9%28%EB%8F%99%EB%B0%A9%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%29.md) ☆ : [환상향](%ED%99%98%EC%83%81%ED%96%A5.md)의 최고신.
  * [아마츠카미](%EC%95%84%EB%A7%88%EC%B8%A0%EC%B9%B4%EB%AF%B8.md)(천진신) : 천상, [타카마가하라](%ED%83%80%EC%B9%B4%EB%A7%88%EA%B0%80%ED%95%98%EB%9D%BC.md)에서 강림한 신.
  * [쿠니츠카미](%EC%BF%A0%EB%8B%88%EC%B8%A0%EC%B9%B4%EB%AF%B8.md)(토착신) : 지상에 나타난 신.  

  * 마계신 : [마계의 창조주.](%EC%8B%A0%ED%82%A4.md)
  * 현인신 : 인간이자 신인 존재. 정확히 말하면, 인간이지만 신앙을 얻어 신격화 한것이라고 할수 있겠다. ☆ `[1]` `[2]`

## 2. 선신/초인 ¶

  * [천인](%EC%B2%9C%EC%9D%B8.md) : [천계](%EC%B2%9C%EA%B3%84.md)의 주민. 신체적으로 매우 튼튼하고 요괴에게 매우 강하다. 
  * [선인](%EC%84%A0%EC%9D%B8.md) : 천인이 되기 위해 수련을 쌓는 자. 요괴에겐 천하일미.  

    * [시해선](%EC%8B%9C%ED%95%B4%EC%84%A0.md) : 죽은 뒤 주검을 시해하여 선인이 된 자.
    * [성인](%EC%84%B1%EC%9D%B8.md) : 선인중에서도 특히 덕과 지혜가 높은 자.
  * [월인](%EC%9B%94%EC%9D%B8.md) : [달의 도시](%EB%8B%AC%EC%9D%98%20%EB%8F%84%EC%8B%9C.md)의 주민. 매우 강하며 기술의 수준도 매우 진보했다.
  * 봉래인 : 봉래의 약을 먹어 불로불사의 몸이 된 존재.
  * 십인의 염마왕 ☆ : 염라대왕.  

    * [염마](%EC%97%BC%EB%A7%88.md) : 염라대왕을 도와 자기 지역 사망자의 판결을 내리는 [지장보살](%EC%A7%80%EC%9E%A5%EB%B3%B4%EC%82%B4.md)들.   

      * [사신](%EC%82%AC%EC%8B%A0.md) : 지옥의 주민. 염마를 보조하며 시비곡직청을 운영하는 직원들.

## 3. 인간/반인 ¶

  * [인간](%EC%9D%B8%EA%B0%84.md) : 많은 수가 [인간 마을](%EC%9D%B8%EA%B0%84%20%EB%A7%88%EC%9D%84.md)에 살고 있다. 예외적으로 [하쿠레이 레이무](%ED%95%98%EC%BF%A0%EB%A0%88%EC%9D%B4%20%EB%A0%88%EC%9D%B4%EB%AC%B4.md),[키리사메 마리사](%ED%82%A4%EB%A6%AC%EC%82%AC%EB%A9%94%20%EB%A7%88%EB%A6%AC%EC%82%AC.md),[이자요이 사쿠야](%EC%9D%B4%EC%9E%90%EC%9A%94%EC%9D%B4%20%EC%82%AC%EC%BF%A0%EC%95%BC.md),[코치야 사나에](%EC%BD%94%EC%B9%98%EC%95%BC%20%EC%82%AC%EB%82%98%EC%97%90.md)는 다른곳에서 산다.  

    * [외래인](%EC%99%B8%EB%9E%98%EC%9D%B8.md) : 바깥 세계에서 넘어 온 인간. 대부분은 **[요괴의 한끼 식사](%EC%82%AC%EB%A7%9D%EC%A0%84%EB%8C%80.md)**가 되지만 운이 좋으면 하쿠레이 신사를 통해 돌아가기도 하고 인간 마을에 정착해 살기도 한다.  

  * [반요](%EB%B0%98%EC%9A%94.md) : 반인반요, 즉 인간과 요괴의 혼혈. 대부분 장수한다.
  * [수인](%EC%88%98%EC%9D%B8%28%EB%8F%99%EB%B0%A9%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%29.md) : 특정한 조건 하에 짐승화하는 인간. 대부분 인간의 편이다.
  * 반인반령 : 인간과 유령의 혼혈. 인간의 몸과 유령의 몸이 따로 있다. 장수한다.

## 4. 요정 ¶

  * [요정](%EC%9A%94%EC%A0%95%28%EB%8F%99%EB%B0%A9%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%29.md) : 자연 현상의 구현. 죽거나 다쳐도 곧 재생한다. 인외의 존재를 전부 요괴로 볼 경우 요괴로 구분되지만, 워낙 수가 많고 요괴와는 구분되는 특성 때문에 요괴 외의 존재로 볼 수도 있다.

## 5. 영체 ¶

  * [망령](%EB%A7%9D%EB%A0%B9.md) : 성불하지 못한 유령. 전생의 모습 그대로다. 무해할 것 같지만 오래 같이있으면 죽는다.
  * [신령](%EC%8B%A0%EB%A0%B9.md) : 망령 중 신으로 모셔지는 망령.
  * [유령](%EC%9C%A0%EB%A0%B9.md) : 기질이 구현화 된 것. 차갑다.
  * 원령 : 삼도천을 건너지 못한 유령. 사신이 영혼을 가져가기 전에 요괴가 시체를 가져가면 원령이 된다. 뜨겁다. 매우 위험하다.
