# 3. 승강제 도입 완료 ¶

  

2012 시즌부터 도입이 시작된 승강제가 2014 시즌에 도입 완료되어 정상적으로 작동될 예정이다. 1,2부 참가팀 수 조정으로 인해
2012 시즌에는 강등만, 2013 시즌에는 2팀 강등에 0.5팀 승격만 가능했던데 비해 2014 시즌부터는 클래식 최하위팀 강등/챌린지
우승팀 승격/클래식 11위팀,[챌린지 1차 승강 플레이오프를 통과한팀](K%EB%A6%AC%EA%B7%B8%20%EC%B1%8C%EB%A6%B0%EC%A7%80#s-6.md)과 최종 승강 플레이오프를
치러 최소 1팀, 최대 2팀의 승강이 이루어지게 된다.
