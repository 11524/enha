  * [잡기](%EC%9E%A1%EA%B8%B0.md)

역풍： ↘ + C  
걷어차기： → + D  

  * [필살기](%ED%95%84%EC%82%B4%EA%B8%B0.md)

백귀살： ↓↙←→ + A or C  
월화참： 근접 →↓↘ + B or D  
앵화참： ↓↙← + A or C  
아신돌： ↓↙← + B or D

리오광： ↓↘→↓↘→ + A or C

┗일섬： BCD

# 10. 네오지오 배틀 콜로세움 ¶

  * 잡기   
찔러 날리기 ~ 비연인 : 접근해서 레버 중립 or → or ← + CD  

  * 특수기   
역풍 : ↘ + C  
옷자락 쓸기 : → + D  

  * 필살기   
앵화참 : ↓↙← + A or C  
동패 광익인 : →↓↘ + A or C  
삼연살 : ↓↘→ + A or C (같은 버튼으로만 3회 연속 입력, 헛쳤을 시엔 추가 입력 불가)  
월화참 : 접근해서 →↓↘ + B  
아신돌 : →↘↓↙← + B (모으기 가능)  
백귀살 : →↘↓↙←→ + A or C  

  * 초필살기   
리 오광 : ↓↘→↓↘→ + A(기 1줄 사용)  
오광참 : ↓↘→↓↘→ + C(기 1줄 사용)  
앵화참·창포 : ↓↙←↓↙← + A or C (기 1줄 사용)  
분노 폭발 : ABC (기 3줄 사용)  

  * 어나더 더블 어설트  
일섬 : 파트너가 하오마루 일 때 분노폭발 사용후 →←→ + CE
