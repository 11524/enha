### 2. 가상현실 체험 ¶

등장인물들과 **넷무비를 보는 사람**이 가상으로 노는 내용. 넷무비 답지 않게 왠지 훈훈하다. 단, **마키의 인형 편 제외.**

  

  * File 프테라 - 에이지(형/오빠)와 놀자  
[히노 에이지](%ED%9E%88%EB%85%B8%20%EC%97%90%EC%9D%B4%EC%A7%80.md)와 같이 여러 가지 게임을
하는 내용. 첫번째 게임은 속담을 말하고 그 속담의 그림을 찾는 게임인데 에이지가 찾을 때마다 빼앗겨서 마지막엔 샤우타로 변신해 채찍으로
카드를 쳐낸다. 두 번째 게임은 가위바위보해서 이기면 때리고 지면 방어 하는 게임인데 가위바위보에서 진다음 맞을때 사고조로 변신, 마지막으로
무궁화 꽃이 피었습니다. 에서 뒤돌아보는 순간 가타키리바로 변신해 당황케 했다.  

  * File 고릴라 - 히나와 가상 동물원  
[이즈미 히나](%EC%9D%B4%EC%A6%88%EB%AF%B8%20%ED%9E%88%EB%82%98.md)와 함께 동물원에 가는

  * File 우나기 - 고토짱과 가상 데이트.

마지막에는 버스로 변신한 고토가 커터 윙으로 하늘을 나는 체험을 시켜준다.  

  * File 트리케라 - 마키와 가상 키요짱

차린 앙크지만 오히려 더 이상해져서 전에 했던 3가지 코스프레를 한번에 섞어 하는 기행을 저질렀다.  

  * File 카마키리 - 버스 다테! 오늘부터 내가 점장이다!  
치요코가 여행을 가 가게를 비우자, 다테가 점장을 한다는 내용.하지만 어째 아무도 오지 않았고 이에 다테가 버스로 변신해서 셀 메달을 넣고
어묵을 소환하는 묘기를 펼친다.<del>(오뎅암)</del>여담이지만 앙크는 어묵 대신 달걀을 집었다가 기절했다.  

  * File 라이온 - 카자리의 [이런 알바는 싫어?](%EC%9D%B4%EB%9F%B0%20%EC%98%81%EC%9B%85%EC%9D%80%20%EC%8B%AB%EC%96%B4.md)

  * File 콘도르 - 타쟈도르 웨이터의 애환  
그리드 **4인 가족**이 쿠스쿠시에를 찾아왔다. 4명이 앙크를 디스하자, 앙크가 에이지를 [타쟈도르콤보](%ED%83%80%EC%9F%88%EB%8F%84%EB%A5%B4%20%EC%BD%A4%EB%B3%B4.md)로 변신시켜 대접을
시킨다. 심지어 4명이 타쟈도르 콤보마저 디스하자, 앙크가 폭발해서 물을 뿌리고, 타쟈도르 콤보가 4명에게 다굴당한다.
