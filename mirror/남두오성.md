## 1. 소개 ¶

[남두육성](%EB%82%A8%EB%91%90%EC%9C%A1%EC%84%B1%EA%B6%8C.md)과는 헷갈리면 안된다.

  

[1992년](1992%EB%85%84.md) [롯데자이언츠](%EB%A1%AF%EB%8D%B0%20%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0.md)의 3할 타자
[전준호](%EC%A0%84%EC%A4%80%ED%98%B8.md),
[이종운](%EC%9D%B4%EC%A2%85%EC%9A%B4.md),
[박정태](%EB%B0%95%EC%A0%95%ED%83%9C.md),
[김민호](%EA%B9%80%EB%AF%BC%ED%98%B8%281961%29.md),
[김응국](%EA%B9%80%EC%9D%91%EA%B5%AD.md)을 합쳐 부른 별명이다. 이들과 나머지 선수들이 합쳐서 세운 팀타율
.289의 기록은 1987년 삼성(팀타율 .300), 1999년 롯데(팀타율 .291) 그리고 전무후무한
<del>타신투병</del>타고투저로 기록될 2014년의 삼성(팀타율 .301), 넥센(팀타율 .298), 두산(팀타율 .293),
SK(팀타율 .291)를 제외하고 아무도 넘지 못한 무서운 장벽. 만약 1991년 .347의 타율을 세운
[장효조](%EC%9E%A5%ED%9A%A8%EC%A1%B0.md)의 전성기가 1년만 더 길었더라면... <del>말 그대로
남두육성</del> 생각만 해도 무섭기 짝이 없다.`[1]`

  

[박정태](%EB%B0%95%EC%A0%95%ED%83%9C.md) 1명 빼고 전부 좌타자에 홈런타자가 아닌 교타자라는 점에서 NPB
[요코하마 베이스타즈](%EC%9A%94%EC%BD%94%ED%95%98%EB%A7%88%20%EB%B2%A0%EC%9D%B4%EC%8A%A4%ED%83%80%EC%A6%88.md)의 머신건 타선이나 MLB [신시내티레즈](%EC%8B%A0%EC%8B%9C%EB%82%B4%ED%8B%B0%20%EB%A0%88%EC%A6%88.md)의 빅 레드 머신에
비견할만한 라인업.

## 2. 선수 ¶

**[전준호](%EC%A0%84%EC%A4%80%ED%98%B8.md)**  

120경기 출장 / 타율 .300 / 140안타(3위) / 5홈런 / 49타점 / 90득점(5위) / 33도루(3위)

리드오프였던 전준호는 3할 타율과 33도루를 기록하며 1번 타자로서의 역할을 톡톡히 해냈다. 특히나 1991년에 보여줬던 가능성을 확실히
터트린 시즌이었다. 단순 방망이로만 따지면 롯데 시절 중 가장 좋은 성적을 기록한 해였다.

  

**[이종운](%EC%9D%B4%EC%A2%85%EC%9A%B4.md)**  

108경기 출장 / 타율 .314 / 126안타 / 3홈런 / 57타점 / 66득점 / 21도루

이종운은 .314의 타율과 **14개의 3루타**`[2]`으로 '쳤다 하면 3루타' 라는 별명까지 만들어낸다. 발도 상당히 빨랐고 작전 수행
능력도 최고 수준이었다.

  

**[박정태](%EB%B0%95%EC%A0%95%ED%83%9C.md)**  

124경기 출장 / 타율 .335(2위) / 132안타(4위) / 14홈런 / 79타점(5위) / 91득점(4위) / 7도루

탱크 박정태는 .335에 14홈런, 그리고 **43개의 2루타**를 쳐내며`[3]` 소총군단이란 비웃음을 뭉개버린다.`[4]`

  

**[김민호](%EA%B9%80%EB%AF%BC%ED%98%B8%281961%29.md)**  

118경기 출장 / 타율 .322(4위) / 133안타 / 16홈런 / 88타점(4위) / 69득점 / 5도루

자갈치 [김민호](%EA%B9%80%EB%AF%BC%ED%98%B8%281961%29.md)는 타율 .322 16홈런, 28개의
2루타(2위)에 88타점을 기록하며 4번 타자로서의 역할을 완수.  
특히나 결정적인 상황에서 한방을 터트리는 능력이 굉장했다. 1992년 [해태타이거즈](%ED%95%B4%ED%83%9C%20%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88.md)와의 플레이오프
5차전에서 1회초 상대선발 [문희수](%EB%AC%B8%ED%9D%AC%EC%88%98.md)로부터 기록한 3점짜리 결승 선제홈런,
[빙그레 이글스](%ED%95%9C%ED%99%94%20%EC%9D%B4%EA%B8%80%EC%8A%A4.md)와의 한국시리즈 1차전
1회에 기선을 제압했던 선제 2타점 적시타등 을 보면 그의 클러치 능력을 알 수 있었다.

  

**[김응국](%EA%B9%80%EC%9D%91%EA%B5%AD.md)**  

119경기 출장 / 타율 .319 / 128안타 / 10홈런 / 79타점 / 76득점 / 29도루(4위)

호랑나비 김응국은 뛰어난 타격능력을 바탕으로 2년연속 3할 타율과 프로 첫 두 자리 수 홈런`[5]`, 12개의 3루타(2위)를 기록하고
29도루를 기록한 빠른 발까지. 수비능력도 굉장히 안정적이었다.

  

아직도 많은 [부산 갈매기](%EB%B6%80%EC%82%B0%20%EA%B0%88%EB%A7%A4%EA%B8%B0.md)들은
남두오성에 하악거리고 있다. 2010시즌에서 제2의 불꽃타선(팀타율 0.288 팀 홈런 185개)을 볼 수 있었나
싶었지만....**불(火)펜**때문에 [망했어요](%EB%A7%9D%ED%96%88%EC%96%B4%EC%9A%94.md). 반대로
2012시즌에서는 불펜이 진짜 불펜이 됐는데 빠따가 남두오성 이전의 소총부대로 돌아왔다...<del>얻는게 있으면 잃는 것도 있는법.

## 3. 은퇴 이후 ¶

하지만 의외로 선수들의 끝이 안 좋았다. [이종운](%EC%9D%B4%EC%A2%85%EC%9A%B4.md)은
[1997년](1997%EB%85%84.md) 시즌을 마치고 [한화이글스](%ED%95%9C%ED%99%94%20%EC%9D%B4%EA%B8%80%EC%8A%A4.md)로
[트레이드](%ED%8A%B8%EB%A0%88%EC%9D%B4%EB%93%9C.md)되어 쓸쓸히 선수 생활을 마감하였다.
[김민호](%EA%B9%80%EB%AF%BC%ED%98%B8%281961%29.md)도 [1995년한국시리즈](1995%EB%85%84%20%ED%95%9C%EA%B5%AD%EC%8B%9C%EB%A6%AC%EC%A6%88.md)
준우승 이후 이듬해 팀에 의해 정리당하고 은퇴, [박정태](%EB%B0%95%EC%A0%95%ED%83%9C.md)는 이후 [롯데자이언츠](%EB%A1%AF%EB%8D%B0%20%EC%9E%90%EC%9D%B4%EC%96%B8%EC%B8%A0.md)가 대놓고
은퇴를 시키려고 어처구니 없는 연봉 협상을 했고, 결국 은퇴했다. [전준호](%EC%A0%84%EC%A4%80%ED%98%B8.md)는
[현대 유니콘스](%ED%98%84%EB%8C%80%20%EC%9C%A0%EB%8B%88%EC%BD%98%EC%8A%A4.md)로
이적해서 현대왕조의 일원이 되었으나, 연봉협상 잡음 끝에 팀을 나가서 롯데에서의 끝은 안 좋다.

  

남두오성 중 [전준호](%EC%A0%84%EC%A4%80%ED%98%B8.md)는 [SK와이번스](SK%20%EC%99%80%EC%9D%B4%EB%B2%88%EC%8A%A4.md)의 주루코치를 맡다가
[미국](%EB%AF%B8%EA%B5%AD.md)에서 연수를 받은 후 [NC다이노스](NC%20%EB%8B%A4%EC%9D%B4%EB%85%B8%EC%8A%A4.md)의 주루코치로 자리를 옮겼다.
[김민호](%EA%B9%80%EB%AF%BC%ED%98%B8%281961%29.md)는 부산고 야구부 감독을 거쳐 롯데의 2군
타격코치를 맡고 있다가 2015년부터 1군 수석코치 자리를 맡게 되었고,
[박정태](%EB%B0%95%EC%A0%95%ED%83%9C.md)는 2012년까지 롯데의 1군 타격코치를 하다가 사퇴하고 KBO
육성위원으로 재직 중이다. [김응국](%EA%B9%80%EC%9D%91%EA%B5%AD.md)은 롯데의 1군 외야코치로 재직 중이며,
이들 중 가장 존재감이 없었던 [이종운](%EC%9D%B4%EC%A2%85%EC%9A%B4.md)은 경남고 야구부 감독을 맡고 있다가
롯데에 복귀해 코치를 맡은 후 2015년부터 롯데의
<del>[델동님](%EA%B0%90%EB%8F%99%EB%8B%98.md)</del>감독이 되었다.
