★ 기본잡기  

  * 발목용: 근접해서 CD  

★ 특수기  

  * 용살각: → + B  

  * 컴비네이션 A: 근접 A ▶ B  

  * 컴비네이션 B: 근접 A ▶ C  

  * 컴비네이션 C: 근접 B ▶ C  

  * 컴비네이션 D: 근접 C ▶ C  

★ 필살기  

  * 제왕천안권: ↓↘→ + A or C  

  * 제왕천이권: →↓↘ + A or C  

  * 제왕신족권: →→ + A or C  

  * 진 제왕신족권: 스텝 시작 직후 → + A or C  

  * 제왕루진권: ↓↙←→ + A or C  

  * 제왕타심권: ↓↙← + A or C  

  * 용전신(전방): ↓↘→ + B or D  

  * 용전신(후방): ↓↙← + B or D  

★ 초필살기  

  * 제왕숙명권: ↓↘→↓↘→ + A or C(버튼 지속으로 탄이 전진)  

  * 제왕용섬권: ↓↙←↓↙← + A or C  

  * 제왕역린권: ↓↙←↙↓↘→ + A or C
