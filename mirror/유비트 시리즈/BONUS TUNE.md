## 2. 보너스 튠 포인트 ¶

보너스 튠 포인트는 후술할 일정 조건을 만족할 때마다 튠마다 포인트가 쌓이고, 파이널 튠까지 1바퀴(1000pt) 이상이 채워진 경우
1000 포인트를 자동으로 소모하여 보너스 튠에 진입한다. 기본적으로 노멀 모드일 때보다 노멀 모어 모드일 때 더 많이 쌓인다. 쌓인
포인트는 12번 패널 위치에`[1]` 원형으로 지금까지 쌓인 총 포인트가, 가운데에 이번 튠에 쌓인 포인트가 표시된다. 로컬 매칭 중이거나
70만점을 넘지 못해 폭사한 경우 보너스튠으로 이동되지 않는다. 보너스 튠 포인트는 최대 1500 포인트까지 저축 할 수 있으며 보존 기한이
있어 최종 플레이 일로부터 1주일간 플레이하지 않으면 남은 포인트가 소멸된다.

  

보너스 튠 진입에 필요한 포인트는 (노멀 모어 기준으로) 모으기 어렵지 않은데, 실제로 [유비트 소서 풀필](%EC%9C%A0%EB%B9%84%ED%8A%B8%20%EC%86%8C%EC%84%9C%20%ED%92%80%ED%95%84.md) 시절에는 코나미 사이트에서 2번째
풀콤보 챌린지를 설정하고 노멀모어 5튠에서 둘 다 성공시키면 성공 2회(300*2 = 600) + 5곡 플레이(80*5 = 400) =
1000으로 바로 1바퀴를 얻게 되었다. [유비트프롭](%EC%9C%A0%EB%B9%84%ED%8A%B8%20%ED%94%84%EB%A1%AD.md)에서는 매 1회 기본 설정되던 풀
콤보 챌린지가 오늘의 추천곡으로 바뀌면서, 코나미 사이트에서 받을 수 있는 변덕스러운 풀 콤보 챌린지로만 보너스 튠 포인트를 받을 수 있게
되어 조금 어려워졌다.

  

보너스 튠에 처음으로 진출한 경우 자동으로 보너스 튠 전용곡 폴더`[2]`가 선택된다.`[3]` 2번째부터는 일반 튠과 같은 사양.

  

[유비트 소서 풀필](%EC%9C%A0%EB%B9%84%ED%8A%B8%20%EC%86%8C%EC%84%9C%20%ED%92%80%ED%95%84.md)에서는 보너스 튠 전용곡을 플레이 할 경우 자동으로 솔로플레이가 되었으나, [유비트프롭](%EC%9C%A0%EB%B9%84%ED%8A%B8%20%ED%94%84%EB%A1%AD.md) 부터는 보너스 튠 전용곡이라도
매칭할 수 있도록 변경되었다. 보너스 튠 또한 해금 포인트 보너스 획득`[4]` 및 보너스 튠 포인트가 쌓인다.
