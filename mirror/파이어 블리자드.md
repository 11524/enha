## 1.1. 아토믹 플레어 ¶

アトミックフレア / Atomic Flare. [게임판](http://www.youtube.com/watch?v=NRtT_nkC5aY)  
위력 : 86 / 소비TP : 52 / 속성 : 火 / 특성 : 슛 블록 / 성장속도 : 빠름

  

2편부터 등장하는 슛 필살기. 홍염의 불꽃을 실어 날리는 강력한 필살슛. 주 사용자는 프로미넌스의 캡틴 [번](%EB%B2%88%28%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%29.md).
게임판에선 일부 프로미넌스 멤버들이 습득하기도 한다.

  

애니에서는 첫등장시 [엔도 마모루](%EC%97%94%EB%8F%84%20%EB%A7%88%EB%AA%A8%EB%A3%A8.md)의
[마신 더 핸드](%EB%A7%88%EC%8B%A0%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)를 가볍게
뚫어버리고, 이후 [더 카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)전에선 [타치무카이 유우키](%ED%83%80%EC%B9%98%EB%AC%B4%EC%B9%B4%EC%9D%B4%20%EC%9C%A0%EC%9A%B0%ED%82%A4.md)의 [무한 더 핸드](%EB%AC%B4%ED%95%9C%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)도
씹어버리는 위력을 보여준다. G2로 진화한 무한 더 핸드에 막힌 적도 있지만, 3기에선 번이 더 파워업했는지 G5까지 진화한 무한 더 핸드를
쉽게 뚫어버렸다.

## 1.2. 노던 임팩트 ¶

ノーザンインパクト / Northern Impact. [게임판](http://www.youtube.com/watch?v=ksGKC1MRqRM)  
위력 : 86 / 소비TP : 52 / 속성 : 風 / 특성 : 슛 체인 / 성장속도 : 빠름

  

2편부터 등장하는 슛 필살기. 북극의 냉기를 실어 날리는 강력한 필살 슛. 주 사용자는 다이아몬드 더스트의 캡틴 [가젤](%EA%B0%80%EC%A0%A4%28%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%29.md). 게임판에선 일부 다이아몬드 더스트 멤버들이 습득하기도 한다. 특이하게도 애니판에서 좌우 다른 방향으로 보여주는 기술이다.

  

애니에서는 첫등장시 [엔도 마모루](%EC%97%94%EB%8F%84%20%EB%A7%88%EB%AA%A8%EB%A3%A8.md)의
G2로 진화한 [정의의 철권](%EA%B0%93%20%ED%95%B8%EB%93%9C.md)를 2번이나 뚫어버리는 위력을 보여준다.
그리고 이후 [더 카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)전에선 [타치무카이 유우키](/
wiki/%ED%83%80%EC%B9%98%EB%AC%B4%EC%B9%B4%EC%9D%B4%20%EC%9C%A0%EC%9A%B0%ED%82%
A4)의 [무한 더 핸드](%EB%AC%B4%ED%95%9C%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)도
씹어버린다. 하지만 이후론 애니에서 등장이 없다.(...) 3기에서 가젤이 들어간 팀 이름이 _파이어_ 드래곤인지라 제작진 입장에선 얼음
계열 슛을 보여주기가 좀 뭐했던 걸지도.

## 1.3. 파이어 블리자드 ¶

ファイアブリザード / Fire Blizzard. [게임판](http://www.youtube.com/watch?v=Q6KT4n6vYe0)  
위력 : 113 / 소비TP : 64 / 속성 : 火 or 風 / 특성 : 없음 / 성장속도 : 보통

  

2편부터 등장하는 2인 합체 필살기. 사용자는 [번](%EB%B2%88.md)과
[가젤](%EA%B0%80%EC%A0%A4.md). 애니 2기에서는 [더카오스](%EB%8D%94%20%EC%B9%B4%EC%98%A4%EC%8A%A4.md)가 내부 분열로 자멸할 위기에 처하자,
솔선수범으로 팀을 하나로 단합시키기 위해 사용해 **[무한 더핸드](%EB%AC%B4%ED%95%9C%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)를 털어먹었다**. 3기에서
번과 가젤이 [파이어드래곤](%ED%8C%8C%EC%9D%B4%EC%96%B4%20%EB%93%9C%EB%9E%98%EA%B3%A4.md)에 들어갔을 때도
[이나즈마 재팬](%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9E%AC%ED%8C%AC.md)과 싸울
때 사용해 **[무한 더 핸드](%EB%AC%B4%ED%95%9C%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)를
털어먹었다**.`[1]`

  

[크로스파이어](%ED%81%AC%EB%A1%9C%EC%8A%A4%ED%8C%8C%EC%9D%B4%EC%96%B4.md)와 마찬가지로
火속성, 風속성의 두 가지 버전이 있는데, 火는 번이 風은 가젤이 사용한다. 물론 양쪽 다 자속 보정을 당연히 받아준다. 단 크로스 파이어와
마찬가지로 사용할 때 파트너는 반대 속성을 가지고 있어야 한다. 즉, 화속성 파이어 블리자드라면 파트너는 풍 속성, 반대로 풍 속성이라면
파트너는 화 속성.

  

연출 자체는 그냥 둘 다 위로 올라가서 같이 슛을 쏘는 썰렁하기 짝이 없는 연출이지만, 위력이 게임내에서 최상급 클래스고, 번&가젤 양쪽 다
희대의 개사기 스킬 [속성강화](%EC%86%8D%EC%84%B1%EA%B0%95%ED%99%94.md)가 있어서 어마어마한 위력을
자랑한다.

## 4.2. 아이스 블록 ¶

アイスブロック / Ice Block [영상](http://www.youtube.com/watch?v=aFOnnaif7t0&NR=1)  
위력 : 80 / 소모TP : 50 / 속성 : 風 / 특성 : 캐치 / 성장속도 : 보통

  

2편부터 등장. 주로 다이아몬드 더스트의 골키퍼 '벨가'가 사용. 냉기를 실은 오른손으로 펀치를 날려 그대로 공을 얼려버린다. 애니에선
[쓰나미](%EC%93%B0%EB%82%98%EB%AF%B8%20%EC%A3%A0%EC%8A%A4%EC%BC%80.md)의 [쓰나미부스트](%EC%93%B0%EB%82%98%EB%AF%B8%20%EB%B6%80%EC%8A%A4%ED%8A%B8.md)를 막아내지만,
이후 고엔지의 폭렬 스톰에 바로 털려버린다.(...) 그래도 '번 아웃'에 비하면 아주 쬐금 활약한 편이다.

  

게임 내에선 다이아몬드 더스트의 디펜더인 아이큐와 클라라가 자력으로 습득하기도 한다.
