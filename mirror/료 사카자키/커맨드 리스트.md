## 25.1. 노멀 료 ¶

★ 기본잡기  

  * 오금대 떨어뜨리기 : 근접해서 ← or → + C
  * 배대되치기 : 근접해서 ← or → + D  

★ 특수기  

  * 열진각 : ↓↓ + D
  * 열충권 : →→ + C  

★ 필살기  

  * 호황권 : ↓↘→ + A or C
  * 호포 : →↓↘ + A or C
  * 비연질풍각 : ↓↙← + B or D  
┗암잠각 : 강 비연질풍각 사용 중 ↓↙← + B or D

  * 잠열권 : →←→ + A or C
  * 맹호뇌신찰 : ↓↘→ + B or D
  * 호포질풍권 : ↓↙← + A or C
  * 맹호뇌신강 : ←↓↙ + A or C  
┗호포 : →↓↘ + A or C

  * 극한류연무권 : →↘↓↙←→ + B or D (잡기)  

★ 초필살기(☆은 기게이지 소모 개수 표시)  

  * 패왕상후권(☆) : →←↙↓↘→ + A
  * 용호난무(☆☆) : ↓↘→↘↓↙← + C
  * 극한호포(☆☆) : ↓↘→↓↘→ + B or D
  * 천지패황권(☆☆☆) : ↓↘→↓↘→ + C  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ A ▶ C ▶ → + C ▶ ← + C
  * 02 : → + B ▶ (반격 성공 후) B ▶ D
  * 03 : → + B ▶ ↘ + B (이후 반복)
  * 04 : ↘ + B ▶ (반격 성공 후) C ▶ C
  * 05 : ↘ + B ▶ → + B (이후 반복)
  * 06 : ← + B ▶ ← + B ▶ ← + B
  * 07 : ← + B ▶ ← + B ▶ → + C
  * 08 : ← + B ▶ D ▶ D ▶ ↓ + D
  * 09 : C ▶ C ▶ C
  * 10 : C ▶ C ▶ ← + C (페인트)
  * 11 : C ▶ → + C
  * 12 : → + C ▶ C ▶ → + C
  * 13 : ← + C ▶ C ▶ C ▶ C
  * 14 : ← + C ▶ C ▶ C ▶ → + C
  * 15 : →← + C ▶ C
  * 16 : → + D ▶ ↓ + B ▶ D
  * 17 : → + D ▶ ↓ + B ▶ ↓ + D
  * 18 : → + D ▶ ↓ + D
  * 19 : → + D ▶ ↙ + D (페인트)
