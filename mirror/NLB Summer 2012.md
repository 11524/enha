## 1. 개요 ¶

  * 주최 : [라이엇게임즈](%EB%9D%BC%EC%9D%B4%EC%97%87%EA%B2%8C%EC%9E%84%EC%A6%88.md)
  * 주관 : [온게임넷](%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7.md), [나이스게임TV](%EB%82%98%EC%9D%B4%EC%8A%A4%EA%B2%8C%EC%9E%84TV.md)
  * 스폰서 : [라이엇게임즈](%EB%9D%BC%EC%9D%B4%EC%97%87%EA%B2%8C%EC%9E%84%EC%A6%88.md)
  * 중계진  

    * 캐스터 : [정진호](%EC%A0%95%EC%A7%84%ED%98%B8.md), [김의중](%EA%B9%80%EC%9D%98%EC%A4%91.md)
    * 해설 : [김태우](%EA%B9%80%ED%83%9C%EC%9A%B0%28%ED%95%B4%EC%84%A4%EC%9E%90%29.md), [김동수](%EA%B9%80%EB%8F%99%EC%88%98.md), [하광석](%ED%95%98%EA%B4%91%EC%84%9D.md), <del>[이형주](%EC%9D%B4%ED%98%95%EC%A3%BC.md)</del>`[1]`
  * 대회 상금  

    * **총상금** : 10,000,000 원  

      * 1위 : 6,000,000 원
      * 2위 : 3,000,000 원
      * 3위 : 1,000,000 원
  * 대회 일정  

    * 전체 일정 : 2012년 6월 26일(화) ~ 8월 31일(목)
    * 세부 일정 :  

      * **16강** : 6월 26일(화) ~ 7월 5일(목)
      * **8강** : 7월 10일(화) ~ 8월 9일(목)
      * **4강** : 8월 14일(화) ~ 8월 28일(화)
      * **결승전** : 8월 30일(목)
  * 진행 방식  

    * **16강** : 조별 듀얼토너먼트  
(4개조 단판제, 조별 1, 2위 8강 진출)  

      * 1경기 - A팀 vs B팀
      * 2경기 - C팀 vs D팀
      * 승자전 - 1경기승리팀 vs 2경기승리팀 - 승자는 8강진출
      * 패자전 - 1경기패배팀 vs 2경기패배팀 - 패자는 탈락
      * 최종전 - 승자전패배팀 vs 패자전승리팀 - 승자는 8강진출
    * **8강** : 조별 듀얼토너먼트  
(2개조 5전 3선승제, 조별 1, 2위 4강 진출)

    * **4강** : 8강의 조별 1,2위의 크로스 매치. 듀얼토너먼트, 5전 3선승제
    * **결승** : 7전 4선승제 진행  
(승자전에서 올라온 팀은 결승전 1승을 얻고 시작한다)

    * 1경기, 2경기, 승자전, 패자전에서 진영은 가위바위보로 정하며, 최종전은 승자전 패배팀이 진영을 선택한다.
  * 대회 규정  

    * [NLB Summer 2012 규정](http://www.nicegame.tv/board/bbs/view/1001/21/-/-/370480/30/)
  * [LOL the Champions](LOL%20the%20Champions.md)와의 연계 시스템  

    * **시드 챌린지** : ‘NLB’ 1위 vs ‘The Champions’ 4위  
( 시드 챌린지 승자는 The Champions 차기 시즌 16강 시드를 부여함)

    * **오프라인 예선 시드** : ‘NLB’ 2위 ~ 4위  
( The Champions 차기 시즌 오프라인 예선 시드를 부여함)

  * 특이 사항  

    * 온게임넷의 롤챔스와는 다르게 대회 클라이언트를 쓰지 않는다. 방송대회의 특성상 만에 하나 존재하는 방플을 방지하기 위해 대회 클라이언트를 쓰지 않는데 이때문에 16강 B조 경기가 있던 6월 28일에 발생한 [리그 오브 레전드](%EB%A6%AC%EA%B7%B8%20%EC%98%A4%EB%B8%8C%20%EB%A0%88%EC%A0%84%EB%93%9C.md)의 서버 다운 당시에 경기를 할 수단이 존재하지 않아 다음 날로 연기가 되기도 했다.
  * 참가 팀 : 총 16개 팀  

    * 챌린저 서킷 섬머 예선 최종 탈락팀 16개팀이 합류.

[신지드](%EC%8B%A0%EC%A7%80%EB%93%9C.md) 장인 **싱선생**(소환사명 : 싱선생)의 맞대결로 모두의 관심을
모았던 경기였다.

  

픽밴에서 신지드와 아리가 밴을 할것으로 예상했지만 모두의 예상을 깨고 아리는 밴을 하지 않았다!. 하지만 황충아리가
[모르가나](%EB%AA%A8%EB%A5%B4%EA%B0%80%EB%82%98.md)를 픽하면서 결국 두 선수의 장인 플레이를 볼순
없었다.

  

경기 내용은 조금 지루했던 편. 더군다나 리틀히포는 바론을 3번이나 땄고 작은 하마팀의 바루스가 "전설의 출현"까지 띄웠지만 싱선생의
[니달리](%EB%8B%88%EB%8B%AC%EB%A6%AC.md)의 폭풍 백도어에 속수무책으로 기방하다 끝난 경기였다.<del>[야!니달리좀 나대지 않게해라!](%EC%95%BC%20%EA%B0%9C%EC%A7%96%EB%8A%94%20%EC%86%8C%EB%A6%AC%20%EC%A2%80%20%EC%95%88%20%EB%82%98%EA%B2%8C%20%ED%95%98%EB%9D%BC.md)</del>
