# 2. [미국](%EB%AF%B8%EA%B5%AD.md)의
[하프트랙](%ED%95%98%ED%94%84%ED%8A%B8%EB%9E%99.md)(반궤도)
[장갑차](%EC%9E%A5%EA%B0%91%EC%B0%A8.md) ¶

  

[M3](M3.md) 장갑차에 [M2](M2.md) 대공기관총 4문을 탑재한 반궤도 장갑차. 정작 실전에서 미군은 적 전투기나
공격기에게 시달릴 일이 거의 없었기 때문에 대공용보다는 대지상용, 특히 보병용으로 많이 사용했다. 덕분에 별명이 고기 다지기(meat
chopper)인데 이 화망에 걸리면 정말 다져지기 때문에 이런 별명이 붙었다...

  

![M16tr.jpg](//rv.wkcdn.net/http://rigvedawiki.net/r1/pds/M16/M16tr.jpg)

[JPG image (52.88 KB)]

  

[컴퍼니 오브 히어로즈](%EC%BB%B4%ED%8D%BC%EB%8B%88%20%EC%98%A4%EB%B8%8C%20%ED%9E%88%EC%96%B4%EB%A1%9C%EC%A6%88.md)에서 등장한다. M3 장갑차에 100뮤니션을 주고 업그레이드할 수 있는데 유일한
연합군의 보병분쇄 차량이다. 다만 업그레이드할 경우 수송 능력과 분대원 보충 능력이 사라진다. 현실과 마찬가지로 대공용으로도 쓸 수 있지만
대공용으로는 빈약한 편이다.

  

[워 썬더](%EC%9B%8C%20%EC%8D%AC%EB%8D%94.md)에서도 등장한다. 미국 대공전차 트리에 2티어를 차지하는데,
대공용으로는 엄청나다.  
참고로 우리나라는 M16을 직접 운용한 적은 없는듯 하나 비슷하게 M2 중기관총을 4문 묶은 대공기관총을
[승공포](%EC%8A%B9%EA%B3%B5%ED%8F%AC.md)라는 이름으로 고정식, 혹은 트럭 탑재형으로 만들어 주로
[공군](%EA%B3%B5%EA%B5%B0.md)에서 [AN-2](AN-2.md)기 저지 및 여기서 투하된 적 특수부대 저지용으로
운용했다.

# 4. [XTM](XTM.md)에서 방송하는 남자공감랭크쇼 ¶

남자공감랭크쇼라는 이름답게 남자들의 흥미를 끌만한 소재들로 순위를 정해서 방송한다.  
주로 군사관련, 스포츠, 액션 영화 순위들이 자주 올라오며, 유지현 →
[류지혜](%EB%A5%98%EC%A7%80%ED%98%9C.md) →
[원자현](%EC%9B%90%EC%9E%90%ED%98%84.md) →
[서유리](%EC%84%9C%EC%9C%A0%EB%A6%AC.md) →
[신아영](%EC%8B%A0%EC%95%84%EC%98%81.md) 이 매번 섹시한 원피스 드레스를 입고 방송하는 것도
[눈요기거리](http://m16.interest.me/Index/12/Board/View?b_seq=4&page=1)에서 확인해보자. 방송
끝자락에 틀어주는 섹시한 뮤직 비디오는 덤이다. 나레이션은 [이정구](%EC%9D%B4%EC%A0%95%EA%B5%AC.md),
[전진아](%EC%A0%84%EC%A7%84%EC%95%84.md).
