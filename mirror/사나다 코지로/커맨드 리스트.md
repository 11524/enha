※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 업어치기 : 근접해서 ← or → + B  

★ 특수기  

  * 천주 찌르기 : 상대 다운시 ↘ + B   

★ 필살기  

  * 준살 : ↓↙← + A or B  
┗ 준살 ・ 연 : 준살 히트시 A or B 연타

  * 허공살 : ↓ 모으고 ↑ + A or B 
  * 질공살 : ← 모으고 → + A or B
  * 낭아(●) : ← 모으고 → + C   
┣ 낭아 ・ 직식(●) : 낭아 히트시 ↓↘→ + C  
┗ 낭아 ・ 사식 : 낭아 히트시 ↓↙← + C  

★ 초오의  

  * 진 ・ 랑아 : ↓↙←→ + AB  

★ 잠재오의  

  * 최종 ・ 랑아 : ↓↙←→ + B (모으기 가능, 최대 축적시 [가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 준속 달리기 : 근접해서 ← or → + CD  

★ 특수기  

  * 이심 찌르기 : 상대 다운시 ↘ + B
  * 무이 ・ 열(●) : 힘모드에서 BC를 모았다 사용 
  * 산화 : 기/극모드에서 → + BC  

★ 필살기  

  * 상미섬 : ↓↙← + A
  * 무명검(●) : ↓↙← + B
  * 허공살 : →↓↘ + A or B 
  * 질공살 : ↓↘→ + A or B
  * 순진 : ↓↙← + C   
┗ 천지 : 순진 중 C  

★ 초오의  

  * 무명검 ・ 지 : ↓↙←→ + AB  

★ 잠재오의  

  * 낭아 ・ 영 : ↓↙←→ + B (모으기 가능, 최대 축적시 [가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))  

★ 난무오의  

  * 백설연참 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, A, ↓↘→ + C  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, → + BC, A, B, ↓↘→ + B
