# 1. [KOF 94](KOF%2094.md) ¶

★ 기본잡기  

  * 덩크 버스터 : 근접해서 ← or → + C
  * 럭키 크래쉬 : 근접해서 ← or → + D  

★ 특수기  

  * 덩크 떨구기 : 공중에서 ↓ + C  

★ 필살기  

  * 데스 바운드 : ↓↘→ + A or C
  * 사이클론 브레이크 : ↓↙←↖ + B or D
  * 럭키 비전 : ←↙↓↘→ + B or D  

★ 초필살기  

  * 헬 바운드 : ↓↙←↙↓↘→ + BC

# 2. [KOF 98](KOF%2098.md) ¶

★ 기본잡기  

  * 덩크 버스터 : 근접해서 ← or → + C
  * 럭키 크래쉬 : 근접해서 ← or → + D  

★ 특수기  

  * 덩크 떨구기 : 공중에서 ↓ + A
  * 럭키 킥 : → + B  

★ 필살기  

  * 데스 바운드 : ↓↘→ + A or C
  * 데스 덩크 : ↓↙← + A or C
  * 데스 슛 : ↓↓ + A or B or C or D
  * 데스 힐 : →↓↘ + B or D
  * 럭키 비전 : ↓↘→ + B or D
  * 사이클론 브레이크 : ↓↙← + B or D  

★ 초필살기  

  * 헬 바운드 : ↓↘→↓↘→ + A or C
  * 럭키 드라이버 : ↓↘→↓↘→ + B or D

# 3. [KOF 98 UM](KOF%2098%20UM.md) ¶

★ 기본잡기  

  * 덩크 버스터 : 근접해서 ← or → + C
  * 럭키 크래쉬 : 근접해서 ← or → + D  

★ 특수기  

  * 덩크 떨구기 : 공중에서 ↓ + A
  * 럭키 킥 : → + B  

★ 필살기  

  * 데스 바운드 : ↓↘→ + A or C
  * 데스 덩크 : ↓↙← + A or C (공중 가능)
  * 데스 슛 : ↓↓ + A or B or C or D
  * 데스 힐 : →↓↘ + B or D
  * 럭키 비전 : ↓↘→ + B or D
  * 사이클론 브레이크 : ↓↙← + B or D  

★ 초필살기  

  * 헬 바운드 : ↓↘→↓↘→ + A or C
  * 럭키 드라이버 : ↓↘→↓↘→ + B or D (잡기)
