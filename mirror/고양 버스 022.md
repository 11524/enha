[보은교통](%EB%B3%B4%EC%9D%80%EA%B5%90%ED%86%B5.md) 022번과 전혀 관계 없다.

  

행신동~화전역앞 구간은 [고양 BRT](%EA%B3%A0%EC%96%91%20BRT.md) 차선에 입선한다. <del>그리고 시작되는

[075번](%EA%B3%A0%EC%96%91%20%EB%B2%84%EC%8A%A4%20075.md) 계열이 이용한다.`[1]`

를 직접경유 운행하며, 대덕동 구역을 [서울 버스7726](%EC%84%9C%EC%9A%B8%20%EB%B2%84%EC%8A%A4%207726.md)과 같이 대덕동 구역을 순환하다가
현천동 난점마을에서 회차한다. 거의 전구간을 순환한다고 해도 과언이 아니지만 승차방향에 유의해야한다. 또한, 화전역과 항공대입구쪽은 방향

### 1.1. 화전역, 항공대학교앞(화랑로), 항공대입구(중앙차로) 정류장에서 승차시 유의사항 ¶

  * 행신동 방향  

항공대정문 → 화전역(앞)(수색방향) → 항공대학교앞(화랑로) → 항공대입구(중앙차로) → 화전역(앞)(대화역방향) → 화도교

  
화전역(앞) : 양방향 승차 가능 (단, **행신동 행선판**으로 운행 중인 022번에 승차해야한다.)  
항공대학교앞(화랑로) : 수색방향 버스정류장  
항공대입구(중앙차로) : 대화역방향 버스정류장  

  * 대덕동, 항공대캠퍼스내부 방향  

화도교 → 화전역(앞)(수색방향) → 항공대입구(중앙차로) → 항공대학교앞(화랑로) → 화전역(앞)(대화역방향) → 항공대정문

  
화전역(앞) : 양방향 승차 가능 (단, **대덕동 행선판**으로 운행 중인 022번에 승차해야한다.)  
항공대입구(중앙차로) : 수색방향 버스정류장  
항공대학교앞(화랑로) : 대화역방향 버스정류장
