# 2. [아랑전설](%EC%95%84%EB%9E%91%EC%A0%84%EC%84%A4.md) 3 ¶

★ 기본잡기  

  * 풍차 무너뜨리기 개 : 근접해서 → + D
  * 몽앵 개 : 공중 근접해서 ↓ + C  

★ 컴비네이션 아츠  

  * 비연연무 : 근접해서 B - → + B - → + B
  * 운작질풍무 : 근접해서 C - C - C - D
  * 롱독락 : 근접해서 C - D
  * 하앵 : 근접해서 C - C - ↓ + B  

★ 특수기  

  * 용의 춤 : ← + D  

★ 필살기  

  * 날다람쥐의 춤 : 공중에서 ↓ + BC or CD
  * 화접선 : ↓↘→ + A or C (BD 입력으로 페인트)
  * 용염무 : ↓↙← + A or C (BD 입력으로 페인트)
  * 필살인봉 : ←↙↓↘→ + C
  * 양염의 춤 : ↓↓ + ABC  

★ 초필살기  

  * 초필살인봉 : →↙→ + CD  

★ 잠재능력  

  * 초필살인봉 : 도발 중 C 누른채로 →↙→ + BD
