## 1 개요 ¶

트로피코 4 모던 타임즈의 역사적 사건들에 따른 섬의 변화에 대한 내용이다.  
플레이하는 독재자의 성향에 따라서 나타나는 사건은 달라진다, 고로 필수적으로 확인할것. 건물이 열리는 시간은 변화없음. 2000년대 이후의
시대변화는 사실상 가상이므로 무시해도 무방.

### 2.1 50년대 ¶

  * 1950~1953년 : [한국전쟁](%ED%95%9C%EA%B5%AD%EC%A0%84%EC%9F%81.md), 모든 군사시설물이 반값
  * 1953~1956년 : [쿠바 공산화](%EC%BF%A0%EB%B0%94/%EC%97%AD%EC%82%AC.md), 소련으로부터 지원이 2배가됨
  * 1954~1956년 : 주류세 , 럼주와 스파이스 럼주의 가격이 50%하락
  * 1957년 : 도시 공원 사용가능
  * 1958~1960년 : 전세계적 [조류독감](%EC%A1%B0%EB%A5%98%EB%8F%85%EA%B0%90.md) 파동,쇠고기와 훈제쇠고기 값이 50% 상승한다.
  * 1959년 : 슈퍼마켓 사용가능

### 2.2 60년대 ¶

  * 1960년 : 극장 사용가능
  * 1961년 : 영리병원 사용가능
  * 1962년 : 어류양식장 사용가능
  * 1962~1964년 : 석유시장 파동, 원유와 석유제품 가격이 50% 하락.
  * 1963년~1965년 : [쿠바 미사일 위기](%EC%BF%A0%EB%B0%94%20%EB%AF%B8%EC%82%AC%EC%9D%BC%20%EC%9C%84%EA%B8%B0.md). 무조건적으로 징병제 실행됨.
  * 1964년 : 수질정화장 사용가능
  * 1965년 : 유람선 식당 사용가능
  * 1965~1968년 : 베트남 전쟁, 매년 반란군이 공격해옴 
  * 1966년 : 현대적 아파트 사용가능
  * 1966~1968년 : 세계적 경제난. 금값이 50% 상승.
  * 1967년 : 바이오 농장 사용가능
  * 1967~1970년 : 항공산업 붐. 보크사이트 가격 50% 하락
  * 1968년 : 유기농 목장 사용가능
  * 1968~1970년 : [전국구 학생시위](68%ED%98%81%EB%AA%85.md). 해외에서 노동자를 영입해오는것이 불가능

### 2.3 70년대 ¶

  * 1970년 : 경찰본부
  * 1970~1973년 : [살바도르 아옌데](%EC%82%B4%EB%B0%94%EB%8F%84%EB%A5%B4%20%EC%95%84%EC%98%8C%EB%8D%B0.md)가 선거에서 이김.
  * 1971년 : 섬의 인구가 일정치를 넘으면 대통령궁 업그레이드
  * 1972년 : [워터게이트 사건](%EC%9B%8C%ED%84%B0%EA%B2%8C%EC%9D%B4%ED%8A%B8%20%EC%82%AC%EA%B1%B4.md) , 미국과의 관계 초기화
  * 1973~1975년 : [아우구스토 피노체트](%EC%95%84%EC%9A%B0%EA%B5%AC%EC%8A%A4%ED%86%A0%20%ED%94%BC%EB%85%B8%EC%B2%B4%ED%8A%B8.md)가 아옌데를 쫒아냄.
  * 1975년 : 태양열 발전소 사용가능
  * 1975~1978년 : [FARC](FARC.md)의 과격활동. 매년 인질극이 벌어짐.
  * 1976년 : 비즈니스 센터 사용가능
  * 1978년 : 국립은행 사용가능
  * 1979년 : 7급 호텔 사용가능

### 2.4 80년대 ¶

  * 1980년 : 가전제품 공장 사용가능
  * 1980~1982년 : [소련-아프가니스탄 전쟁](%EC%86%8C%EB%A0%A8-%EC%95%84%ED%94%84%EA%B0%80%EB%8B%88%EC%8A%A4%ED%83%84%20%EC%A0%84%EC%9F%81.md), 소련으로부터의 모든 수입이 중단됨.
  * 1981년 : 지하철 사용가능
  * 1982년 : [포클랜드 전쟁](%ED%8F%AC%ED%81%B4%EB%9E%9C%EB%93%9C%20%EC%A0%84%EC%9F%81.md). 유럽과의 관계가 리셋됨.
  * 1983년 : 신형 비행장 사용가능
  * 1985년 : 현대식 콘도 사용가능
  * 1985~1988년 : 과학자들이 소금이 건강에 좋다는 사실을 발견. 소금 가격이 50% 상승한다.
  * 1986년 : TCC 본사 사용가능
  * 1987년 : 자동차 공장 사용가능
  * 1988년 : 초호화 대성당 사용가능

### 2.5 90년대 ¶

  * 1990년 : 우주 프로그램 사용가능
  * 1991년 : [걸프전](%EA%B1%B8%ED%94%84%EC%A0%84.md) , 중동과의 관계 초기화
  * 1994년 : 지구라트 사용가능
  * 1996년 : 바벨타워 사용가능
