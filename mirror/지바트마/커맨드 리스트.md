# 1. [KOF MI2](KOF%20MI2.md) ¶

  

★ 기본잡기  

  * 유 : 근접해서 ← or → + C
  * 맹 : 근접해서 ← or → + D  

★ 특수기  

  * 업화진전 : →→ + B or D
  * 창관 : ↙ + C  

★ 필살기  

  * 신우라 : ↓↘→ + A or C
  * 무연조 : ↓↘→ + B or D
  * 재염 : ↓↙← + A or C
  * 황천 : →↘↓↙← + B or D (잡기)  

★ 초필살기 (☆은 기게이지 소모 개수 표시)  

  * 어둠의 손톱(☆) : ↓↙←↙→ + A or C
  * 연옥 치(☆☆) : ↓↘→↓↘→ + B
  * 연옥 체(☆☆☆) : ↓↘→↓↘→ + D  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ C
  * 02 : → + A ▶ C (업화진전으로 이행 가능)
  * 03 : ↓ + A ▶ ↓ + A ▶ ↓ + C ▶ ↓ + C
  * 04 : B ▶ D
  * 05 : B ▶ ← + D (페인트)
  * 06 : ← + B ▶ D (업화진전으로 이행 가능)
  * 07 : C ▶ C ▶ C
  * 08 : C ▶ ← + C ▶ C
  * 09 : → + C ▶ → + C ▶ C
  * 10 : → + C ▶ ← + C ▶ C
  * 11 : ↓ + C ▶ C ▶ CD
  * 12 : →→ + C ▶ C ▶ C
  * 13 : 업화진전 중 ← or → or ↑ or ↓
  * 14 : 업화진전 중 →→
  * 15 : 업화진전 중 → + AB
  * 16 : 업화진전 중 ← + AB
  * 17 : 업화진전 중 →→ + B or D (업화진전 해제)
  * 18 : 업화진전 중 A
  * 19 : 업화진전 중 B
  * 20 : 업화진전 중 C
  * 21 : 업화진전 중 D
  * 22 : 업화진전 중 CD (업화진전 해제)
  * 23 : 업화진전 중 A ▶ C ▶ B
  * 24 : 업화진전 중 D ▶ ← + B ▶ D (업화진전 해제)

# 2. [KOF MIA](KOF%20MIA.md) ¶

  

★ 기본잡기  

  * 유 : 근접해서 ← or → + AC
  * 맹 : 근접해서 ← or → + BD  

★ 특수기  

  * 업화진전 : →→ + B or D
  * 창관 : ↙ + C
  * 보살 : ↘ + D  

★ 필살기  

  * 신우라 : ↓↘→ + A or C
  * 무연조 : ↓↘→ + B or D
  * 재염 : ↓↙← + A or C
  * 황천 : →↘↓↙← + B or D (잡기)
  * 연꽃 : 업화진전 중 →←→ + C  

★ 초필살기(☆은 기게이지 소모 개수 표시)  

  * 어둠의 손톱(☆) : ↓↙←↙→ + A or C
  * 연옥 치(☆☆) : ↓↘→↓↘→ + B
  * 연옥 체(☆☆☆) : ↓↘→↓↘→ + D
  * 만다라(☆☆☆) : ↓↙←↓↙← + B or D  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ C
  * 02 : → + A ▶ C (업화진전으로 이행 가능)
  * 03 : ↓ + A ▶ ↓ + A ▶ ↓ + C ▶ ↓ + C
  * 04 : B ▶ D
  * 05 : B ▶ ← + D (페인트)
  * 06 : ← + B ▶ D (업화진전으로 이행 가능)
  * 07 : C ▶ C ▶ C
  * 08 : C ▶ ← + C ▶ C
  * 09 : → + C ▶ → + C ▶ C
  * 10 : → + C ▶ ← + C ▶ C
  * 11 : ↓ + C ▶ C ▶ CD
  * 12 : →→ + C ▶ C ▶ C
  * 13 : 업화진전 중 ← or → or ↑ or ↓
  * 14 : 업화진전 중 →→
  * 15 : 업화진전 중 → + AB
  * 16 : 업화진전 중 ← + AB
  * 17 : 업화진전 중 →→ + B or D (업화진전 해제)
  * 18 : 업화진전 중 A
  * 19 : 업화진전 중 B
  * 20 : 업화진전 중 C
  * 21 : 업화진전 중 D
  * 22 : 업화진전 중 CD (업화진전 해제)
  * 23 : 업화진전 중 A ▶ C ▶ B
  * 24 : 업화진전 중 D ▶ ← + B ▶ D (업화진전 해제)
