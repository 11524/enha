## 2. File ¶

  * 상위 항목 : [컴퓨터 관련 정보](%EC%BB%B4%ED%93%A8%ED%84%B0%20%EA%B4%80%EB%A0%A8%20%EC%A0%95%EB%B3%B4.md)
<del>부모님이 뒤져보면 불안한것</del>  
컴퓨터에 쓰이는 데이터들을 모아놓은 것으로 어원은 1번.

  

보조 기억장치에 저장된 것을 말하며, 필요시마다 주 기억장치로 로드되어 사용된다. 각 파일은 파일 이름과
[확장자](%ED%99%95%EC%9E%A5%EC%9E%90.md)로 구분한다.

  

[OS](OS.md)별로 각각 파일 시스템이 다르며,
[윈도우즈](%EC%9C%88%EB%8F%84%EC%9A%B0%EC%A6%88.md)의 경우 [NTFS](NTFS.md)라는 파일
시스템을 사용하고 있다. 아직 [FAT32](FAT32.md)도 사용되고 있긴 하다.

  

파일 시스템이 달라도 호환되는 확장자의 파일은 서로 교환하여 사용 가능하다.
