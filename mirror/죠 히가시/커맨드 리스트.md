## 12.2. 裏 버전 ¶

★ 기본잡기  

  * 무릎지옥 : 근접해서 ← or → + C
  * 레그 스루 : 근접해서 ← or → + D  

★ 특수기  

  * 로우 킥 : → + B
  * 슬라이딩 : ↘ + B  

★ 필살기  

  * 폭열권 : A or C 연타  
┗ 폭열권 피니시 : ↓↘→ + A or C

  * 허리케인 어퍼 : ←↙↓↘→ + A or C
  * 프레셔 니 : →↓↘ + A or C  
┗ 황금의 장딴지 : ↓↙← + B or D

  * 슬래시 킥 : ←↙↓↘→ + B or D
  * 타이거 킥 : →↓↘ + B or D  

★ 초필살기  

  * 스크류 어퍼 : ↓↘→↓↘→ + A or C
  * 썬더 파이어 : ↓↘→↓↘→ + B or D

# 20. [KOF XIII](KOF%20XIII.md) ¶

※ EX 대응기(●)는 A or C로 나가는 기술은 AC로, B or D로 나가는 기술은 BD로 EX 필살기를 쓸 수 있다.

  

★ 기본잡기  

  * 무릎지옥 : 근접해서 ← or → + C or D  

★ 특수기  

  * 스텝 인 미들 킥 : → + B
  * 슬라이딩 : ↘ + B  

★ 필살기  

  * 허리케인 어퍼(●) : ←↙↓↘→ + A or C
  * 타이거 킥(●) : →↓↘ + B or D
  * 슬래시 킥(●) : ←↙↓↘→ + B or D
  * 폭열권(●) : A or C 연타  

┗ 폭열권 피니시 : ↓↙← + A or C

  * 황금의 발꿈치(●) : ↓↙← + B or D  

★ 초필살기  

  * 스크류 어퍼(●) : ↓↘→↓↘→ + A or C
  * 폭열 허리케인 타이거 발꿈치 : ↓↘→↘↓↙← + A or C  

★ NEO MAX 초필살기  

  * 스크류 스트레이트 : ↓↘→↓↘→ + BD
