# 1. 노선 정보 ¶

  * 기점 : 안산역
  * 종점 : 금산섬유
  * 첫차 : 안산역 5:20 / 금산섬유 5:40
  * 막차 : 안산역 22:30 / 금산섬유 22:30
  * 배차간격 : 20~30분  

[경원여객](%EA%B2%BD%EC%9B%90%EC%97%AC%EA%B0%9D.md)에서 운행중인
[시내버스](%EC%8B%9C%EB%82%B4%EB%B2%84%EC%8A%A4.md) 노선으로
[안산역](%EC%95%88%EC%82%B0%EC%97%AD.md)을 출발해
[초지역](%EC%B4%88%EC%A7%80%EC%97%AD.md)을 거쳐 반월공단(안산스마트허브) 염색단지를 잇는 노선이다.

# 2. 특징 ¶

1-1번의 전신은 마을버스 1번으로 운행업체인 진보운수가 폐업하자, 대체버스를 운행하던
[경원여객](%EA%B2%BD%EC%9B%90%EC%97%AC%EA%B0%9D.md)이 기존 마을버스 노선과 100% 같은 루트로 정식
시내버스 인가를 받아 신설하였다.

  

[안산역](%EC%95%88%EC%82%B0%EC%97%AD.md) 맞은편 버스정류장에는 정차하지 않는다.
