# 1. 개요 ¶

이 해에는 [독일](%EB%8F%85%EC%9D%BC.md)
[베를린](%EB%B2%A0%EB%A5%BC%EB%A6%B0.md)에서 [하계올림픽](%EC%98%AC%EB%A6%BC%ED%94%BD.md)이 개최될 예정이었으나, [제1차세계대전](%EC%A0%9C1%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)으로 인해
전면취소되었고, 종전 후인 [1920년](1920%EB%85%84.md)
[벨기에](%EB%B2%A8%EA%B8%B0%EC%97%90.md)
[앤트워프](%EC%95%A4%ED%8A%B8%EC%9B%8C%ED%94%84.md)에서 다시 개최하게 되었다.

# 2. 사건 ¶

[[edit](http://rigvedawiki.net/r1/wiki.php/1916%EB%85%84?action=edit&section=3

## 2.1. 실제 ¶

  * [7월 1일](7%EC%9B%94%201%EC%9D%BC.md) : [솜 전투](%EC%86%9C%20%EC%A0%84%ED%88%AC.md)가 시작되었다. 이는 [1차대전](1%EC%B0%A8%EB%8C%80%EC%A0%84.md) 최대의 사상자를 낸 전투였다.
  * [9월 15일](9%EC%9B%94%2015%EC%9D%BC.md) : [솜 전투](%EC%86%9C%20%EC%A0%84%ED%88%AC.md)에서 최초로 [전차](%EC%A0%84%EC%B0%A8.md)가 사용되었다.
  * [10월 11일](10%EC%9B%94%2011%EC%9D%BC.md) : 박중빈이 [원불교](%EC%9B%90%EB%B6%88%EA%B5%90.md)를 창시했다.
  * [11월 21일](11%EC%9B%94%2021%EC%9D%BC.md) : 에게해에서 [타이타닉](%ED%83%80%EC%9D%B4%ED%83%80%EB%8B%89.md)의 자매함인 브리태닉(Britannic)호가 기뢰를 건드려 침몰해 30명이 사망했다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/1916%EB%85%84?action=edit&section=4
)]

## 2.2. 가상 ¶

  * [4월 24일](4%EC%9B%94%2024%EC%9D%BC.md) : [한국](%ED%95%9C%EA%B5%AD.md)의 공작으로 [아일랜드](%EC%95%84%EC%9D%BC%EB%9E%9C%EB%93%9C.md)가 [영국](%EC%98%81%EA%B5%AD.md)을 상대로 독립 봉기를 일으켰다. - [1904 대한민국](1904%20%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)
  * 일자불명 : 갈라틴 평행계에서 전쟁에 승리한 독일이 자신들의 지배를 받는 [유럽연합](%EC%9C%A0%EB%9F%BD%EC%97%B0%ED%95%A9.md)을 창설했다. - [겁스 무한세계](%EA%B2%81%EC%8A%A4%20%EB%AC%B4%ED%95%9C%EC%84%B8%EA%B3%84.md)
