PolyGame Master

[대만](%EB%8C%80%EB%A7%8C.md)의 게임 회사 [IGS](IGS.md)가 사용하는
[기판](%EA%B8%B0%ED%8C%90.md). 1997년에 처음 나왔다. 후속 기판 PGM2는 2007년에 나왔으니 10년 가량
사용된 셈.

[네오지오](%EB%84%A4%EC%98%A4%EC%A7%80%EC%98%A4.md)나
[CPS1](CPS1.md),[CPS2](CPS2.md)같이 [68000](68000.md)/[Z80](Z80.md)
듀얼 CPU를 채택하였다. 국내에서는 [삼국전기](%EC%82%BC%EA%B5%AD%EC%A0%84%EA%B8%B0.md)나 [데몬프론트](%EB%8D%B0%EB%AA%AC%20%ED%94%84%EB%A1%A0%ED%8A%B8.md)가 [문방구게임기](%EB%AC%B8%EB%B0%A9%EA%B5%AC%20%EA%B2%8C%EC%9E%84%EA%B8%B0.md)쪽으로 나름
보급되어 있으며, 일본에서는 이 기판으로 [케이브](%EC%BC%80%EC%9D%B4%EB%B8%8C.md)의 [탄막 슈팅게임](%ED%83%84%EB%A7%89%20%EC%8A%88%ED%8C%85%20%EA%B2%8C%EC%9E%84.md)이 몇개
나왔기 때문에 그럭저럭 보급되어 있는 편. 이후 케이브는
[벌레공주님](%EB%B2%8C%EB%A0%88%EA%B3%B5%EC%A3%BC%EB%8B%98.md)부터는
[히타치](%ED%9E%88%ED%83%80%EC%B9%98.md)의 SH-3 CPU를 사용한 자체 기판을 사용한다.

PGM2의 경우 스펙이 제대로 공개되지 않아서 정보를 찾을 수가 없다(...)

2012년, PGM3 기판이 등장했다. 현재 대세를 따라가는 것인지, PC기반으로 추정된다.

### 성능 ¶

  * PGM  

    * CPU : 68000 20Mhz & [Z80](Z80.md) 8.468Mhz(후기 게임에서는 복제방지용 CPU로 [ARM](ARM.md) 코어 기반의 ASIC도 사용)
    * 사운드 : ICS 2115 Wavetable MIDI Synthesizer(최대발음수 32음)
    * 그래픽 : 448x224 해상도, 최대발색수 32768색, 8x8 16색 텍스트 타일, 32x32 32색 배경 타일, 최대 512x2048 크기의 32색 스프라이트(최대 표시수 256개)  

스프라이트 표시수만 제외하고는`[1]` 사실상 16비트 CPU를 메인으로 한 기판으로서는 최상의 성능이다.`[2]`

  

  * PGM 2`[3]`  

    * CPU : IGS036 (differs per game, internal code)
    * 사운드 : Yamaha YMZ774-S
    * 그래픽 : IGS037
    * Protection chip: R5F21256SN (extra MCU for protection and IC Card communication)
    * 저장장치 : ROM (Custom program ROM module (KOV3 only))  

  * PGM 3  

    * CPU : [인텔 아톰 듀얼코어 프로세서](%EC%9D%B8%ED%85%94%20%EC%95%84%ED%86%B0%20%EC%8B%9C%EB%A6%AC%EC%A6%88.md)
    * 칩셋 : Intel D525 + ICH8M
    * 메모리 : 2GB
    * 저장장치 : SD카드 (최대 1GB)
