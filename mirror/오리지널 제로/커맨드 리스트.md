# 1. [KOF 2001](KOF%202001.md) ¶

  

★ 기본잡기  

  * 사해식찰 : 근접해서 ← or → + C or D  

★ 특수기  

  * 스트라이커 소환 (크리자리드) : → + A
  * 스트라이커 소환 (글루건) : → + B
  * 스트라이커 소환 (롱) : → + C
  * 환마각상 : ← or → + A (레버 유지)

  * 참풍연파 살마 : ↓↘→ + C
  * 참풍연파 황아 : →↓↘ + A or C  

★ 초필살기  

  * 옥쇄참진 : ↓↘→↓↘→ + A or C
  * 백라멸정 : ↓↘→↓↘→ + B or D

★ 잡기

  * 백라멸정 : ↓↘→↓↘→ + B or D
