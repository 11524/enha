### 2.1. 개요 ¶

울산광역시의 시내버스 노선.

  

이 노선은 2013년 11월 2일 개편으로 신설된 노선이다.  
각 노선별로 1일 2회(총 1일 4회)씩 방어진순환도로로의 지원 운행을 담당한다.

  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%9A%B8%EC%82%B0%20%EB%B2%84%EC%8
A%A4%20101?action=edit&section=4)]

### 2.2. 주요 경유지 ¶

**1번 노선**  
꽃바위 → 방어동주민센터 → 일산해수욕장 → 현대중공업(현대예술회관) → 현대패밀리 명덕1,2차 → 한마음회관 → 동구청 → 꽃바위

  

**2번 노선**  
꽃바위 → 화암중학교 → 동구청 → 한마음회관 → 현대패밀리 명덕2,1차 → 현대중공업(현대예술회관) → 일산해수욕장 → 방어동주민센터 →
꽃바위

  

**지원 운행** (총 1일 4회)  
꽃바위 → 방어동주민센터 → 일산해수욕장 → 현대중공업 → 남목 → 성내 → 현대미포조선 → 문현관 → 꽃바위

### 2.4. 기타 ¶

  * 2000년도 초반에 운행했던 방어진 순환노선인 32번, 41번, 42번과 유사한 형태로 운행한다. 
  * 남목의 입장에서는 출퇴근지원운행이긴하나 무려 15여년만(대략)에 부활한 미포조선행 노선이라는 점에 의의가 있다. 또 승객 입장에서는 남목을 일부시간대를 제외하고는 들리지 않기 때문에 조심해야할 노선 1순위로 꼽힌다. 특히나 입석노선이므로 더더욱!
  * 미포조선측에서 이미 동구주거지 방면 사원통근버스를 운행하고 있어서 그런 지 출근시간대 성내 부근에서 지원운행하는 101번을 보면 그냥 공기수송 하는 모습만 보인다. 1,2번 운행계통도 이용하는 승객이 별로없는 건 마찬가지.안습
