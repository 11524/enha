## 1. 2012년 테마별 퀴즈 진행방식 - 791회 ~ 837회 ¶

총 3라운드로 진행되며, 매주 특정 주제에 관련된 문제만 나온다. <del>본격 [덕후](%EB%8D%95%ED%9B%84.md)를
위한 퀴즈 프로그램.</del> 매주 4명이 출연하며, 연승제에서 매주 우승자를 가리는 형식으로 진행된다. 1라운드와 파이널 라운드 진행
형식은 [미션 퀴즈](%EB%AF%B8%EC%85%98%20%ED%80%B4%EC%A6%88.md)의 진행 방식을 차용했다.

### 1.2. 1라운드 - 스피드 퀴즈 ¶

  * 각 출연자가 차례대로 등장하여 12개의 주관식 문제를 푼다.`[2]`
  * 문제당 제한시간 5초. 
  * 최하위를 제외한 나머지 3명이 다음 라운드에 진출하며, 동점자가 있을시 맞힌 문제의 난이도 총합이 높은 사람이 올라간다.
  * 819회부터는 1위가 파이널 라운드로 직행하며, 동점자가 있을 때에는 버저 문제 하나가 주어지고 이를 빨리 푸는 사람이 파이널 라운드에 진출한다.

### 1.3. 2라운드 - 군계일학 퀴즈 ¶

  * 남은 세 출연자는 10개`[3]`의 삼지선다형 문제를 풀게 된다. 
  * 문제당 제한시간은 3초이며, 문제당 점수는 (4-정답자 수)*10점이다. 
  * 문제를 다 푼 다음 상위 2명이 결승에 진출하며(819회부터는 1위만 결승 진출), 동점자가 있을 때에는 버저 문제 하나가 주어지고 이를 빨리 푸는 사람이 파이널 라운드에 진출한다.
  * 중간에 점수차가 많이 벌어지면 남은 문제와 상관없이 [콜드게임](%EC%BD%9C%EB%93%9C%EA%B2%8C%EC%9E%84.md)으로 결승진출자가 결정된다.`[4]`

### 1.4. 파이널 라운드 - 승부차기 퀴즈 ¶

  * 남은 두 출연자 앞에 놓인 태블릿 PC에 10개의 주관식 문제가 주어진다. 
  * [축구](%EC%B6%95%EA%B5%AC.md)의 [승부차기](%EC%8A%B9%EB%B6%80%EC%B0%A8%EA%B8%B0.md)와 비슷하게, 2라운드 성적이 좋은 사람`[5]`부터 번갈아가며 상대방에게 출제할 문제를 고른다. 
  * 이를 기본적으로 3문제까지 진행해서 더 많은 문제를 맞힌 사람이 우승하게 되며, 동점일 경우 승부가 갈릴 때까지 번갈아가며 문제를 풀게 된다. 
  * 주어진 10문제를 다 풀고도 동점일 경우, 새로운 10문제`[6]`가 두 출연자에게 주어진다. 
  * 819회부터는 문제 푸는 사람이 문제를 못 맞힐 경우에는 문제 출제자에게 기회가 넘어가게 되며, 825회부터는 3문제까지 풀었을 때 동점일 경우 버저 문제 세 문제 가운데서 먼저 두 문제를 맞힌 사람이 우승하게 된다.

## 2. 2012 왕중왕전/40주년 퀴즈제왕전 진행방식 - 832~837회 ¶

### 2.2. 2라운드 ¶

  * 객관식 5문제 : 형식은 예전과 동일
  * 주관식 5문제 : 한 문제 당 50점이 주어지며, 어느 문제에는 부분 점수(20점)가 걸린 문제도 있다. 공통 문제로 진행되며, 문제 유형은 2개 다답형, 나열 문제 등이 주어진다.
  * 버저 문제 12문제 : 30점 3문제, 40점 4문제, 50점 5문제가 주어지며, 맞히지 못해도 감점은 없다. 많은 점수를 획득한 사람이 우승자로 등극하며, 1위와 2위의 점수차가 너무 많이 나면 남은 문제와 상관없이 [콜드게임](%EC%BD%9C%EB%93%9C%EA%B2%8C%EC%9E%84.md)으로 우승자가 결정된다.
  * 퀴즈제왕전은 2012 왕중왕전 형식과 동일하고 2라운드 마지막 형식은 681~790회 2라운드 형식과 유사하며, 앞 4문제는 기본 문제 10점, 보너스 문제 20점, 뒤 3문제는 각각 2배의 점수 배점으로 구성되어 있다(감점은 없음).
