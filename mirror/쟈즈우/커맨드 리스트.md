  * 목 부러뜨려 던지기 : 근접해서 ← or → + C

  * 아지랑이각 : B 연타

  * 사빙마 : ↓↑ + A
  * 무도 수류파 : →↘↓↙← + B

  * 사하 절명주 : →↘↓↙←→ + A

# 2. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 목 부러뜨려 던지기 : 근접해서 ← or → + C or D  

★ 특수기  

  * 서서 무기 공격 : → + C
  * 앉아 무기 공격 : ↘ + C  

★ 필살기  

  * 배수 규환살 : ↓↘→ + A
  * 초승달 : ↓↘→ + C
  * 굉천 : ↓↙← + A or C
  * 호염탄 : →↓↘ + A or B or C
  * 카마이타치 : ←↓↙ + B or D
  * 사빙마 : ↓↑ + A or C
  * 무도 수류파 : →↘↓↙← + B or D
  * 무도 천륜조 : 공중에서 벽에 닿는 순간 ← + C  

★ 초필살기  

  * 아지랑이권 : →↘↓↙←→ + A or C
  * 아지랑이각 : →↘↓↙←→ + B or D  

★ 리더 초필살기  

  * 사하 절명주 : →↘↓↙←→ + E
