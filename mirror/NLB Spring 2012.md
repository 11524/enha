## 1. 개요 ¶

  * 주최 : [라이엇게임즈](%EB%9D%BC%EC%9D%B4%EC%97%87%EA%B2%8C%EC%9E%84%EC%A6%88.md)
  * 주관 : [온게임넷](%EC%98%A8%EA%B2%8C%EC%9E%84%EB%84%B7.md), [나이스게임TV](%EB%82%98%EC%9D%B4%EC%8A%A4%EA%B2%8C%EC%9E%84TV.md)
  * 스폰서 : [라이엇게임즈](%EB%9D%BC%EC%9D%B4%EC%97%87%EA%B2%8C%EC%9E%84%EC%A6%88.md)
  * 중계진  

    * 캐스터 : [정진호](%EC%A0%95%EC%A7%84%ED%98%B8.md), [김의중](%EA%B9%80%EC%9D%98%EC%A4%91.md)
    * 해설 : [이형주](%EC%9D%B4%ED%98%95%EC%A3%BC.md), [김태우](%EA%B9%80%ED%83%9C%EC%9A%B0%28%ED%95%B4%EC%84%A4%EC%9E%90%29.md), <del>[오성균](%EC%98%A4%EC%84%B1%EA%B7%A0.md)</del>`[1]`
  * 대회 상금  

    * **총상금** : 10,000,000 원  

      * 1위 : 6,000,000 원
      * 2위 : 3,000,000 원
      * 3위 : 1,000,000 원
  * 대회 일정  

    * 전체 일정 : 2012년 3월 13일(화) ~ 5월 10일(목)
    * 세부 일정  

      * 16강 : 3월 13일(화) ~ 3월 15일(목)
      * 8강 : 3월 20일(화) ~ 4월 26일(목)
      * 4강 : 5월 1일(화) ~ 5월 3일(목)
      * 3·4위전 : 5월 8일(화)
      * 결승전 : 5월 10일(목)
  * 진행 방식  

    * **16강** : 단판제 싱글 토너먼트  
(비공개 선택 방식)

    * **8강** : 2개조 풀 리그, 3전 2선승제`[2]`  
(1~2세트는 토너먼트 드래프트, 3세트는 비공개 선택 방식)

    * **4강**~**결승** : 8강의 조별 1,2위의 크로스 토너먼트 방식. 싱글 앨리미네이션, 5전 3선승제  
(1~4세트는 토너먼트 드래프트, 5세트는 비공개 선택 방식)

  * 대회 규정  

    * [NLB Spring 2012 규정](http://www.nicegame.tv/board/bbs/view/1001/21/-/-/370473/30/)
  * [LOL the Champions](LOL%20the%20Champions.md)와의 연계 시스템  

    * **시드 챌린지** : ‘NLB’ 1위 vs ‘The Champions’ 4위  
( 시드 챌린지 승자는 The Champions 차기 시즌 16강 시드를 부여함)

    * **오프라인 예선 시드** : ‘NLB’ 2위 ~ 4위  
( The Champions 차기 시즌 오프라인 예선 시드를 부여함)

  * 특이 사항  

    * [나이스게임TV](%EB%82%98%EC%9D%B4%EC%8A%A4%EA%B2%8C%EC%9E%84TV.md)의 첫 [리그 오브 레전드](%EB%A6%AC%EA%B7%B8%20%EC%98%A4%EB%B8%8C%20%EB%A0%88%EC%A0%84%EB%93%9C.md) 공식 리그.
    * 시즌 중간에 [해설자](%EC%98%A4%EC%84%B1%EA%B7%A0.md)의 이탈로 객원해설이 대타로 들어왔다.
  * 참가 팀 : 총 16개 팀  

    * 정규리그 오프라인 예선전 탈락 12개 팀
    * 온라인 예선전 조별 2위팀 중 추가 선발전을 통해 선발된 4개 팀  
-> 추가 선발전 진행 : 4개 조 단판 싱글 토너먼트를 통해 각 조별 1위, 총 4개 팀 선발
    * 참가팀 명단
