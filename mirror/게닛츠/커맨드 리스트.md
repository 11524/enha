# 1. [KOF 96](KOF%2096.md) ¶

  

★ 기본잡기  

  * 명부의 문 (МЭЙФУ НО МОН) : 근접해서 ← or → + C
  * 쌍괄살 (СОКАЦУСАЦУ) : 근접해서 ← or → + D  

★ 필살기  

  * 세상의 바람 (ЁНОКАЗЭ) : ←↙↓↘→ + A or B or C or D
  * 만박 상와 (ВАНГЁ ТОКОБУСЭ) : ↓↙← + A
  * 만박 마멸 (ВАНГЁ МАМЭЦУ) : ↓↙← + C
  * 빙하 (ГЁГА) : ↓↙← + B or D  

★ 초필살기  

  * 암통곡 (ЯМИДОКОКУ) : 근접해서 →↘↓↙←→↘↓↙← + C
  * 진 팔치녀 교 (СИН ЯОДОМЗ МИЗУЧИ) : ↓↙←↙↓↘→ + A or C
  * 진 팔치녀 십조각 (СИН ЯОДОМЗ ЖИССОКОКУ) : ↓↙←↙↓↘→ + A or C (MAX 전용)
