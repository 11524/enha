소니 컴퓨터 엔터테인먼트에서 개발한 [플레이스테이션3](%ED%94%8C%EB%A0%88%EC%9D%B4%EC%8A%A4%ED%85%8C%EC%9D%B4%EC%85%983.md) 기반의
[아케이드](%EC%95%84%EC%BC%80%EC%9D%B4%EB%93%9C.md)용
[게임](%EA%B2%8C%EC%9E%84.md)[기판](%EA%B8%B0%ED%8C%90.md). 2007년 [철권6](%EC%B2%A0%EA%B6%8C%206.md)에 사용되면서 데뷔했다.

실질적으로 그전부터 [SCE](SCE.md) 콘솔베이스의 기판을 자주 써온 [반다이남코게임즈](%EB%82%A8%EC%BD%94.md)의 아케이드 주력 시스템 보드.

위키피디아에 작성된 하드웨어 정보 [＃](http://en.wikipedia.org/wiki/Namco_System_357)  

[CPU](CPU.md): Cell Broadband Engine consisting of one 3.2GHz PowerPC-based
Power Processing Element (PPE) and six Synergistic Processing Elements (SPEs).  
System memory: 256MB XDR DRAM  
Graphics: [NVIDIA](NVIDIA.md) GeForce 7900-based RSX 'Reality Synthesizer'
with 256MB GDDR3 RAM  
Media: internal [HDD](HDD.md) used for storage of games.

**UI: [Sony](Sony.md) [XMB](XMB.md)**

기반이 PS3다 보니 성능상으로 PS3와 매우 흡사하며, 그래픽 출력도 비슷. **소니 XMB를 그대로 사용한다!**
[유튜브](http://www.youtube.com/watch?v=_ss-MHvWDCk)  
다른 점이라면 PS3에 있는 블루레이가 여기에는 없다는 것과 하드디스크가 다르다는 것인데, PS3의 경우 2.5인치 하드디스크
5400rpm을 사용하나, SYSTEM 357의 경우 3.5인치 하드디스크 7200rpm을 사용한다.

아종 기판으로 네트워크의 강화와 시스템 처리를 강화한 SYSTEM 369가 있다. 첫 대응 게임은 당연하지만 [이 게임](%EC%B2%A0%EA%B6%8C%20%ED%83%9C%EA%B7%B8%20%ED%86%A0%EB%84%88%EB%A8%BC%ED%8A%B8%202.md).
