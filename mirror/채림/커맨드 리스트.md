# 2. [KOF MI2](KOF%20MI2.md) ¶

  

★ 기본잡기  

  * 정수리 떨구기 : 근접해서 ← or → + C
  * 살각 던지기 : 근접해서 ← or → + D  

★ 특수기  

  * 유성락 : ←→ + B, B or D, D
  * 내려차기 : → + B
  * 편익의 진 : 배수의 진 중 BD  

★ 필살기  

  * 비연참 : ↓ 모으고 ↑ + B or D
  * 반월참 : ↓↙← + B or D
  * 공사진 : →↓↘ + B or D (공중 가능)  
┗ 천승참 : ↓ + D

  * 패기각 : ↓↓ + B or D  
┗ 더블 되차기 : 약 패기각 사용 중 →↓↘ + B or D  

┗ 천승참 : ↓ + D

  * 비상각 : 공중에서 ↓ + B or D  
┗ 공사진 : →↓↘ + B or D  

┗ 천승참 : ↓ + D

  * 배수의 진 : ↓↙← + A or C or 편익의 진 중 BD  
┗ 배수의 진 해제 : 배수의 진 중 ↓↙← + A  

★ 초필살기 (☆은 기게이지 소모 개수 표시)  

  * 봉황각(☆) : ↓↙←↙→ + B or D (공중 가능)
  * 봉황비천각(☆) : ↓↘→↓↘→ + B or D  
┗ 배수진 : ↓↙← + A or C

  * 봉황배수연무각(☆☆) : 배수의 진 중 ↓↙←↙→ + A or C
  * 제로거리 봉황각(☆☆☆) : 근접해서 ↓↙←↙→ + ABCD  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ ↓ + C ▶ ↑ + C ▶ C
  * 02 : B ▶ D ▶ ↘ + D ▶ D 
  * 03 : ← + B ▶ ↓ + B ▶ ↓ + D
  * 04 : ← + B ▶ C
  * 05 : ↘ + B ▶ B ▶ D
  * 06 : ↘ + B ▶ B ▶ ← + D ▶ CD (편익의 진으로 이행)
  * 07 : → + C ▶ C ▶ C ▶ D
  * 08 : → + C ▶ ↘ + C ▶ D
  * 09 : ← + C ▶ ← + C
  * 10 : D ▶ D ▶ D
  * 11 : D ▶ D ▶ → + D
  * 12 : D ▶ D ▶ ↓ + D
  * 13 : →→ + D ▶ D ▶ C
  * 14 : →→ + D ▶ D ▶ ↘ + C ▶ CD (편익의 진으로 이행)
  * 15 : B ▶ B ▶ → + B ▶ CD (편익의 진으로 이행)
  * 16 : D ▶ D ▶ CD (편익의 진으로 이행)
  * 17 : D ▶ ↘ + D ▶ CD (편익의 진으로 이행)
  * 18 : → + D ▶ → + D (이후 → + D로 반복. 도중 CD로 편익의 진으로 이행)
  * 19 : →→ + D ▶ CD (편익의 진으로 이행)
  * 20 : 배수의 진 중 ↓
  * 21 : 배수의 진 중 A (도발)
  * 22 : 배수의 진 중 B (배수의 진 해제)
  * 23 : 배수의 진 중 C (배수의 진 해제)
  * 24 : 배수의 진 중 D (배수의 진 해제)
  * 25 : 배수의 진 중 → + D ▶ CD (편익의 진으로 이행)
  * 26 : 편익의 진 중 →
  * 27 : 편익의 진 중 ←
  * 28 : 편익의 진 중 A (편익의 진 해제)
  * 29 : 편익의 진 중 B
  * 30 : 편익의 진 중 C
  * 31 : 편익의 진 중 D ▶ D (편익의 진 해제)
  * 32 : 편익의 진 중 AB (버튼 지속)
  * 33 : 편익의 진 중 ↓ + AB (버튼 지속)
