## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 오월우 : 근접해서 ← or → + B  

★ 특수기  

  * 일도 ・ 수무월 : ↘ + A
  * 일도 ・ 화잔 : ← + B
  * 일도 ・ 식 : 상대 다운시 ↘ + B   

★ 필살기  

  * 일도 ・ 롱 상단 : ↓↙← + A (모으기 가능)
  * 일도 ・ 롱 중단 : ↓↙← + B (모으기 가능)
  * 일도 ・ 롱 하단 : ↓↙← + C (모으기 가능)  
┣ 발도해제 : 일도 ・ 롱을 모으는 중 D  
┗ 재대기 : 일도 ・ 롱 발동 중 A or B or C

  * 일도 ・ 신월 : →↓↘ + A   
┗ 일도 ・ 쌍월 : 일도 ・ 신월 중 →↓↘ + B

  * 일도 ・ 신월 이 : →↓↘ + B
  * 일도 ・ 월영(●) : ↓↘→ + A or B (연속 3회 입력)
  * 대도 ・ 보월 : ←↙↓ + A or B or C  
┗ 대도 ・ 롱 : 대도 ・ 보월중 A or B or C  

★ 초오의  

  * 활살 ・ 십육야월화 : →←↙↓↘→ + AB  

★ 잠재오의  

  * 활살 ・ 난설월화 : →←↙↓↘→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 오월우 : 근접해서 ← or → + CD  

★ 특수기  

  * 일도 ・ 수무월 : ↘ + A
  * 일도 ・ 화잔 : ← + B
  * 일도 ・ 식 : 상대 다운시 ↘ + B   

★ 필살기  

  * 일도 ・ 롱 상단 : ↓↙← + A (모으기 가능)
  * 일도 ・ 롱 중단 : ↓↙← + B (모으기 가능)
  * 일도 ・ 롱 하단 : ↓↙← + C (모으기 가능)  
┣ 발도해제 : 일도 ・ 롱을 모으는 중 D  
┗ 재대기 : 일도 ・ 롱 발동 중 A or B or C

  * 일도 ・ 신월 : →↓↘ + A   
┗ 일도 ・ 쌍월 : 일도 ・ 신월 중 →↓↘ + B

  * 일도 ・ 신월 이 : →↓↘ + B
  * 일도 ・ 월영(●) : ↓↘→ + A or B (A는 연속 3회 입력, B는 연속 4회 입력)
  * 대도 ・ 보월 : ←↙↓ + A or B or C  
┗ 대도 ・ 롱 : 대도 ・ 보월중 A or B or C  

★ 초오의  

  * 활살 ・ 십육야월화 : →←↙↓↘→ + AB  

★ 잠재오의  

  * 활살 ・ 난설월화 : →←↙↓↘→ + B  

★ 난무오의  

  * 활살 ・ 백야 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, A, ↓↘→ + C  
┣ 띄우기 루트 : 히트 후 A, B, C,↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C,↓ + C, A, → + BC, A, B, ↓↘→ + B
