## 1. 하드웨어 ¶

  * 순차 접근 메모리 : SAM = Sequential Access Memory. 자기 테이프, 자기 디스크 같은 것들.
  * 임의 접근 메모리 : [RAM](RAM.md) = Random Access Memory.   

    * 읽기 전용 메모리 : [ROM](ROM.md) = Read Only Memory
    * 읽기 쓰기 메모리 : [RWM](RAM.md) = Read Write Memory

## 4. [애니메이션](%EC%95%A0%EB%8B%88%EB%A9%94%EC%9D%B4%EC%85%98.md) [TheBig-O](The%20Big-O.md)의 메모리 ¶

40년전 모종의 사건으로 기억을 잃어버린 [패러다임시티](%ED%8C%A8%EB%9F%AC%EB%8B%A4%EC%9E%84%20%EC%8B%9C%ED%8B%B0.md)의 모든 사람들과
안드로이드에게서 종종 깨진 조각처럼 떠오르는 기억들을 일컫는 말. [로저스미스](%EB%A1%9C%EC%A0%80%20%EC%8A%A4%EB%AF%B8%EC%8A%A4.md)는
[메가데우스](%EB%A9%94%EA%B0%80%EB%8D%B0%EC%9A%B0%EC%8A%A4.md)인
[Big-o](Big-O%28The%20Big-O%29.md)를 조종할 수 있는 메모리가, 집사인 [노먼버그](%EB%85%B8%EB%A8%BC%20%EB%B2%84%EA%B7%B8.md)는 [로저스미스](%EB%A1%9C%EC%A0%80%20%EC%8A%A4%EB%AF%B8%EC%8A%A4.md)가 옛날부터 그의 주인이라는 것과
Big-O를 수리할 수 있는 기억이 메모리라는 식이다. 작중 각종 음모와 죽음, 사건들의 중심이며 본 작품의 메인 테마이기도 하다.

  

**주의! 내용 누설이 있습니다.**  
  
본 항목과 하위 항목 또는 이 틀 아래에 있는 내용은 **작품의 줄거리나 결말, 반전 요소를 설명하고 있습니다.**  
작품의 내용 누설을 원하지 않는다면 이 문서를 닫아 주세요.

  

역설적이게도 40년전 기억은 애초에 존재하지도 않았고, 빅오의 모든 등장인물은 PD인 [엔젤](%EC%97%94%EC%A0%A4.md)
혹은 더 큰 존재에 의해 꾸며진 **무대 또는 방송의 등장인물**에 지나지 않았다. 단편적으로 기억나는 메모리는 인물들에게 주어진 배역,
혹은 촬영된 내용. 리셋되기 전 옛 다른 시놉시스(?)에서 있었던 일 등 복합적인 것들이다. 최종화에서 보면 방송장비 앞에서 우는
[엔젤](%EC%97%94%EC%A0%A4.md)과 그녀에게 나타난 [로저스미스](%EB%A1%9C%EC%A0%80%20%EC%8A%A4%EB%AF%B8%EC%8A%A4.md)와 [R. 도로시 웨인라이트](/
wiki/R.%20%EB%8F%84%EB%A1%9C%EC%8B%9C%20%EC%9B%A8%EC%9D%B8%EB%9D%BC%EC%9D%B4%E
D%8A%B8)가 나오는데, 벽에는 **빅오의 홍보포스터**같은 것도 붙어 있었다. 결국 엔젤이 <del>연재중단</del> 새로운 빅오를
시작할 때마다 루프처럼 사람들은 기억을 잃고 새로운 패러다임 시티에서 새로운 빅오를 촬영하는 것을 무한 반복하고 있었던 것이다.

  

굳이 따지자면 [엔젤](%EC%97%94%EC%A0%A4.md)이 메모리 그 자체이다.
