## 1. 개요 ¶

[독일](%EB%8F%85%EC%9D%BC.md)의 자동차 회사
[아우디](%EC%95%84%EC%9A%B0%EB%94%94.md)에서 생산중인 중형 세단/스테이션 왜건.`[1]`

  

경쟁 차종으로는 [메르세데스-벤츠 E클래스](%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0%20E%ED%81%B4%EB%9E%98%EC%8A%A4.md), [BMW5시리즈](BMW%205%EC%8B%9C%EB%A6%AC%EC%A6%88.md), [재규어XF](%EC%9E%AC%EA%B7%9C%EC%96%B4%20XF.md), [현대제네시스](%ED%98%84%EB%8C%80%20%EC%A0%9C%EB%84%A4%EC%8B%9C%EC%8A%A4.md), [렉서스ES](%EB%A0%89%EC%84%9C%EC%8A%A4%20ES.md), [렉서스GS](%EB%A0%89%EC%84%9C%EC%8A%A4%20GS.md) 등이 있다.

## 2. 특징 ¶

1968년부터 생산된 『아우디 100』이 A6의 전신인데, 인기리에 4세대 모델까지 나오다가 1994년에 페이스리프트를 거치면서 아우디의
새로운 작명법에 따라 A6가 되었다.

  

  
1986년에 아우디 100이 보여준 [스키점프대 오르는 광고](http://youtu.be/79R2L4aws00)를 2005년에 A6로
재현한 광고가 유명하다.

  

A6 3세대 모델같은 경우엔 2005~06년에 각종 언론에서 최고의 자동차로 선정되는등 큰 주목을 받았다. 흠잡을데 없고 단정한
디자인(별명이 [아우토반](%EC%95%84%EC%9A%B0%ED%86%A0%EB%B0%98.md)의 공주님)과 성능으로 꽤 잘
팔렸지만 모델이 점점 늙어(...)가면서 세일즈가 떨어진 듯하다. 결국 가격 인하를 감행하여 막판 판매량이 올랐다.

  

2010년 12월에 4세대 모델이 공개되었으며 이전 모델보다 많이 젊어진 듯... 하이브리드 모델이 추가되었다. 김기태 기자의 오토뷰에서
리뷰에서 별5개 만점을 기록할 정도로 엄청난 호평을 받았다.(참고로
김기태<del>[쉐슬람](%EC%89%90%EC%8A%AC%EB%9E%8C.md)</del> 기자의 리뷰는 혹독하기로 유명하다.)

  

여하튼 국내에서 가장 흔하게 볼 수 있는 아우디 차량.

  

FF도 나오며, 예외 없이 세로배치하고 [CVT](CVT.md)로 메워 버린다.

## 3. 라인업 ¶

  * 휘발유  

    * **A6 2.0 TFSI quattro** : 2.0L 직렬 4기통 TFSI 엔진 / 220마력 / 풀타임 4륜구동(AWD)
    * **A6 2.0 Hybrid** : 2.0L 직렬 4기통 엔진 & 전기모터 / 211마력 / 전륜구동(FF) 
    * **A6 3.0 TFSI quattro** : 3.0L V6 TFSI 엔진 / 310마력 / 풀타임 4륜구동(AWD)
    * **S6 4.0 quattro** : 4.0L V8 트윈터보 엔진`[2]` / 420마력 / 풀타임 4륜구동(AWD) / A6의 고성능 모델 
    * **RS6 4.0 quattro**`[3]``[4]` : 4.0L V8 트윈터보 엔진 / 560마력 / 풀타임 4륜구동(AWD) / A6의 초고성능 모델  

  * 경유  

    * **A6 2.0 TDI Dynamic** : 2.0L 직렬 4기통 TDI 엔진 / 177마력 / 전륜구동(FF)
    * **A6 3.0 TDI quattro** : 3.0L V6 TDI 엔진 / 245마력 / 풀타임 4륜구동(AWD)
    * **A6 3.0 TDI quattro** : 3.0L V6 BI TDI 트윈터보엔진 / 313마력 / 풀타임 4륜구동(AWD)  
  
  

(기타)  

  * **A6L** : 롱 휠 베이스 버전으로 오직 중국에서만 판매중인 모델. 엔진종류등으로 구분하면 10가지 트림이 있다.
