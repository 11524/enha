### 1.2. 설명 ¶

전반적으로 노래가 어렵지 않은 노래로 구성되어 있는 [First Step](First%20Step.md)`[2]` 이나[ElectroEP](Electro%20EP.md)`[3]`에 비해서 세트 난이도가 Hard로 설정된
[HeartBeat](HeartBeat.md)와 더불어 Hard난이도를 양분하는 것 처럼 보이지만......

  

막상 Core Sound와 Heartbeat를 둘 다 플레이한 플레이어의 대부분은 Core Sound가 더 **어렵다고** 말한다.
**(....)**

  

일반적인 이유는 이렇다.

  

1\. **초보 입문곡**이 무투뿐인데다가 처음 플레이하는 유저들이 클리어하기 힘들든곡들이 너무많다.  

> 무투 다음으로 쉽다는 Para Q는 타임라인이 빠르고 노래가 Drum N Bass 라서 주 멜로디가 거의없어 노트가 참으로 엇박자가
많다. 당연히 처음 듣는 초보자가 바로 깰 수 있는 곡이 아니다. 그리고 다른 곡들은... [더 이상의 자세한 설명은 생략한다](%EB%8D%94%20%EC%9D%B4%EC%83%81%EC%9D%98%20%EC%9E%90%EC%84%B8%ED%95%9C%20%EC%84%A4%EB%AA%85%EC%9D%80%20%EC%83%9D%EB%9E%B5%ED%95%9C%EB%8B%A4.md).

  

2\. 스코어 어택을 위해서 **정확도**를 올리기가 힘들다.  

> Core Sound의 스코어 어택을 위해선 히든곡이 나오기 전의 MAX 수치를 95%이상으로 높여서(이는 매우 정확하게 연주해야
가능하다) 내게로 와 TP를 해야 된다. 그런데 히든곡을 위해서 쓰이는 스코어 어택 곡 셋 중 Landscape TP에서 정확도와 랭크를 다
깍아먹어서 히든곡 내게로 와 TP를 쉽게 플레이할 수 없기 때문이다. Landscape TP가 MAX 95% 이상이 나오더라도 Flea
TP에서 정확도와 랭크를 다 깎아먹고 SIN PP를 연주하기도 하니 말 다한거다.

  

3\. 초강력 지뢰곡 [JBG](JBG.md) TP.  

> [I want You](I%20Want%20You%28DJMAX%29.md), [Your OwnMiracle](Your%20Own%20Miracle.md) 따위는 명함도 내밀지 못한다. 처음 이 곡을 골랐다면 거의 100% 빨피
내지 폭사. 모두가 공인하는 테크니카의 더러운 채보 1순위.

  

**[안습](%EC%95%88%EC%8A%B5.md)**

  

첫 클리어를 위해선 **무투 / Color / Flea ▷ SIN** (연타가 약하다면 Color 대신 Para Q)  
스코어 어택으론 **Landscape / Para Q / Flea ▷ 내게로 와** 루트로 플레이.

  

지뢰곡이 많다는 악평 때문인지 [테크니카2](DJMAX%20TECHNIKA%202.md)에서는 아예 **삭제**되었다. <del>아
[망했어요](%EB%A7%9D%ED%96%88%EC%96%B4%EC%9A%94.md)</del> 다른 세트는 1곡정도만
삭제되거나했는데, 이 세트는 3곡`[4]`이나 짤려나가버려서 아마도 넣기 곤란했을것이다.
