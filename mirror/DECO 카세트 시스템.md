**DECO 카세트 시스템**이란, [데이터 이스트](%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%9D%B4%EC%8A%A4%ED%8A%B8.md)에서 제작한 기판이다. 카세트`[1]`를 사용하는 기판으로, 인텔 I 시리즈의 6MHZ 클럭의 메인 CPU가 달려 있었다. 사용기간은 1981년부터 1985년 동안 약 4년의 기간으로, 데이타 이스트 자사 제작의`[2]` 게임이 20개의 게임중 전부다.

## 성능 ¶

  

  * CPU : I8X41 (500 kHz)   

  * 보조 CPU : M6502 (750 kHz); M6502 (500 kHz);  

  * 음원 : AY-8910 x 2  

## 사용 게임 ¶

  * [볼더 데쉬](%EB%B3%BC%EB%8D%94%20%EB%8D%B0%EC%89%AC.md) (1985)
  * #부마존 (1983)
