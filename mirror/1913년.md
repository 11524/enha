## 2.1. 실제 ¶

  * [2월 2일](2%EC%9B%94%202%EC%9D%BC.md) : [티베트](%ED%8B%B0%EB%B2%A0%ED%8A%B8.md)와 [몽골](%EB%AA%BD%EA%B3%A8.md)이 [청](%EC%B2%AD.md)으로부터 독립하기 위해 서로 협력한다는 내용의 몽장 조약을 체결했다.
  * [2월 11일](2%EC%9B%94%2011%EC%9D%BC.md) : [가츠라 타로](%EA%B0%80%EC%B8%A0%EB%9D%BC%20%ED%83%80%EB%A1%9C.md) 내각이 붕괴했다. (민중봉기로 인함)
  * [4월 8일](4%EC%9B%94%208%EC%9D%BC.md) : 최초의 [유치원](%EC%9C%A0%EC%B9%98%EC%9B%90.md)인 [경성](%EA%B2%BD%EC%84%B1.md) 유치원이 설립되었다.`[3]`.
  * [5월 29일](5%EC%9B%94%2029%EC%9D%BC.md) : [이고르 스트라빈스키](%EC%9D%B4%EA%B3%A0%EB%A5%B4%20%EC%8A%A4%ED%8A%B8%EB%9D%BC%EB%B9%88%EC%8A%A4%ED%82%A4.md)의 [발레곡](%EB%B0%9C%EB%A0%88%EA%B3%A1.md) 〈[봄의 제전](%EB%B4%84%EC%9D%98%20%EC%A0%9C%EC%A0%84.md)〉이 초연되었다.
  * [6월 29일](6%EC%9B%94%2029%EC%9D%BC.md) : 제2차 [발칸 전쟁](%EB%B0%9C%EC%B9%B8%20%EC%A0%84%EC%9F%81.md)이 발발했다.
  * [8월 10일](8%EC%9B%94%2010%EC%9D%BC.md) : 제2차 [발칸 전쟁](%EB%B0%9C%EC%B9%B8%20%EC%A0%84%EC%9F%81.md)이 [불가리아](%EB%B6%88%EA%B0%80%EB%A6%AC%EC%95%84.md)의 패배로 끝났다. 그리고 [1년 후...](%EC%A0%9C1%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)
  * [8월 31일](8%EC%9B%94%2031%EC%9D%BC.md) : [네덜란드](%EB%84%A4%EB%8D%9C%EB%9E%80%EB%93%9C.md) [축구](%EC%B6%95%EA%B5%AC.md)팀 [아인트호벤](%EC%95%84%EC%9D%B8%ED%8A%B8%ED%98%B8%EB%B2%A4.md) 창단.
  * [10월 5일](10%EC%9B%94%205%EC%9D%BC.md) : [영국](%EC%98%81%EA%B5%AD.md)이 [키프로스](%ED%82%A4%ED%94%84%EB%A1%9C%EC%8A%A4.md)를 합병하면서 [프랑스](%ED%94%84%EB%9E%91%EC%8A%A4.md)와 함께 [오스만 제국](%EC%98%A4%EC%8A%A4%EB%A7%8C%20%EC%A0%9C%EA%B5%AD.md)에 전쟁을 선포했다.
  * [12월 1일](12%EC%9B%94%201%EC%9D%BC.md) : [헨리 포드](%ED%97%A8%EB%A6%AC%20%ED%8F%AC%EB%93%9C.md), [포드](%ED%8F%AC%EB%93%9C.md) [자동차](%EC%9E%90%EB%8F%99%EC%B0%A8.md)에 최초로 [컨베이어](%EC%BB%A8%EB%B2%A0%EC%9D%B4%EC%96%B4.md)를 도입했다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/1913%EB%85%84?action=edit&section=4

## 2.2. 가상 ¶

  * [1월 1일](1%EC%9B%94%201%EC%9D%BC.md) : 신시중앙박물관이 개관되었다. - [1904 대한민국](1904%20%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)
  * [4월 18일](4%EC%9B%94%2018%EC%9D%BC.md) : [한국](%ED%95%9C%EA%B5%AD.md)의 비밀요원들에 의해 [라스푸틴](%EB%9D%BC%EC%8A%A4%ED%91%B8%ED%8B%B4.md)이 암살되었다. - [1904 대한민국](1904%20%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)
  * [5월 3일](5%EC%9B%94%203%EC%9D%BC.md) : [러시아](%EB%9F%AC%EC%8B%9C%EC%95%84.md)가 라스푸틴 암살의 책임을 물어 [한국](%ED%95%9C%EA%B5%AD.md)과 국교를 단절했다. - [1904 대한민국](1904%20%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)  

  * 나폴레옹이 러시아와 4년을 더 싸운 뒤 [폴란드](%ED%8F%B4%EB%9E%80%EB%93%9C.md)와 [영일동맹](%EC%98%81%EC%9D%BC%EB%8F%99%EB%A7%B9.md) 간에 전쟁이 일어날 조짐을 보이는 보나파르트-3의 현재 연도.
  * [센트럼](%EC%84%BC%ED%8A%B8%EB%9F%BC.md)의 과학자들이 생물병기에 대한 예방책과 치료법을 개발하고 파괴된 세계로 나가기 시작했다. - [겁스 무한세계](%EA%B2%81%EC%8A%A4%20%EB%AC%B4%ED%95%9C%EC%84%B8%EA%B3%84.md)
