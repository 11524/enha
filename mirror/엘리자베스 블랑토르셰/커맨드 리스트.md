# 1. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 마니에르 : 근접해서 ← or → + C or D  

★ 특수기  

  * 피델리테 : → + A
  * 피에르테 : → + B  

★ 필살기  

  * 에탄셀 : ↓↘→ + A or C
  * 플라틴 미라지 : ↓↙← + B or D
  * 쿠드방 : →↓↘ + A or C  

★ 초필살기  

  * 그랑 라팔 : ↓↘→↘↓↙← + A or C
  * 시엘 에트왈 : →↘↓↙←→↘↓↙← + A or C (잡기)  

★ 리더 초필살기  

  * 노블 블랑 : ↓↘→↓↘→ + E

# 3. [KOF XIII](KOF%20XIII.md) ¶

※ EX 대응기(●)는 A or C로 나가는 기술은 AC로, B or D로 나가는 기술은 BD로 EX 필살기를 쓸 수 있다.

  

★ 기본잡기  

  * 마니에르 : 근접해서 ← or → + C or D  

★ 특수기  

  * 앙 쿠 도 피에 : → + B  

★ 필살기  

  * 에탄셀(●) : ↓↘→ + A or C
  * 레베리-스에테 : ↓↘→ + B or D
  * 레베리-프리에 : ↓↙← + B or D
  * 레베리-쥴레(●) : ↓↙← + A or C
  * 미스트랄(●) : →↘↓↙←→ + A or C (잡기)
  * 쿠드방(●) : →↓↘ + A or C  

★ 초필살기  

  * 노블 블랑(●) : ↓↘→↓↘→ + A or C
  * 그랑 라팔 : ↓↘→↘↓↙← + A or C  

★ NEO MAX 초필살기  

  * 에토와르 피란트 : ↓↙←↙↓↘→ + BD
