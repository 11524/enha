# 2. 패턴 ¶

아이스 큐브를 이용해 배리어를 형성하고, 빠른 기동력으로 상대를 공략하는 타입. 상대할 경우 일단 아이스 큐브들을 부수는 것이 최우선
과제이다. 어려운 보스는 아니다.

  

  * 아이스 브레스
전방에 아이스 큐브를 생성한다. 생성중에는 공격이 가능하지만, 생성된 후에는 아이스 큐브를 부숴야 공격할 수 있다. 아이스 큐브가 부서질
때마다 세로 1줄씩 아이스 큐브를 다시 생성한다. 바로 바로 생성하는 건 아니니 중간의 시간차를 이용해서 공격하거나, 아예 봄버 계열 등
큐브 너머를 공격하는 칩으로 공략하면 된다.

  

  * 아이스 큐브 밀어내기
부서지지 않은 아이스 큐브가 있을 경우, 발로 차서 밀어버린다. 아이스 큐브는 록맨이 맞거나 에리어 끝까지 가면 부서진다.

  

  * 아이스 타워 볼
전방 3칸 앞에 십자(十)모양으로 아이스 타워를 생성하는 공을 던진다. 피격당하면 얼어붙기 때문에 바로 아이스 큐브 밀기 공격이 추가로
들어온다.
