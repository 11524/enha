[리모컨](%ED%83%9C%EC%A7%84%EB%85%B8%EB%9E%98%EB%B0%A9/%EB%A6%AC%EB%AA%A8%EC%BB%A8.md)

  
[TJ미디어](%ED%83%9C%EC%A7%84%EB%85%B8%EB%9E%98%EB%B0%A9.md)가 개발,생산하고 있는
스피커이다.

TJ미디어는 [TKR시리즈](%ED%83%9C%EC%A7%84%EB%85%B8%EB%9E%98%EB%B0%A9/%EC%A7%88%EB%9F%ACTKR%EC%8B%9C%EB%A6%AC%EC%A6%88.md)때부터 노래방용 스피커를 개발, 생산하여 판매하고 있었다.

주로 10인치 우퍼와 8인치 우퍼를 장착한 스피커를 생산 하고 있다. [MR시리즈](%ED%83%9C%EC%A7%84%EB%85%B8%EB%9E%98%EB%B0%A9/%EC%A7%88%EB%9F%ACMR%EC%8B%9C%EB%A6%AC%EC%A6%88.md)때에는 서브우퍼
스피커도 판매 했었지만 2013년 현재 판매하고 있지는 않다.

구형제품군에 대해선 정보부족으로 인해 2013년 12월 현재 TJ미디어 사이트에 게시된 제품으로만 생성하였다.

# 1. TS-S100 ¶

  

![http://www.tjmedia.co.kr/images/products/speaker/con_TS_S100_img1.png?width=
515](http://www.tjmedia.co.kr/images/products/speaker/con_TS_S100_img1.png)

[[PNG external
image]](http://www.tjmedia.co.kr/images/products/speaker/con_TS_S100_img1.png)

  

정격 입력 150W 최대 입력 500W 스피커 이다. 10인치 우퍼 한개와 3인치 트위터 두개로 구성된 제품이다.

  

주파수 대역은 40~20000Hz이고 임피던스는 8옴이다.

  

우퍼 방열형 보이스 코일을 탑재했으며 트위터 자동 복귀형 회로를 적용 했다고 한다.

정격 입력 85W 최대 입력 250W 스피커 이다. 8인치 우퍼 한개와 3인치 트위터 두개로 구성된 제품이다.

저역 최적화 튜닝을 통해 풍부한 저음력을 재생한다고 한다.

# 4. TS-100CK ¶

  

![http://www.tjmedia.co.kr/images/products/speaker/TS-
S100CK_img.jpg?width=515](http://www.tjmedia.co.kr/images/products/speaker/TS-

[[JPG external image]](http://www.tjmedia.co.kr/images/products/speaker/TS-

정격입력 150W 최대 입력 500W 간이 방자형 스피커 이다. 10인치 우퍼 한개와 3인치 트위터 두개로 구성된 제품이다.

  

주파수 대역은 35~20000HZ이고 임피던스는 8옴이다.

자동 복귀형 회로 채용으로 하울링과 과대 입력으로부터 스피커를 보호한다고 한다.
