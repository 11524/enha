# 1. [아랑 MOW](%EC%95%84%EB%9E%91%20MOW.md) ¶

  

★ 기본잡기  

  * 그리폰 타워 : 근접해서 ← or → + C  
┗ 그리폴 : ↘↘ + C

  * 그리허그 : 근접해서 ← or → + D

  * 그리드로 슈퍼 킥 페인트 : → + AC
  * 다이달로스 어택 페인트 : ↓ + AC  

★ TOP기  

  * 그리드로 슈퍼 킥 : CD  

★ 필살기  

  * 액티브 튀폰 : →↓↘ + A or C
  * 포세이돈 웨이브 : ←↓↙ + A or C
  * 올림포스 오버 : ↓↙← + B or D
  * 저스티스 허리케인 : 레버 1회전 + A or C (잡기)
  * 헤라클레스 스루 : ←↙↓↘→ + B (잡기)
  * 이카로스 크래쉬 : 공중 근접해서 ↓↘→ + A or C  

★ 초필살기  

  * 빅 폴 그리폰 : 레버 2회전 + A (잡기)
  * 다이달로스 어택 : ↓↘→↓↘→ + B  

★ 잠재능력  

  * 빅 폴 그리폰 : 레버 2회전 + C (잡기)
  * 다이달로스 어택 : ↓↘→↓↘→ + D

# 3. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 그리폰 타워 : 근접해서 ← or → + C or D  

★ 특수기  

  * 상단회피 공격 : ↘ + C
  * 그리드로 슈퍼 킥 : ↘ + D  

★ 필살기  

  * 액티브 튀폰 : →↓↘ + A or C
  * 포세이돈 웨이브 : ←↓↙ + A or C  
┗ 브레이킹 : 강 포세이돈 웨이브 중 AB

  * 올림포스 오버 : ↓↙← + B or D (공중 가능)  
┗ 브레이킹 : 강 올림포스 오버 중 AB

  * 저스티스 허리케인 : →↘↓↙← + A or C (잡기)
  * 헤라클레스 스루 : ←↙↓↘→ + B (잡기)
  * 이카로스 크래쉬 : 공중 근접해서 ↓↘→ + A  

★ 초필살기  

  * 다이달로스 어택 : ↓↘→↓↘→ + B or D
  * 빅 폴 그리폰 : →↘↓↙←→↘↓↙← + A or C (잡기)  

★ 리더 초필살기  

  * 빅 폴 그리폰 : →↘↓↙←→↘↓↙← + E (잡기)
