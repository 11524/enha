## 1. 사양 ¶

  * CPU : [히타치](%ED%9E%88%ED%83%80%EC%B9%98.md) SH-3 133Mhz`[1]`
  * GPU : Altera Cyclone FPGA
  * 사운드 : [야마하](%EC%95%BC%EB%A7%88%ED%95%98.md) YMZ770C-F
  * RAM : 32MB * 2 + 8MB(CV1000D는 32MB * 2 + 16MB)
  * 게임 저장 매체 : 128MB 플래시 메모리(그래픽 데이터), 4MB * 2 플래시 메모리(사운드 데이터), 2MB(CV1000D는 4MB) 플래시 메모리(프로그램 데이터)  

[벌레공주님](%EB%B2%8C%EB%A0%88%EA%B3%B5%EC%A3%BC%EB%8B%98.md)을 시작으로
[MAME](MAME.md)로 구현되었다. 2014년 현재 덤프된 게임은 [데스 스마일즈 메가 블랙레이블](%EB%8D%B0%EC%8A%A4%20%EC%8A%A4%EB%A7%88%EC%9D%BC%EC%A6%88%20%EB%A9%94%EA%B0%80%20%EB%B8%94%EB%9E%99%EB%A0%88%EC%9D%B4%EB%B8%94.md)까지
