  * 본 문서에서는 [아크 시스템 웍스](%EC%95%84%ED%81%AC%20%EC%8B%9C%EC%8A%A4%ED%85%9C%20%EC%9B%8D%EC%8A%A4.md) 격투게임 [길티기어 시리즈](%EA%B8%B8%ED%8B%B0%EA%B8%B0%EC%96%B4%20%EC%8B%9C%EB%A6%AC%EC%A6%88.md)의 캐릭터 [자파](%EC%9E%90%ED%8C%8C.md)가 사용하는 기술 커맨드에 대해 설명하고 있으며 1P 시작 위치 기준입니다.   * 각 기술명 옆에 붙어있는 표시의 의미는 다음과 같습니다.  

    * (○)는 유령을 빙의한 상태에서 사용 가능합니다.
    * (□)는 검을 빙의한 상태에서 사용 가능합니다.
    * (△)는 개를 빙의한 상태에서 사용 가능합니다.
    * (☆)는 [라오우](%EB%9D%BC%EC%98%A4%EC%9A%B0.md)를 빙의한 상태에서 사용 가능합니다.

## 2. 기술명 ¶

★ 필살기  

  * 무빙의 상태(無憑依状態 / No Summon)  

    * 안녕하세요 세마리 지네(こんにちは三匹のムカデ / Hello, Three Centipedes)
  * 삼둥이(三つ子 / Triplets)  

    * 그대로 돌아가서 오지 말아주세요(そのまま帰ってこないで下さい / Please Don't Come Back)
  * 검(剣 / Sword)  

    * 아프겠어, 라기보다 아파(痛そう、っていうか痛い / This Has Gotta Hurt)
    * 떨어져 주세요(落ちといてください / Please Fall)
    * 가까이 오면 베어 버립니다(近づくと逝きます / Come Close, and I'll Kill You)
    * 예, 지나갑니다(はい、通ります / Coming Through)
  * 개(犬 / Sleeping Dog)  

    * 개 공격(犬攻撃 / Awaken Dog)
  * 라오우(羅王 / Raoh)  

    * 다크니스 안셈(ダークネスアンセム / Darkness Anthem)
    * 라스트 에드가이(ラストエドガイ / Last Edguy)
  * 이용은 계획적으로(ご利用は計画的に) - EX 전용기
  * 방금 전까지 있었다구!(さっきまで居たんだよ！) - EX 전용기  

★ 포스 브레이크  

  * 예의범절은 여기서부터(礼儀作法はここから / Etiquette Starts Here)
  * 불행 중 다행(不幸中の幸い) - EX 전용기  

★ 각성필살기  

  * 태어난다!(産まれる！！ / Birth!!)
  * 벨로우즈 마리스(ベロウズマリス / Bellows Malice)
  * 너클 카이저(ナックルカイザー) - EX 전용기  

★ 일격필살기  

  * 무서워요~(こわいよー / I'm Scared...)
