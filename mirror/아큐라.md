## 1.2. [혼다](%ED%98%BC%EB%8B%A4.md)의 자회사 Acura ¶

[혼다](HONDA.md)의 북미 시장 전용 럭셔리 브랜드다.
[토요타](%ED%86%A0%EC%9A%94%ED%83%80.md)와
[렉서스](%EB%A0%89%EC%84%9C%EC%8A%A4.md)와의 관계를 생각하면 이해하기 쉽다.  
혼다 브랜드의 중대형차를 브랜드 이름만 바꿔서 북미시장에 내놓았다. 일본차가 독립 럭셔리 브랜드로 북미에 진출한 사례로는 렉서스보다 더
빨랐다. <del>렉서스에 비해 존재감이 없는건 일단 무시하자</del>

  

초창기에는 혼다 레전드나 인스파이어 같은 중~대형차들을 들여왔으나 후에는 인테그라나 NSX같은 스포츠카 역시 수입해왔다.  
2010년대 초까지 혼다 차에서 뱃지만 바꾼 모델이 대다수여서 브랜드 파워가 매우 낮았다. 2013년부터 혼다의 플랫폼을 사용하더라도 북미
전용으로 따로 개발된 차들로 라인업을 재편성하며 재평가받는 중.

  

최근 들어 거의 전모델에 SH-AWD라는 4륜구동 시스템을 채용하고 있다.

### 1.2.1. 생산 차량 ¶

[2015년](2015%EB%85%84.md) 현재 존재하는 라인업 우선으로 서술.  
ILX -> [혼다 시빅](%ED%98%BC%EB%8B%A4%20%EC%8B%9C%EB%B9%85.md)의 변형 모델로, 구 CSX와
TSX의 통합 후속형이다.  
TLX -> [혼다 어코드](%ED%98%BC%EB%8B%A4%20%EC%96%B4%EC%BD%94%EB%93%9C.md)의 변형
모델로, 구 TSX와 TL의 통합 후속형이다.  
RLX -> [혼다 레전드](%ED%98%BC%EB%8B%A4%20%EB%A0%88%EC%A0%84%EB%93%9C.md)의 북미형
모델. 원래 이름이 RL이었으나 2013년부터 RLX로 바뀌었다.  
RDX -> [혼다 CR-V](%ED%98%BC%EB%8B%A4%20CR-V.md)의 변형 모델.  
MDX -> SUV형 차량. 이 쪽은 역으로 일본 혼다가 수입해가고 있다.

  

아래부터는 단종 차량  
CL -> 혼다 레전드 쿠페  
TL -> 혼다 인스파이어 (2003년 이후 모델 인스파이어는 북미에서 혼다 어코드로 변경)  
Vigor -> 혼다 비거  
Legend 혹은 RL -> 혼다 레전드  
인테그라 혹은 RSX -> 혼다 인테그라  
CSX, EL -> 혼다 시빅의 변형 모델 (캐나다 전용)  
TSX -> 혼다 어코드 유로 (아시아, 유럽 전용 어코드)  
NSX -> 혼다 NSX  
SLX -> [이스즈](%EC%9D%B4%EC%8A%A4%EC%A6%88.md) 트루퍼 기반의 프레임 SUV. 저조한 판매로 인해
단종되었다.  
ZDX -> MDX 기반의 쿠페형 SUV. [BMW X6](BMW%20X6.md)의 성공을 보고 투입한 모델이었으나 대실패하고 조기
단종되었다.
