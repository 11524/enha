# 1. 개요 ¶

[제1차 세계대전](%EC%A0%9C1%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)이
발발한 비극적인 해이다. 이를 기점으로 모든 것이 좋아질거라던 [벨에포크](%EB%B2%A8%20%EC%97%90%ED%8F%AC%ED%81%AC.md) 시대가 막을 내린다.
[조선총독부](%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80.md) 통치 하의
[식민지](%EC%8B%9D%EB%AF%BC%EC%A7%80.md) [조선](%EC%A1%B0%EC%84%A0.md)에서는
[일제](%EC%9D%BC%EC%A0%9C.md)가 [부군면통폐합](%EB%B6%80%EA%B5%B0%EB%A9%B4%20%ED%86%B5%ED%8F%90%ED%95%A9.md)을 실시하여 여러
군과 면이 통폐합되어 사라졌으며, 현재 [대한민국](%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)의
[행정구역](%ED%96%89%EC%A0%95%EA%B5%AC%EC%97%AD.md) 명칭과 영역의 기반이 되었다.

)]

## 2.1. 실제 ¶

  * [5월 29일](5%EC%9B%94%2029%EC%9D%BC.md) : [캐나다](%EC%BA%90%EB%82%98%EB%8B%A4.md) 퀘벡에서 출발해 [영국](%EC%98%81%EA%B5%AD.md) 리버풀로 도착할 예정이던 임프레스 오브 아일랜드(Empress of Ireland)호가 세인트로렌스강 하구에서 Storstad호와 충돌하여 침몰했다. 탑승 인원 1477명 중 1012명이 사망했다.
  * [6월 28일](6%EC%9B%94%2028%EC%9D%BC.md) : [페르디난트](%ED%8E%98%EB%A5%B4%EB%94%94%EB%82%9C%ED%8A%B8.md) 황태자 부부가 암살되었다. 이 사건은 [제1차 세계대전](%EC%A0%9C1%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)의 도화선이 되었다.
  * [7월 28일](7%EC%9B%94%2028%EC%9D%BC.md) : [오스트리아-헝가리 제국](%EC%98%A4%EC%8A%A4%ED%8A%B8%EB%A6%AC%EC%95%84-%ED%97%9D%EA%B0%80%EB%A6%AC%20%EC%A0%9C%EA%B5%AD.md)이 [세르비아](%EC%84%B8%EB%A5%B4%EB%B9%84%EC%95%84.md)에 [선전포고](%EC%84%A0%EC%A0%84%ED%8F%AC%EA%B3%A0.md)를 했다.
  * [8월 1일](8%EC%9B%94%201%EC%9D%BC.md) : [독일 제국](%EB%8F%85%EC%9D%BC%20%EC%A0%9C%EA%B5%AD.md)이 [러시아](%EB%9F%AC%EC%8B%9C%EC%95%84.md)에 선전포고를 했다.
  * [8월 3일](8%EC%9B%94%203%EC%9D%BC.md) : [독일 제국](%EB%8F%85%EC%9D%BC%20%EC%A0%9C%EA%B5%AD.md)이 [프랑스](%ED%94%84%EB%9E%91%EC%8A%A4.md)에 선전포고를 했다.
  * [8월 4일](8%EC%9B%94%204%EC%9D%BC.md) : [영국](%EC%98%81%EA%B5%AD.md)이 [독일 제국](%EB%8F%85%EC%9D%BC%20%EC%A0%9C%EA%B5%AD.md)에 선전포고를 했다.
  * [8월 15일](8%EC%9B%94%2015%EC%9D%BC.md) : [파나마 운하](%ED%8C%8C%EB%82%98%EB%A7%88%20%EC%9A%B4%ED%95%98.md)가 개통되었다.
  * [8월 16일](8%EC%9B%94%2016%EC%9D%BC.md) : [경원선](%EA%B2%BD%EC%9B%90%EC%84%A0.md)이 개통되었다.
  * [8월 23일](8%EC%9B%94%2023%EC%9D%BC.md) : [일본 제국](%EC%9D%BC%EB%B3%B8%20%EC%A0%9C%EA%B5%AD.md)이 [독일 제국](%EB%8F%85%EC%9D%BC%20%EC%A0%9C%EA%B5%AD.md)에 선전포고를 했다.
  * [8월 26일](8%EC%9B%94%2026%EC%9D%BC.md) : 독일과 러시아의 [탄넨베르크 전투](%ED%83%84%EB%84%A8%EB%B2%A0%EB%A5%B4%ED%81%AC%20%EC%A0%84%ED%88%AC.md)가 일어났다.
  * [8월 30일](8%EC%9B%94%2030%EC%9D%BC.md) : 독일, [파리](%ED%8C%8C%EB%A6%AC%28%EB%8F%84%EC%8B%9C%29.md)에 최초의 [항공기](%ED%95%AD%EA%B3%B5%EA%B8%B0.md) [공습](%EA%B3%B5%EC%8A%B5.md) [폭격](%ED%8F%AD%EA%B2%A9.md). 
  * [9월 1일](9%EC%9B%94%201%EC%9D%BC.md) : [미국](%EB%AF%B8%EA%B5%AD.md) 신시내티 동물원에 남아 있던 마지막 [여행비둘기](%EC%97%AC%ED%96%89%EB%B9%84%EB%91%98%EA%B8%B0.md)가 오후 1시에 죽으면서 멸종되었다. 
  * [11월 2일](11%EC%9B%94%202%EC%9D%BC.md) : [러시아](%EB%9F%AC%EC%8B%9C%EC%95%84.md)가 [오스만 제국](%EC%98%A4%EC%8A%A4%EB%A7%8C%20%EC%A0%9C%EA%B5%AD.md)에 선전포고를 했다.
  * [11월](11%EC%9B%94.md) : [막스 폰 라우에](%EB%A7%89%EC%8A%A4%20%ED%8F%B0%20%EB%9D%BC%EC%9A%B0%EC%97%90.md) [노벨 물리학상](%EB%85%B8%EB%B2%A8%20%EB%AC%BC%EB%A6%AC%ED%95%99%EC%83%81.md) 수상. ([X선](X%EC%84%A0.md) [회절](%ED%9A%8C%EC%A0%88.md))
  * [12월 25일](%EC%84%B1%ED%83%84%EC%A0%88.md) : 인류전쟁사 최대의 기적인 [크리스마스 휴전](%ED%81%AC%EB%A6%AC%EC%8A%A4%EB%A7%88%EC%8A%A4%20%ED%9C%B4%EC%A0%84.md)이 비공식적으로 채결되었다.
