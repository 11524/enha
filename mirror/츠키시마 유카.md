## 2.1. 자기 소개 ¶

유카 : 홋! 얏! 탓!  
유카 : 어라, 못 보던 얼굴인데─?  
유카 : …… 댄스에 흥미가 있어?  
유카 : 에? 스쿨 아이돌??  
유카 : 들어본 적 없는 장르인걸─  
유카 : 내가 하는 건 말이야, 힙합이라든지, 하우스라든지, 락킹이라든지… 뭐, 멋있는 계열이 좋다고 할까!  
유카 : 세계적인 댄서를 목표로 하고 있으니까, 어떤 댄스라도 춰보이겠어!  
유카 : …그럼, 오늘은 여기까지일려나─ 응, 그래도 좋은 느낌!  
유카 : 나는 츠키시마 유카. 항상 여기 있으니까, 괜찮다면 놀러와!  
유카 : 꼭이야─!
