## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 무명 : 근접해서 ← or → + CD  

★ 특수기  

  * 무명 : 상대 다운시 ↘ + B  

★ 필살기  

  * 무명 (일)(●) : ↓↘→ + A or B
  * 무명 (이) : →↓↘ + A or B 
  * 무명 (삼) : ↓↙← + B
  * 무명 (사) : →↓↘ + C
  * 무명 (오) : →↘↓↙← + C  
┗ 무명 (오)의 추(●) : 무명 (오) or 무명 (극) 발동 중 ←↙↓↘→ C  

★ 초오의  

  * 무명 (절) : ↓↘→↓↘→ + AB (모으기 가능)  

★ 잠재오의  

  * 무명 (극) : →↘↓↙←→↘↓↙← + B  
┗ 무명 (극)의 추 : 무명 (극) 발동 중 ←↙↓↘→ + AB  

★ 난무오의  

  * 무명 (멸) : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C,↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C,↓ + C,↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C,↓ + C, BC, ↓↘→ + B
