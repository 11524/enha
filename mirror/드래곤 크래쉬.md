## 1.1. 드래곤 크래쉬 ¶

ドラゴンクラッシュ / Dragon Crash. [영상](https://www.youtube.com/watch?v=TVi42WZSB50)
[게임판](http://www.youtube.com/watch?v=NujYzicDtWY)  
위력: 34 / 소비TP : 28 / 속성 林 / 특성 : 슛 체인 / 성장속도 : 빠름

  

1편부터 등장. 주 사용자는 [소메오카류고](%EC%86%8C%EB%A9%94%EC%98%A4%EC%B9%B4%20%EB%A5%98%EA%B3%A0.md). 용의 기운을
머금은 슛을 날리는 단순한 기술. 슛을 날리면 뒤에서 푸른 [용](%EC%9A%A9.md)이 공을 뒤따라 날아가는 연출.

  

애니에서는 [고엔지 슈야](%EA%B3%A0%EC%97%94%EC%A7%80%20%EC%8A%88%EC%95%BC.md)의 활약을 본
팀원들이 고엔지 타령을 해대자, 프라이드에 상처를 입은 소메오카가 말그대로 뼈를 깎고 피를 토하는 미칠듯한 수련 끝에 간신히 습득해냈다.
[오컬트 중학교](%EC%98%A4%EC%BB%AC%ED%8A%B8%20%EC%A4%91%ED%95%99%EA%B5%90.md)전에서
처음 사용하여 첫 득점을 올렸다. 그후에도 자주 등장했다.

  

근데 게임판에서는 [이나즈마 일레븐(축구팀)](%EC%9D%B4%EB%82%98%EC%A6%88%EB%A7%88%20%EC%9D%BC%EB%A0%88%EB%B8%90%28%EC%B6%95%EA%B5%AC%ED%8C%80%29.md)의 멤버인 아이다 치카라 아저씨가
소메오카에게 전수해준다. [코믹스판](%EC%BD%94%EB%AF%B9%EC%8A%A4%ED%8C%90.md)에서는 오컬트전에서 한계를
타파하라는 엔도의 외침에 각성하면서 처음으로 발현되며, 기술 시전 시에 나타나는 용의 모습이 게임/애니와 다르다.

## 1.2. 와이번 크래쉬 ¶

ワイバーンクラッシュ / Wyvern Crash. [영상](https://www.youtube.com/watch?v=sbEJbOhiwKQ)
[게임판](https://www.youtube.com/watch?v=Z2qYbYWtqmo)  
위력 : 80 / 소비TP : 49 / 속성 林 / 특성 없음 / 성장속도 : 빠름

  

2편부터 등장. 드래곤 크래쉬의 강화판. 주 사용자는 [소메오카류고](%EC%86%8C%EB%A9%94%EC%98%A4%EC%B9%B4%20%EB%A5%98%EA%B3%A0.md). 소메오카가 공을
위로 차올리는 순간, 지면에서 푸른 [와이번](%EC%99%80%EC%9D%B4%EB%B2%88.md)이 상승. 와이번이 공을 물고
다시 하강하면 소메오카가 그대로 슛을 날린다. (사실 와이번이라기 보다는
[드래곤](%EB%93%9C%EB%9E%98%EA%B3%A4.md)에 더 가깝다.)

  

홋카이도에서 [후부키 시로](%ED%9B%84%EB%B6%80%ED%82%A4%20%EC%8B%9C%EB%A1%9C.md)의 도움으로
어떻게 습득하여 [제미니 스톰](%EC%A0%9C%EB%AF%B8%EB%8B%88%20%EC%8A%A4%ED%86%B0.md)과의
3번째 시합에서 처음으로 사용했다. 하지만 [진 제국학원](%EC%A7%84%20%EC%A0%9C%EA%B5%AD%20%ED%95%99%EC%9B%90.md)과의 싸움에서 소메오카가
[리타이어](%EB%A6%AC%ED%83%80%EC%9D%B4%EC%96%B4.md) 해버려 2기 내의 사용 횟수는 그다지 많지
않았다.

  

반면 소메오카가 [다크엠퍼러즈](%EB%8B%A4%ED%81%AC%20%EC%97%A0%ED%8D%BC%EB%9F%AC%EC%A6%88.md)에 들어갔을
때는 후부키를 가뿐히 씹어제끼고, [타치무카이 유우키](%ED%83%80%EC%B9%98%EB%AC%B4%EC%B9%B4%EC%9D%B4%20%EC%9C%A0%EC%9A%B0%ED%82%A4.md)의 [무한 더핸드](%EB%AC%B4%ED%95%9C%20%EB%8D%94%20%ED%95%B8%EB%93%9C.md)마저 개발라버리는 [충격과공포](%EC%B6%A9%EA%B2%A9%EA%B3%BC%20%EA%B3%B5%ED%8F%AC.md)를 선사했다.

  

3기에서는 일본 대표 선발전에서 V2로 진화해 등장한다. 하지만 어째선가 무한 더 핸드에 막혀버린다. 약해졌다?

## 2.1. 드래곤 토네이도 ¶

ドラゴントルネード / Dragon Tornado [영상](https://www.youtube.com/watch?v=0rdl9SrYLu0)
[게임판](https://www.youtube.com/watch?v=B6Ot1GO_Vjo&spfreload=10)  
위력 : 73 / 소비TP : 46 / 속성 : 火 / 특성 없음 / 성장속도 : 빠름

  

1편부터 등장. 사용자는 [소메오카류고](%EC%86%8C%EB%A9%94%EC%98%A4%EC%B9%B4%20%EB%A5%98%EA%B3%A0.md), 보조는 [고엔지슈야](%EA%B3%A0%EC%97%94%EC%A7%80%20%EC%8A%88%EC%95%BC.md). 비교적 게임 초반에 등장하는
기술로 소메오카가 '드래곤 크래쉬'로 공을 띄우면 공중에서 고엔지가 [파이어 토네이도](%ED%8C%8C%EC%9D%B4%EC%96%B4%20%ED%86%A0%EB%84%A4%EC%9D%B4%EB%8F%84.md)로 슛을 날리는 2인 합체기.`[1]`

  

드래곤 크래쉬는 숲속성인데 마무리가 파이어 토네이도라서 그런지 속성은 화속성. 발동은 소메오카 쪽에서 발동한다. 소메오카가 숲속성이라 속성
보정을 못 받는건 아쉽지만 TP 소모도 높지 않고 위력도 괜찮아서 중후반까지도 활약할 수 있는 슛기술. 단 최후반부에 쓰기에는 다소 위력이
모자라다.

  

애니메이션에서도 꽤 초반에 등장해서 라이몬의 주력 득점원 중 하나가 된다. 물론 애니메이션에서도 후반에 가면 각종 필살기가 등장해 조금 빛이
바래지는 편이지만.

## 2.2. 와이번 블리자드 ¶

ワイバーンブリザード / Wyvern Blizzard.
[영상](https://www.youtube.com/watch?v=LE_z9_hN26Q)
[게임판](https://www.youtube.com/watch?v=uOeHRmbcRPw)  
위력 : 93 / 소비TP : 55 / 속성 風 / 특성 없음 / 성장속도 : 느림

  

2편부터 등장. 사용자는 소메오카 류고, 보조 역할은 [후부키시로](%ED%9B%84%EB%B6%80%ED%82%A4%20%EC%8B%9C%EB%A1%9C.md). 소메오카가 '와이번 크래쉬'로
후부키에게 패스하면 후부키가 그대로 [이터널 블리자드](%EC%9D%B4%ED%84%B0%EB%84%90%20%EB%B8%94%EB%A6%AC%EC%9E%90%EB%93%9C.md)로 연결해 슛을 날리는 2인 합체기.

  

애니에서는 2기 39화에서 후부키와 소메오카가 새로 만든 필살기라면서 연습 시합에서 처음으로 사용. 하지만 당시 소메오카의 다리 부상이
심각했던지라 결국 팀에서 빠지면서 실전에서 제대로 활약도 못해보고 묻혀버린 비운의 필살기.

## 3.1. 코로드라 슛 ¶

コロドラシュート. [게임판](http://www.youtube.com/watch?v=Pniq-OvTmwo)  
위력 : 28 / 소비TP : 25 / 속성 風 / 특성 없음 / 성장속도 : 빠름 / 습득조건 : S사이즈

  

2편부터 등장. 드래곤 크래쉬의 약체화(?) 버전. 애니메이션에선 미등장. 슛 날리는 포즈를 잡으면 뒤에서 검은색의 꼬마용(코로드라)이
나타나 윙크 한번 해주고, 그 뒤 슛을 날리면 번개를 두른 공과 함께 용이 날아간다. 전체적인 연출은 드래곤 크래쉬와 비슷. 번개 계열
슛이라서 속성은 風.
