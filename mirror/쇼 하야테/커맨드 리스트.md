# 2. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 급소 질러 떨구기 : 근접해서 ← or → + C or D  

★ 특수기  

  * 비곤 저족 : 근거리에서 → + C
  * 비곤 가사 베기 : 원거리에서 → + C
  * 비곤 다리 후리기 : 근거리에서 ↘ + C
  * 비곤 사선 베기 : 원거리에서 ↘ + C
  * 비곤 떨궈 베기 : 공중에서 ← + C
  * 팔꿈치 맞추기 : ← + C  

★ 필살기  

  * 비곤 투술 : ↓↘→ + A or C (공중 사용 가능)
  * 강렬참 : ↓↙← + C (연속 2회 입력)
  * 비조 차기 : ↓↙← + B or D (공중 사용 가능)
  * 비천 승왕각 : →↓↘ + B or D
  * 열풍살 : 공중에서 ↓ + A or C  

★ 초필살기  

  * 염도 비곤 : ↓↘→↓↘→ + A or C
  * 환영 비곤 : ↓↙←↓↙← + A or C  

★ 리더 초필살기  

  * 오의! 무쌍 난무 : ↓↘→↘↓↙← or →↘↓↙←→ + E
