## 프로필 ¶

  * 1985년 11월 20일(윈도우즈 1.0 발매일)
  * 신장 170cm, 체중 비밀
  * 일본인 아버지([마도베 나나미](%EB%A7%88%EB%8F%84%EB%B2%A0%20%EB%82%98%EB%82%98%EB%AF%B8.md)의 숙부)와 프랑스인 어머니 사이에서 태어남
  * 아버지와 어머니외에 10살 위`[1]`의 오빠(이름은 클로드 마도베)가 있음.
  * 공대 석사 출신. 현재 클라우드 개발자 3년차.
  * [트위터](http://twitter.com/Claudia_Azure)

## 개요 ¶

[Microsoft](Microsoft.md)의 [클라우드컴퓨팅](%ED%81%B4%EB%9D%BC%EC%9A%B0%EB%93%9C%20%EC%BB%B4%ED%93%A8%ED%8C%85.md)
서비스 [Microsoft Azure](Microsoft%20Azure.md)를 홍보하기 위한 이미지 캐릭터. 담당 성우는 [키타무라에리](%ED%82%A4%ED%83%80%EB%AC%B4%EB%9D%BC%20%EC%97%90%EB%A6%AC.md)

  

Microsoft 공식이다! Microsoft Azure 교육만화 "[클라우드 걸 - 창과 구름과 파란 하늘(クラウドガール -
窓と雲と碧い空)](http://msdn.microsoft.com/ja-jp/windowsazure/claudia/)"에 처음 등장했는데,
연재하는 곳이 개발자 지원 사이트 [MSDN](MSDN.md)... <del>[아이자와히카루](%EC%95%84%EC%9D%B4%EC%9E%90%EC%99%80%20%ED%9E%88%EC%B9%B4%EB%A3%A8.md)
이후로 MS가 [신사](%EC%8B%A0%EC%82%AC.md)력에 눈을 뜬 듯</del>

  

[나나미](%EB%A7%88%EB%8F%84%EB%B2%A0%20%EB%82%98%EB%82%98%EB%AF%B8.md)의
사촌으로(나나미도 덩달아 공식 인증...), 간만에 일본에 놀러왔다가 서버 문제로 고심해하던 나나미에게 Microsoft Azure에 대해서
가르쳐 주는 것이 만화의 내용. 공순이(...)답게 박식하고 냉정한 쿨뷰티로, 다소 덜렁거리는
[도짓코](%EB%8F%84%EC%A7%93%EC%BD%94.md)로 표현되는 나나미를 이끄는 역할. 설정상으로 나나미보다 7살
많다.`[2]`

  

이런 둘의 관계는 트위터에서도 표현되므로, 두명의 트위터를 모두 팔로우한다면 <del>손발의 오글거림을 참아낼 수 있다면</del> 재미있게
볼 수 있을듯.

  

일본 MS에서는 이 캐릭터를 적극적으로 활용하여, Microsoft Azure 이외에도 각종 MS 기술의 소개 만화에 등장시키고 있으며,
[그림 소재와 캐릭터 보이스를 포함한 이용 가이드라인](http://msdn.microsoft.com/ja-
jp/windowsazure/claudia00_03) 제공을 제공하고 있다. 여기까지는 그냥 그러려니 할 수 있겠는데, 그칠 줄 모르는 일본
MS의 신사력은 [Visual Studio 2010 with MSDN을 구입시 시리얼 넘버가 들어간 완전 한정판 피겨를 제공하는
캠페인](http://www.microsoft.com/japan/msdn/vstudio/save/campaign/02)을 벌이기에 이르른다!
