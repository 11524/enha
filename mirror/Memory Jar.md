1999년 2월 중순 카드가 발매되어 3월 1일 토너먼트에서 사용되기 시작했으나, 2주 뒤
[비엔나](%EB%B9%84%EC%97%94%EB%82%98.md)에서 개최된
[그랑프리](%EA%B7%B8%EB%9E%91%ED%94%84%EB%A6%AC.md)가 끝나자 마자 즉시 금지되어 **최단 시간에
금지된 카드**로 기록되었다. 마나비용이 크기 때문에 별 문제 없을 것이라고 생각해서 카드를 일곱 장이나 뽑게 만들고, 혼자 뽑으면 뭔가
불공평한 듯한 느낌이 들어 상대방에게도 똑같이 카드를 뽑게 만들었지만, 그 결과 밸런스를 무너뜨리는 최악의 카드 디자인이 나왔다.

  

특히 Urza's Legacy 발매 전 [Windfall](Windfall.md)(횡재)을 금지한 사례에서 **카드를 버리게 만드는
것이 당장 카드를 뽑는 것에 대해 그리 큰 페널티가 되지 않는다**는 것을 목격했음에도 불구하고 같은 컨셉의 카드가 나온 것에 대해 비난이
있었다. 더구나 카드를 뽑는 데 특화된 청색에서조차 [Braingeyser](Braingeyser.md)로 카드 7장을 뽑으려면
{7}{U}{U}가 필요한데 **무색 마나 5개로 카드 7장 뽑기**는 기존 매직 더 개더링에서의 밸런싱을 무시하는 비정상적으로 높은
효율이다.

  

이후 모든 타입에서 금지가 되고 빈티지에서 제한이 되어버린 이후에는 주로 커맨더의 [Nekusar, theMindrazer](Nekusar%2C%20the%20Mindrazer.md) 덱에서 활약하고 있다. Time Spiral에서는 이것의
생물 버전인 [Magus of the Jar](Magus%20of%20the%20Jar.md)이 나온 적도 있지만, 소환후유증이 있고
없고는 효용성 면에서 매우 큰 차이가 있다.

### 2.2. 운영 ¶

상대방은 순간마법이 없다면 카드 일곱 장을 고스란히 버려야 하므로 편두통에 의해 Memory Jar 하나 당 14점의 피해를 입고, 따라서
Memory Jar 두 개와 편두통 하나면 게임이 끝난다. 순간마법이 있는 덱을 상대로도 Defense Grid 등을 이용해 카드를 버리기
전에 사용할 가능성을 봉쇄할 수 있다.

  

Lion's Eye Diamond를 사용할 수 있었던 **익스텐디드에서는 1턴 킬**이 나왔고, Lotus Petal과 Mox
Diamond가 건재했기에 **스탠다드에서조차 갓핸드면 1턴 킬**이 가능했다.`[1]`

  

아래 예시한 패턴 외에도 다양한 방법으로 콤보를 2턴만에 완성할 수 있다.  
턴1: Ancient Tomb, Grim Monolith  
턴2: [Underground River](Underground%20River.md)(지저천), Grim Monolith로 Mana
Vault 꺼내 바로 탭해 {5}로 Memory Jar, 바로 능력 사용하여 카드 일곱 장 뽑음.  
Tinker로 마나 저장실 희생하여 두 번째 Memory Jar 찾아옴. Mox Diamond 꺼내 {B} 뽑아 Dark Ritual,
Megrim.  
다시 두 번째 Memory Jar 능력 사용하면 턴 끝에 상대방은 카드 14장을 버리면서 28점 피해를 입음.

  

Memory Jar로 뽑는 카드가 마땅치 않으면 Yawgmoth's Will까지 써서 Memory Jar와 각종 마나 원천들을 무덤에서
재활용할 수도 있다.

### 2.3. 평가 ¶

콤보를 구성하는 카드가 Memory Jar와 Megrim 두 종류 뿐이고 그 중 하나가 드로우 수단까지 겸하기 때문에, 덱을 구성함에 있어
여유 슬롯이 다른 콤보덱들에 비해 엄청나게 많은 편이다. 따라서 모든 역량을 콤보를 빨리 완성하는 데 쏟아부을 수 있었다. 특히 5판이
6판에 밀려나기 전, Urza's Legacy가 발매되면서 마나 저장실과 Grim Monolith를 함께 사용할 수 있었던 점은 스탠다드
환경에서조차 마법물체를 기반으로 한 콤보덱들을 비정상적으로 빠르게 만들었다.

  

대회 하나만 보고 즉시 카드를 금지시킬 만큼 강력한 카드였으나, 아이러니하게도 해당 대회의 우승은 [HighTide](High%20Tide.md) 덱(해당 항목 참고; [당시 덱 리스트](http://sales.starcitygames.com
/deckdatabase/displaydeck.php?DeckID=37314))이라는, 사이드보드 외에는 청색 단색인 헤비 퍼미션/콤보 덱이
차지하였다. **Memory Jar와 Tinker, Defense Grid만 무효화**하면 이 덱은 상대방에게 아무런 위협을 줄 수 없기
때문이다.
