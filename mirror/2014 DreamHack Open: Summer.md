# 2. 이모저모 ¶

* 네덜란드의 1998년생 프로게이머 'Sjaak' 워우터 브리(Wouter Bree)가 16강에 진출하는 이변을 만들었다.  
* [윤영서가](%EC%9C%A4%EC%98%81%EC%84%9C%EA%B0%80.md) 단 한 세트도 내주지 않으며 전승우승을 달성했다.

ction=edit&section=3)]
