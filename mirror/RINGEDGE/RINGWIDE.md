■ RINGEDGE 사양  

・ CPU : INTEL Pentium 프로세서 E2160(동작주파수: 1.8GHz) <del>오버클럭 시키고 싶다</del>`[1]`  
・ 메인메모리 : DDR2 SDRAM PC2-6400 1GB  
・ GPU : NVIDIA사의 GPU탑재, GDDR3 메모리 384MB탑재, Shader Model 4.0 대응, 2화면출력(최대해상도
1920x1200)  
・ 사운드 : High Definition Audio탑재, 192KHz/32bit, 5.1ch출력대응  
・ 입출력 : 비디오 - 아날로그/디지털출력 DVI-I단자x2, 사운드 - 5.1ch(스테레오 미니잭x3), LAN - On board
10BASE-T, 100BASE-TX, 1000BASE-T, 업무용 게임 입출력 - JVS I/O 커넥터, 시리얼 - 3ch,
USB2.0(USB미니A단자)x3port, I/O용 전원출력커넥터(12V/5V/3.3V(  
・ 저장장치 : SATA SSD 32GB 탑재  
・ OS : Microsoft® Windows® Embedded Standard 2009  
・ 기타 : 전원 내장, ALL.Net 접속 대응의 통신 기능을 표준 서포트(배포, 과금에 대응)

・ 메인보드 : Intel Q57 Express  
・ 메인메모리 : 2GB (DDR3로 추정)  
・ GPU : NVIDIA Geforce GT545

・ 저장장치 :32GB TDK GBDISK RS3 SSD  

■ RINGWIDE 사양  

・ CPU : INTEL Celeron 프로세서 440(동작주파수: 2GHz)  
・ 메인메모리 : DDR2 SD-RAM PC2-5300 1GB  
・ GPU : AMD제 GPU탑재, GDDR3 메모리 128MB(시스템메모리에 의한 확장가능), Shader Model 4.0 대응,
2화면출력(최대해상도 1920x1200)  
・ 사운드 : High Definition Audio탑재, 192KHz/32bit, 5.1ch출려대응  
・ 입출력 : 비디오 - 아날로그출력 D-Sub 15pin, 아날로그/디지털출력 DVI-I단자의 2출력, 사운드 - 5.1ch(스테레오
미니잭x3),

LAN - On board 10BASE-T, 100BASE-TX, 1000BASE-T, 업무용 게임 입출력 - JVS I/O 커넥터, 시리얼
- 3ch, USB2.0(USB미니A단자)x2port, I/O용 전원출력 커넥터(12V/5V)  

・ 저장장치 : CF 8GB 탑재  
・ OS : Microsoft® Windows® Embedded Standard 2009  
・ 기타 : 외부12W단일전원으로 구동, ALL.Net 접속 대응의 통신 기능을 표준 서포트(배포, 과금에 대응)  

[LINDBERGH](LINDBERGH.md)의 후속 기판으로, RINGEDGE는 고급형, RINGWIDE는 보급형 기판이다.
LINDBERGH와는 달리 [윈도우즈](%EC%9C%88%EB%8F%84%EC%9A%B0%EC%A6%88.md) 기반.  
그리고 'RINGEDGE 2'라는 기판이 2012년을 시작으로 보급중이다.  
일본의 아케이드 게임 시장이 2010년 현재들어 줄어들고 있는 것도 있고 해서 그렇게 잘 나가는 기판은 아니지만, 어쨌든 이 기판으로 제작된
게임이 하나둘 씩 추가되고 있는 중. 특히 LINDBERGH에서 지원하지 않았던 서드파티 정책을 다시 지원하기 시작하여, 탄탄한 서드파티로
좋은 보급률을 보여주었던 [Type X](Type%20X.md)와의 경쟁이 예상된다.
