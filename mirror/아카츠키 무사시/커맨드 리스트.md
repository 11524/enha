## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 베개 누르기 : 근접해서 ← or → + B  

★ 특수기  

  * 바닥 뽑기 : 상대 다운시 ↘ + B
  * 비도이연 : ↘ + B
  * 겁염검 : ← + B
  * 집중 : A버튼 지속  

★ 필살기  

  * 간헐천 : ↓↘→ + B
  * 십자세 : →↓↘ + B (모으기 가능)
  * 위협하기(●) : ←↙↓↘→ + A
  * 침이연 : 근접해서 →↘↓↙← + B  

★ 초오의  

  * 오륜 겹치기 : →↘↓↙←→ + AB  

★ 잠재오의  

  * 이천무쌍검 : →↘↓↙←→↘↓↙← + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ([네오지오CD](%EB%84%A4%EC%98%A4%EC%A7%80%EC%98%A4%20CD.md)/[드림캐스트](%EB%93%9C%EB%A6%BC%EC%BA%90%EC%8A%A4%ED%8A%B8.md) 이식판) ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 베개 누르기 : 근접해서 ← or → + CD  

★ 특수기  

  * 바닥 뽑기 : 상대 다운시 ↘ + B
  * 비도이연 : ↘ + B
  * 집중 : ↓↙← + A버튼 지속  

★ 필살기  

  * 간헐천 : ↓↘→ + B
  * 십자세 : →↓↘ + B (모으기 가능)
  * 위협하기(●) : ←↙↓↘→ + A
  * 침이연 : 근접해서 →↘↓↙← + B  

★ 초오의  

  * 오륜 겹치기 : →↘↓↙←→ + AB  

★ 잠재오의  

  * 이천무쌍검 : →↘↓↙←→↘↓↙← + B  

★ 난무오의  

  * 침략 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, ↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, BC, ↓↘→ + B
