## 2. 고급언어 ¶

  * [씨앗](%EC%94%A8%EC%95%97.md)
  * [알골](%EC%95%8C%EA%B3%A8.md) : 최초의 구조적 언어. 널리 쓰이진 않았지만 C등의 언어에 영향을 주었다.
  * [액션스크립트](%EC%95%A1%EC%85%98%EC%8A%A4%ED%81%AC%EB%A6%BD%ED%8A%B8.md)
  * [오베론](%EC%98%A4%EB%B2%A0%EB%A1%A0.md)
  * [창조](%EC%B0%BD%EC%A1%B0.md)
  * [코볼](%EC%BD%94%EB%B3%BC.md) : 사무처리용 언어로 포트란과 함께 언어분야를 양분했**었**다.
  * [파스칼](%ED%8C%8C%EC%8A%A4%EC%B9%BC.md)
  * [포트란](%ED%8F%AC%ED%8A%B8%EB%9E%80.md) : 최초의 고급언어. 수치해석에 뛰어나다.
  * [Ada](%EC%97%90%EC%9D%B4%EB%8B%A4#s-2.md)
  * [BASIC](BASIC.md) : 시분할 개념이 도입
  * [C](C%28%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D%20%EC%96%B8%EC%96%B4%29.md) : 상당히 초기에 개발된 언어면서 동시에 고급언어임에도 어셈블리에 맞먹는 하드웨어 제어가 가능해서 혼자 '중급언어'라는 카테고리를 만들어 들어가는 경우도 있다.
  * [C++](C++.md)
  * [C#](C%23.md)
  * [Caml](Caml.md) : [프랑스](%ED%94%84%EB%9E%91%EC%8A%A4.md)에서 만들어진 [ML](ML.md)계열의 함수형언어
  * [OCaml](OCaml.md) : Objective-Caml. Caml 에 객체지향 개념 추가.
  * [F#](F%23.md) : OCaml 의 .net 버전
  * [Clojure](Clojure.md) : LISP의 후계자격인 언어. 자바 VM에서 실행된다.
  * [Scala](Scala.md) : 다중패러다임 언어. 자바 VM에서 실행된다.
  * [D](D.md) : 처음에는 Mars라는 이름으로 불렸었다.
  * [Erlang](Erlang.md) : 병행성 언어
  * [Haskell](Haskell.md) : 함수형 언어
  * [Go](Go.md) : 구글에서 C와 같이 시스템 프로그래밍 목적으로 만든 언어. [홈페이지](http://golang.org/)
  * [Java](Java.md)
  * [JavaScript](JavaScript.md)
  * [Julia](Julia.md)
  * [Coffee Script](Coffee%20Script.md) : JavaScript를 간략하게 만든 언어. 결과물로 JavaScript가 나온다
  * [LISP](LISP.md) : 함수형 언어
  * [Lua](Lua.md)
  * [Objective-C](Objective-C.md) : 보통 Obj-C로 쓴다. [애플](%EC%95%A0%ED%94%8C%28%EA%B8%B0%EC%97%85%29.md) 제품에 주로 쓰인다.
  * [Perl](Perl.md)
  * [PHP](PHP.md)
  * [Prolog](Prolog.md) : 논리형 언어. [인공지능](%EC%9D%B8%EA%B3%B5%EC%A7%80%EB%8A%A5.md)에 사용.
  * [Processing](Processing.md) : 미디어 아트 지향 언어.
  * [Python](Python.md)
  * [RPG](RPG#s-4.md)
  * [Ruby](Ruby.md)
  * [Rust](Rust.md) : [모질라 재단](%EB%AA%A8%EC%A7%88%EB%9D%BC#s-1.md)에서 개발중이며 C++을 대체하기 위한 언어.
  * [Scheme](Scheme.md) : Lisp에서 유래한 함수형 언어.
  * [Smalltalk](Smalltalk.md) : 최초로 객체지향을 구현한 언어
  * [Swift](Swift.md) : WWDC 2014에서 공개된 Objective-C를 대체할 애플의 차세대 언어.
  * [TypeScript](TypeScript.md) : JavaScript를 개선한 언어. Coffee Script 처럼 결과물로 JavaScript가 나온다. [링크](http://www.typescriptlang.org/)

## 3. [프로그래밍언어](%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D%20%EC%96%B8%EC%96%B4.md)로
보기 애매한 언어 ¶

  

튜링-완전하지 않거나, 특수목적에 한정된 언어(Domain Specific Language)들을 이곳에 나열한다.

  

  * [마크다운](%EB%A7%88%ED%81%AC%EB%8B%A4%EC%9A%B4.md) : 마크업 언어중 하나.
  * [HLSL](HLSL.md) : 그래픽카드 쉐이더 언어
  * [HTML](HTML.md) : 엄밀히 따지면 프로그래밍 언어는 아니다. '마크업 언어'라고 한다. `[1]`
  * [OWL](OWL.md)
  * [RDF](RDF.md)
  * [SQL](SQL.md)
  * [TeX](TeX.md)
  * [VHDL](VHDL.md) : 하드웨어 기술 언어
  * [Verilog](Verilog.md) : 하드웨어 기술 언어
  * [UML](UML.md) : C나 Java처럼 바로 소프트웨어로 만들어지는 언어는 아니지만, 소프트웨어를 설계할수 있는 언어.
  * [XML](XML.md) : 마크업 언어의 일종으로, 데이터를 구조적으로 저장하고 교환하기 위해 만든 문서 양식.
