# 1. 개요 ¶

**KernelEX** 는 Windows 9x 계열 에서 Windows NT 계열`[2]` 의 어플리케이션 을 구동하기위한 오픈소스 호환레이어 이다.  
얼핏 보면 [Wine](Wine.md) 과 비슷하다고 할수는 있지만 이녀석은 Windows 9x/구NT 계열 만 지원 하기 때문에 약간
다르다고 보면 된다.`[3]`

  

[[edit](http://rigvedawiki.net/r1/wiki.php/KernelEX?action=edit&section=2)]
