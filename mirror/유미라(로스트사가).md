# 3. 기본기 ¶

  * 기본 사양 : 적을 공격 할 때마다 쌓이는 **차력 게이지**가 존재한다. 차력 게이지가 노란색일 때 여포 봉선 모드로의 전환이 가능하다.
  * DDD : [목검](%EB%AA%A9%EA%B2%80.md)으로 세번 베는 평타.
  * D살짝 : 목검으로 후려쳐 경직시키는 특수 공격. 2회 피격시 공중으로 뜬다. 차력 게이지가 주황색일 때 D꾹을 해도 이게 나간다.
  * D꾹 : 목검을 집어 던지며 여포 봉선 모드로 전환. 목검에 피격 된 적은 기절한다. 자세한 것은 후술.
  * A D : 공중에서 목검으로 베는 점프 공격.
  * A D꾹 : 공중에서 목검으로 땅을 내리 찍어 넘어진 적을 공격하는 점프 특수 공격. 다운한 상태의 적에게 유효하다.
  * 대시 D : 달리는 중 목검으로 적을 띄우는 대시 공격.
  * (방어 후)D : 돌면서 목검으로 벤다.

## 3.1. 기본기(여포 봉선) ¶

여포 봉선 모드 중에는 공격력ㆍ이동속도 등 능력치가 증가한다. 다른 변신형 용병인 [웨어울프](%EC%9B%A8%EC%96%B4%EC%9A%B8%ED%94%84%28%EB%A1%9C%EC%8A%A4%ED%8A%B8%EC%82%AC%EA%B0%80%29.md)와는 달리 특수
모드를 유지하는 에너지인 차력 게이지가 자동으로 닳지 않고, 적에게 공격 받을 때마다 게이지가 닳는다. 게이지가 다 닳으면 여포 봉선 모드가
강제로 해제 된다.  
여포 봉선 모드 중엔 기본기가 아래와 같이 바뀐다.

  

  * DDD : 검을 휘두르는 동작을 **맨손으로** 하는 평타. 막타에 적을 경직시킨다.
  * D꾹 : **방천화극**. 차력 게이지 15를 소모하고`[1]` 전방으로 방어가능한 지상 장풍을 날린다. 피격 된 적은 띄워지며,땅긁이 가능하다
  * D살짝 : **무쌍난무**. 3연타 후 손에서 폭발을 일으켜 적을 무릎꿇게 만드는 특수기,방어파괴 공격 이며 막타는 넘어진 적도 적중은 가능하나 별다른효과가없으니 쓰지말자. 차력 게이지 10 소모.`[2]`
  * ADDDD : **탄지신공**. 점프 중에 대각선 아래 방향으로 쇠구슬을 연속으로 날리는 점프 공격. 2타 이상 성공하면 적을 넘어뜨리며, 넘어진 적도 타격한다. 넉백은 약하다.
  * A 대시 : 공중에서 해당 방향으로 발차기를 날리는 공중 대시 공격.
  * 대시 DD : 달리는 중 적을 연타 해 띄우는 대시 공격.
  * 대시 D꾹 : 대시 공격의 2타째의 공격을 하는 대시 공격.
  * (다운 중) 방향키 : 다운 된 상태에서 굴러서 일어난다. [닷지](%EC%8A%A4%ED%8E%98%EC%85%9C%ED%8F%AC%EC%8A%A4%28%EB%A1%9C%EC%8A%A4%ED%8A%B8%EC%82%AC%EA%B0%80%29.md)의 기본기 버전.
  * (방어 후)D : 변신 전과 반격기는 동일. 다만 반격속도가 빠르다.이 속도가 어느정도냐함은 이오리,마이등의 커맨드계열용병이 커맨드연계를하고있을때 반격을하면 커맨드 연계를 끊어먹을정도(물론 자기도 한두대 맞는다)
