# 1. [KOF 99](KOF%2099.md) ¶

  

★ 기본잡기  

  * 반동격 : 근접해서 ← or → + C
  * 회선풍 : 근접해서 ← or → + D  

★ 특수기  

  * 엽호격 : → + B (버튼 지속시 엽호진으로 이행)
  * 유농축 : 공중에서 ↓ + B  

★ 필살기  

  * 만월참 : ↓↙← + B or D (B로 사용하고 버튼 지속시 엽호진으로 이행)
  * 배기격 : ↓↙← + A or C (A로 사용하고 버튼 지속시 저취진으로 이행)
  * 공사진 : ↓ 모으고 ↑ + A or C (A로 사용하고 버튼 지속시 저취진으로 이행)
  * 저취진 : ↓↓ + A (버튼 지속)  
┠ 취조각 상단 : 저취진 지속 중 ↑ + C  
┠ 취조각 하단 : 저취진 지속 중 C  
┠ 태극파 : 저취진 지속 중 D  
┠ 변환 동작 : 저취진 지속 중 B (엽호진으로 이행)  
┗ 변환 공격 : 저취진 지속 중 ← or → + B (버튼 유지시 엽호진으로 이행)

  * 엽호진 : ↓↓ + B (버튼 지속)  
┠ 비호격 : 엽호진 지속 중 ↑ + C  
┠ 맹호격 : 엽호진 지속 중 C  
┠ 습호격 : 엽호진 지속 중 ↓ + C  
┠ 대극피 : 엽호진 지속 중 D  
┠ 변환 동작 : 엽호진 지속 중 A (저취진으로 이행)  
┗ 변환 공격 : 엽호진 지속 중 ← or → + A (버튼 유지시 저취진으로 이행)  

★ 초필살기  

  * 봉황열조각 : ↓↘→↓↘→ + B or D
  * 봉황천무각 : 공중에서 ↓↘→↓↘→ + B or D

# 4. [KOF 2003](KOF%202003.md) ¶

  

★ 기본잡기  

  * 회선풍 : 근접해서 ← or → + C or D  

★ 특수기  

  * 저취격 : → + A (버튼 지속시 저취진으로 이행)
  * 엽호격 : → + B (버튼 지속시 엽호진으로 이행)
  * 유농축 : 공중에서 ↓ + B  

★ 필살기  

  * 만월참 : ↓↙← + B or D (B로 사용하고 버튼 지속시 엽호진으로 이행)
  * 배기격 : ↓↙← + A or C (A로 사용하고 버튼 지속시 저취진으로 이행)
  * 공사진 : ↓ 모으고 ↑ + A or C (A로 사용하고 버튼 지속시 저취진으로 이행)
  * 저취진 : ↓↓ + A  
┠ 만월참 : 저취진 지속 중 B  
┠ 취조각 상단 : 저취진 지속 중 ↑ + C  
┠ 취조각 하단 : 저취진 지속 중 C  
┠ 고응파 : 저취진 지속 중 ← + B  
┠ 변환 공격 : 저취진 지속 중 A (버튼 지속시 엽호진으로 이행)  
┗ 자세 풀기 : 저취진 지속 중 D

  * 엽호진 : ↓↓ + B  
┠ 공사진 : 엽호진 지속 중 ↑ + A  
┠ 호열조 : 엽호진 지속 중 B  
┠ 비호격 : 엽호진 지속 중 ↑ + C  
┠ 맹호격 : 엽호진 지속 중 C  
┠ 습호격 : 엽호진 지속 중 ↓ + C  
┠ 변환 공격 : 엽호진 지속 중 A (버튼 지속시 저취진으로 이행)  
┗ 자세 풀기 : 엽호진 지속 중 D

  * 봉황열조각 : ↓↘→↓↘→ + B or D / 저취진 사용 중 ↓↘→↓↘→ + B
  * 봉황천무각 : 공중에서 ↓↘→↓↘→ + B or D  

★ 리더 초필살기  

  * 전식 열조비천각 : ↓↘→↓↘→ + AC / 엽호진 사용 중 ↓↘→↓↘→ + C
