# 1. 개요 ¶

  

[동방쇄환전](%EB%8F%99%EB%B0%A9%EC%87%84%ED%99%98%EC%A0%84.md)에 등장하는 영웅. 모델 有/無.
기반 영웅은+++.

  

이름 : [+++](+++.md)  
클래스 : +++  
사정거리 : +++(근거리/원거리)  
이동속도 : +++  
공격속도 : +++  
기본 공격력 : ++~++(속성 제외)

  

힘 +++  
민첩 +++
