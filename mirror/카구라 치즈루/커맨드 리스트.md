# 1. [KOF 96](KOF%2096.md) ¶

  

★ 기본잡기  

  * 영월 : 근접해서 ← or → + C
  * 회천 : 근접해서 ← or → + D  

★ 필살기  

  * 백활 천신의 리 : →↓↘ + A or C
  * 백팔활 옥향의 슬음 : ←↙↓↘→ + A or C
  * 이백십이활 신속의 축사 : →↘↓↙← + A or B or C or D  
┗ 이백십이활 신속의 축사 천서 : ↓↙← + (신속의 축사 사용버튼)  

★ 초필살기  

  * 이면 팔십오활 영기의 주춧돌 : ↓↙←↙↓↘→ + A or C
  * 이면 일활 삼뢰의 포진 : ↓↘→↓↘→ + B or D

# 4. [KOF 2003](KOF%202003.md) ¶

  

★ 기본잡기  

  * 영월 : 근접해서 ← or → + C or D  

★ 특수기  

  * 제활 쟁쟁 : → + A
  * 제활 창창 : → + B
  * 제활 종종 : ↘ + B
  * 제활 실실 : 공중에서 CD  

★ 필살기  

  * 백활 천신의 리 : →↓↘ + A or C
  * 백팔활 옥향의 슬음 : ↓↘→ + A or C
  * 이백십이활 신속의 축사 쏘아올림 : ←↓↙ + A or B or C or D  
┗ 이백십이활 신속의 축사 천서 : ←↓↙ + A or B or C or D

  * 이백십이활 신속의 축사 떨어붙임 : ↓↙← + A or B or C or D  
┗ 이백십이활 신속의 축사 천서 : ↓↙← + A or B or C or D

  * 이백십이활 을식 정문의 일침 : ↓↓ + A or B or C or D
  * 제활 보보 : →↓↘ + B or D  

★ 초필살기  

  * 이면 팔십오활 영기의 주춧돌 : ↓↙←↙↓↘→ + A or C
  * 이면 일활 삼뢰의 포진 : ↓↘→↓↘→ + B or D  

★ 리더 초필살기  

  * 삼신기의 삼 : ↓↘→↓↘→ + AC
