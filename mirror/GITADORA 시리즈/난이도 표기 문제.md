#### 2.1. 개요 ¶

  * 개인적인 감상이 아닌 **여러 유저들도 공감할 수 있는 수준**의 경우만 표기하도록 한다.
  * 레벨 표기는 최신작인 [GITADORA OverDrive](GITADORA%20OverDrive.md)의 표기 난이도를 기준으로 하며, 최신 버전 이전에 삭제된 구곡의 채보에 대해서는 해당 곡이 수록되었던 최신 시리즈의 레벨표기가 기준이 된다. 또한, 해당 곡은 최신 및 그 이전 버전에서는 삭제되어 즐길 수 없다는 부가적인 설명을 따로 표시한다.`[1]`
  * **같은 레벨대에서 어렵거나 쉽다고 무조건 사칭/역사칭으로 적지 않는다.** 예를 들어 레벨 9.xx 전반 중에서는 하위권이나, 일반적인 레벨 8.xx 후반 곡들보다는 강하거나 레벨 8.xx 후반 상위권 곡들과 비슷할 경우에는 물렙곡에 속하지 않는 것을 원칙으로 한다. 최소한 해당 레벨의 평균대보다 0.4 이상 격차가 있는, 그러니까 레벨이 한 단계 이상 상승/하락해도 위화감이 없는 패턴들을 작성할 것.

#### 2.2. 사칭/역사칭/개인차/특정 패턴 주의 구분 ¶

  * **사칭곡**(불렙)은 아래와 같은 기준을 만족하며, 붉은색으로 작성한다.  

    * 해당 곡이 표기 레벨에 해당하는 평균적인 레벨의 곡보다 난이도가 높은 경우.
    * 해당 곡의 난이도가, 해당 곡보다 레벨이 0.4 높은 곡들 중 최소한 하나 이상과 비슷할 경우.   

  * **역사칭곡**(물렙)은 아래와 같은 기준을 만족하며, 푸른색으로 작성한다.  

    * 해당 곡이 표기 레벨에 해당하는 평균적인 레벨의 곡보다 난이도가 낮을 경우.
    * 해당 곡의 난이도가, 해당 곡보다 레벨이 0.4 낮은 곡들 중 최소한 하나 이상과 비슷할 경우.   

  * **개인차 곡**은 아래와 같은 기준을 만족하며, 보라색으로 작성한다.  

    * 해당 곡의 난이도가 플레이어에 따라 크게 달라진다고 인정되는 경우.
    * 해당 곡의 채보에 특정 패턴`[2]`이 집중되어 있는 경우.  

  * **풀콤보 한정 사칭곡**은 아래와 같은 기준을 만족하며, 올리브색으로 작성한다.  

    * 해당 곡의 난이도 표기에 비해 풀 콤보 난이도가 매우 높은 경우
    * 해당 곡이 동 레벨 사이에서 사칭/역사칭에 해당하지 않는 경우  

  * **복****합****채****보**는 위의 6조건중 2가지 이상을 만족할 경우`[3]`일때 곡 제목을 대응하는 색들로 표현하도록 한다.

##### 2.2.1. 기타프릭스 한정 ¶

  * **속주 주의 곡**은 아래와 같은 기준을 만족하며, 초록색으로 작성한다.  

    * 해당 곡이 난이도 표기에는 타당하나, 속주 구간 때문에 풀콤보 및 스코어링 난이도가 높아질 경우
    * 속주 구간을 칠 수 있다는 가정하에 사칭/역사칭에 해당하지 않을 경우  

  * **넥변환 주의 곡**은 아래와 같은 기준을 만족하며, 갈색으로 작성한다.  

    * 해당 곡이 난이도 표기는 알맞지만 넥변환에 있어서 세심한 테크닉이 요구되는 경우

##### 2.2.2. 드럼매니아 한정 ¶

  * **체력 주의 곡**은 아래와 같은 기준을 만족하며, 초록색으로 작성한다.  

    * 해당 곡이 난이도 표기는 타당하지만 더블 베이스 같은 체력을 요구하는 노트의 빈도가 심할 경우
    * 해당 구간을 넘길 수 있는 체력이 있다는 가정하에 사칭/역사칭에 해당하지 않는 경우  

  * **박자 주의 곡**은 아래와 같은 기준을 만족하며, 갈색으로 작성한다.  

    * 해당 곡에 대해 박자가 말릴 경우 클리어가 굉장히 힘들어질 경우
    * 박자를 전부 맞춘다는 가정하에 사칭/역사칭에 해당하지 않을 경우  

각 항목은 사칭곡→ 역사칭곡 → 개인차 곡 → 속주 주의 곡/체력 주의 곡 → 넥변환 주의 곡/박자 주의 곡 → 풀콤보 한정 사칭곡
→복합채보 곡 순으로 적는 것을 원칙으로 한다. 같은 레벨의 같은 속성의 곡이라면, 먼저 수록된 곡부터 적도록 한다.

  

곡명 뒤에 곡에 해당하는 난이도(BASIC, ADVENCED, EXTREME, MASTER)를 표기하고, 그 뒤에 모드(기타 표준, 기타
베이스, 드럼매니아)와 첫 등장 버전을 표기한다. 필요하다면 동영상 링크도 걸도록 한다. **항목이나 데이터 로딩이 길어지는 것을 방지하기
위해 동영상은 링크로 걸어두는 것을 권장한다.**
