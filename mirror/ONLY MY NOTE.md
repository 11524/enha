# 1. [아이돌 마스터 ONE FOR ALL](%EC%95%84%EC%9D%B4%EB%8F%8C%20%EB%A7%88%EC%8A%A4%ED%84%B0%20ONE%20FOR%20ALL.md) 수록곡 ¶

  
[아이돌 마스터 ONE FOR ALL](%EC%95%84%EC%9D%B4%EB%8F%8C%20%EB%A7%88%EC%8A%A4%ED%84%B0%20ONE%20FOR%20ALL.md)의 타이틀 곡이다. 게임 실행 전 인트로 음악도 이 곡이고, 오프닝 무비의 곡도 이 곡이다.
