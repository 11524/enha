# 1. 개요 ¶

이 해에는 [대한제국](%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD.md)이
[일제](%EC%9D%BC%EC%A0%9C.md)로부터 가장 치욕스러운 일을 당했던 시기로, [헤이그특사](%ED%97%A4%EC%9D%B4%EA%B7%B8%20%ED%8A%B9%EC%82%AC.md) 건을 빌미로
[고종](%EA%B3%A0%EC%A2%85.md)이 [일제](%EC%9D%BC%EC%A0%9C.md)에 의해 강제 퇴위하고 뒤를
이어 마지막 황제 [순종](%EC%88%9C%EC%A2%85.md)이 즉위하였으며 급기야는
[대한제국](%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD.md) 군대가 강제해산되고 이를 계기로 전국 각지에
[의병](%EC%9D%98%EB%B3%91.md) 활동이 활기를 띠게 되었던 시기였다.

## 2.1. 실제 ¶

  * [5월 22일](5%EC%9B%94%2022%EC%9D%BC.md) : [이토 히로부미](%EC%9D%B4%ED%86%A0%20%ED%9E%88%EB%A1%9C%EB%B6%80%EB%AF%B8.md)의 압력에 의해 [이완용](%EC%9D%B4%EC%99%84%EC%9A%A9.md) 내각이 성립되었다. 그리고 나라를 팔아먹었다.
  * [7월 19일](7%EC%9B%94%2019%EC%9D%BC.md) : [고종](%EA%B3%A0%EC%A2%85.md)이 강제퇴위되었다.
  * [8월 1일](8%EC%9B%94%201%EC%9D%BC.md) : 대한제국군이 강제해산되었다.
  * [8월 2일](8%EC%9B%94%202%EC%9D%BC.md) : [순종](%EC%88%9C%EC%A2%85.md)이 황제에 즉위하였다.
  * [9월 26일](9%EC%9B%94%2026%EC%9D%BC.md) : [뉴질랜드](%EB%89%B4%EC%A7%88%EB%9E%9C%EB%93%9C.md)와 [뉴펀들랜드](%EB%89%B4%ED%8E%80%EB%93%A4%EB%9E%9C%EB%93%9C.md)가 [영국](%EC%98%81%EA%B5%AD.md)의 자치령이 되었다.
  * [10월 12일](10%EC%9B%94%2012%EC%9D%BC.md) : [시카고 컵스](%EC%8B%9C%EC%B9%B4%EA%B3%A0%20%EC%BB%B5%EC%8A%A4.md)가 [디트로이트 타이거즈](%EB%94%94%ED%8A%B8%EB%A1%9C%EC%9D%B4%ED%8A%B8%20%ED%83%80%EC%9D%B4%EA%B1%B0%EC%A6%88.md)를 상대로 팀의 첫 [월드 시리즈](%EC%9B%94%EB%93%9C%20%EC%8B%9C%EB%A6%AC%EC%A6%88.md) 우승을 달성했다. 
  * [11월 6일](11%EC%9B%94%206%EC%9D%BC.md) : [미국](%EB%AF%B8%EA%B5%AD.md)이 [파나마](%ED%8C%8C%EB%82%98%EB%A7%88.md)의 독립을 승인했다.
