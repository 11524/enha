# 1. [KOF 2003](KOF%202003.md) ¶

  

★ 기본잡기  

  * 추수 충고 : 근접해서 ← or → + C or D  

★ 특수기  

  * 풍가전사 : ↘ + D  
┗ 뇌가전사 : D

  * 비가전사 : 소점프 or 대점프 중 B, D  

★ 필살기  

  * 환무각 내아 : →↓↘ + B or D
  * 환무각 외아 : ←↓↙ + B or D
  * 비모각 : ↓↘→ + B or D
  * 환마 비상경공 : ↓↙← + B or D
  * 사기종룡 : ↓↘→ + A or C (연속 3회 입력)  
┗ 원폐룡신파 : 1 히트 중 → + B or D  

┗ 사량발생근 : ↓↙← + A or C

  * 추수 마각보 : →↘↓↙←→ + A or C (잡기)  
┗ 추수 난재귀 : ↓↙← + A or C  

★ 초필살기  

  * 비전 환몽 주원사혼 : ↓↙←↙↓↘→ + A or C
  * 비전 환몽 원벽 : →↘↓↙←→↘↓↙← + A or C  

★ 리더 초필살기  

  * 오의 다중 환영 암경 : A, C, B, D

# 2. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 추수 충고 : 근접해서 ← or → + C or D  

★ 특수기  

  * 변발권 : → + A
  * 풍가전사 : ↘ + D  
┗ 뇌가전사 : E  

★ 필살기  

  * 환무각 : →↓↘ + B or D (B로 사용시 캔슬)
  * 비모각 : ↓↘→ + B or D
  * 환마 비상경공 : 점프 중 ↓↙← + B or D
  * 사기종룡 : ↓↘→ + A or C (연속 3회 입력)  
┗ 원폐룡신파 : 1 히트 중 → + B or D  

┗ 사량발생근 : ↓↙← + C

  * 추수 마각보 : →↘↓↙←→ + A or C (잡기)  

★ 초필살기  

  * 비전 환몽각 : ↓↘→↓↘→ + B or D
  * 비전 환몽 원령사혼 : ↓↙←↙↓↘→ + A or C
  * 비전 환몽 원령벽 : →↘↓↙←→↘↓↙← + A or C  

★ 리더 초필살기  

  * 오의 다중 환영 암경 : A, C, B, D

# 4. [KOF XIII](KOF%20XIII.md) ¶

  

※ EX 대응기(●)는 A or C로 나가는 기술은 AC로, B or D로 나가는 기술은 BD로 EX 필살기를 쓸 수 있다.

  

★ 기본잡기  

  * 추수 충고 : 근접해서 ← or → + C or D  

★ 필살기  

  * 환무각(●) : → + B
  * 공중환무각(●) : 공중에서 → + B
  * 비모각 전(●) : ↓↘→ + B or D
  * 비모각 후(●) : ↓↙← + B or D
  * 환무권(●) : → + A
  * 공중환무권(●) : 공중에서 → + A
  * 사기종룡(●) : ↓↘→ + A or C (연속 3회 입력)
  * 주원사혼(●) : ↓↙← + A or C  

★ 초필살기  

  * 비전 환몽 폭토사혼(●) : ↓↙←↙↓↘→ + A or C  

★ NEO MAX 초필살기  

  * 비전 환몽 주원사혼 장 : →↘↓↙←→↘↓↙← + AC (잡기)
