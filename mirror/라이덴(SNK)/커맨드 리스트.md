# 4. [KOF XIII](KOF%20XIII.md) ¶

  

※ EX 대응기(●)는 A or C로 나가는 기술은 AC로, B or D로 나가는 기술은 BD로 EX 필살기를 쓸 수 있다.

  

★ 기본잡기  

  * 넥 행잉 : 근접해서 ← or → + C or D  

★ 필살기  

  * 자이언트 봄(●) : ↙ 모으고 → + A or C  
┗ 자이언트 봄 페인트 : AB

  * 독무(●) : ↓↙← + A or C
  * 라이덴 봄(●) : →↓↘ + A or C
  * 헤드 크래쉬(●) : ←↙↓↘→ + B or D (잡기)
  * 슈퍼 드롭 킥 : B or D를 4초 이상 누르고 있다가 뗌 (축적 시간이 길 수록 파워가 증가)  

★ 초필살기  

  * 슈퍼 라이덴 드롭(●) : →↘↓↙←→↘↓↙← + A or C (잡기)
  * 크레이지 트레인 : ↓↘→↓↘→ + A or C  

★ NEO MAX 초필살기  

  * 라이덴 봄버 : ↓↘→↓↘→ + BD

## 5.1. 노멀 버전 ¶

  

★ 기본잡기  

  * 크래셔 봄 : 근접해서 ← or → + 강P 
  * 넥 행잉 브레이커 : 근접해서 ← or → + 강K   

★ 특수기  

  * 플라잉 바디 프레스 : 대각선 점프 중 ↓ + 강P
  * 스탬프 킥 : ↘ + 강K   

★ 필살기  

  * 슈퍼 드롭 킥 : K 누르고 있다가 뗌 
  * 독무 : →↘↓↙← + P 
  * 자이언트 봄 : ← 모으고 → + P 
  * 콤비네이션 바디 블로 : ←↓↙ + P   
┠ 헤드 버트 : 콤비네이션 바디 블로 후 ↓↙← + P  
┗ 프론트 스플렉스 : 콤비네이션 바디 블로 후 ↓↙← + K

  * 썬더 크러쉬 봄 : 레버 1회전 + K (잡기)   

★ 슈퍼 콤보 or 초필살기  

  * 파이어 브레스 : →↘↓↙←→↘↓↙← + P 
  * 디스트럭션 드롭 : 레버 2회전 + K (잡기)
