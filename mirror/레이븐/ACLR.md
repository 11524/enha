### 1.1 [얼라이언스](%EC%96%BC%EB%9D%BC%EC%9D%B4%EC%96%B8%EC%8A%A4%28%EC%95%84%EB%A8%B8%EB%93%9C%20%EC%BD%94%EC%96%B4%29.md) 소속 레이븐 ¶

  * [에반제](%EC%97%90%EB%B0%98%EC%A0%9C.md) / 오라클`[1]` : 밸런스를 중시한 표준2각형 AC. 실탄EO를 동반한 적극적인 전투를 전개, 피탄시의 발열에도 주의를 필요로 한다.  

  * [골디 고든](%EA%B3%A8%EB%94%94%20%EA%B3%A0%EB%93%A0.md) / 레이징 트렌트 Ⅳ : 방어력이 우수한 중장2각형 AC. 중거리에서 위력을 발휘하는 무장이 많고, EO와 오비트를 휘감은 변칙전투를 좋아한다.  

  * [쟈우저](%EC%9F%88%EC%9A%B0%EC%A0%80.md) / 헤븐즈레이 : 공격에 특화한 표준2각형 AC. 방어력부족을 단기결전으로 만회하기 위해, 고속접근으로부터의 러시를 반복한다.  

  * [모리 카도르](%EB%AA%A8%EB%A6%AC%20%EC%B9%B4%EB%8F%84%EB%A5%B4.md) / 핀치백 : 고화력을 자랑하는 표준2각형 AC. 양어깨의 고화력 그레네이드를 주력으로 하나, 원거리전에서의 사격정밀도가 낮다.  

  * [프린시펄](%ED%94%84%EB%A6%B0%EC%8B%9C%ED%8E%84.md) / 선다이얼 페더 : 기동력을 중시한 플로트형 AC. 특출난 것이 없는 기체구성이지만 주력이 되는 양어깨 수직미사일은 강력하다.  

  * [트롯 S 스파](%ED%8A%B8%EB%A1%AF%20S%20%EC%8A%A4%ED%8C%8C.md) / 바리오스 크산토스`[2]` : 무기형의 완부를 사용하는 4각형 AC. 강력한 연장 레이저에 주의를 필요로 하나, 기체의 종합성능은 그다지 높지 않다.

  * [라이운](%EB%9D%BC%EC%9D%B4%EC%9A%B4.md) / 스트럭썬더 : EN무기에 집중한 중장역각형 AC. 전체적으로 높은 화력을 보여주는 강적이며, 특히 양어깨 레이저 캐논은 위협적이다.  

  * [오대로](%EC%98%A4%EB%8C%80%EB%A1%9C.md) / 에이밍 호크 : 기동력중시의 경량2각형 AC. 고속비행으로 적의 사각을 가져가며, 빈틈이 적은 공격을 반복하는 난적.  

  * [오메가](%EC%98%A4%EB%A9%94%EA%B0%80.md) / ClownCrown : 접근전을 전문으로 하는 중장역각형 AC. ECM 메이커를 뿌리며 적에게 접근, 고밀도화력으로 일거에 결착을 짓는다.

## 2 VR 아레나 랭커(레이븐네임/AC명) ¶

  * 1위 - 다이=아몬 / 루키펠 : 종합성능을 높인 표준2각형 AC. 비행으로부터의 근거리전을 전문으로 하여, 사각에서의 고밀도화력을 일거에 때려넣는다.  

  * 2위 - MOODY.M / NOVA-RG : 밸런스를 중시한 표준2각형 AC. 기동력을 살린 고속전투로부터, 강력한 하이 레이저 라이플의 일격을 노린다.  

  * 3위 - 月影 / 네온마이트 : 원거리전을 위주로 하는 탱크형 AC. 고화력, 고열량의 EN무기를 복수 탑재, 방어력도 높고, 정면에서의 사격 승부를 좋아한다.  

  * 4위 - 스사노오 / 에이프릴 풀 : 밸런스를 중시한 플로트형 AC. 높은 기동력과 표준적인 방어력을 갖추고, 중거리고속전투에서 진가를 발휘한다.  

  * 5위 - [Code:Crimson](Code%3ACrimson.md) / 베이비즈 브레스`[3]` : 기동력을 중시한 표준2각형 AC. 방어를 도외시한 단기결전사양기체, 중거리로부터의 맹렬한 러시로 공세를 취한다.  

  * 6위 - 소터너티 / 버그베어드 : 화력중심의 4각형 AC. 근거리의 지상전을 전술의 주체로 하여, 주력이 되는 그레네이드를 연발한다.  

  * 7위 - BJ / [캐스케이드 레인지](%EC%BA%90%EC%8A%A4%EC%BC%80%EC%9D%B4%EB%93%9C%20%EB%A0%88%EC%9D%B8%EC%A7%80.md)`[4]` : 대응능력이 높은 표준2각형 AC, 다채로운 무장으로 전투거리를 선택하지 않으나, 특히 핸드 레일 건이 주력이 된다.  

  * 8위 - 패리 맹크스 / B.Boss : 일점호화주의(一点豪華主義)의 중장2각형 AC. 2기의 대형 미사일에 특화한 기체구성, 기동력은 부족하고 방어력도 충분하진 않다.  

  * 9위 - 캐리커처 / 라이프 이즈 마인 : 블레이드 주체의 경량2각형 AC. 고기동을 살린 초접근전을 전개, 눈이 돌아가는 움직임으로 대전상대를 농락한다.  

  * 10위 - 크레즈나 / 스톤 블록 : 중화기를 장비한 탱크형 AC. 강력한 EO를 동반한 고화력이 특징이지만 기동력이 부족한 반고정포대이기도 하다.  

  * 11위 - Sir.Fire / 더티슬레이머`[5]` : 기동력과 화력을 양립한 표준2각형 AC. 틈이 없는 전투를 전개하는 밸런스 타입, 특히 고화력의 그레네이드에 주의를 필요로 한다.

  * 16위 - 四海之盃 / ChickTack : 무기형의 완부를 사용하는 경량역각형 AC. 머신건에 의한 근거리 모드에 더해 미사일에 의한 원거리 모드를 갖는다.

  * 18위 - 프리=리사 / 메리=리사 : 경이적인 속도를 자랑하는 플로트형 AC. 스피드를 살린 고속전투를 전개하여, 중거리에서 안정된 전투를 계속한다.

  * 22위 - 게이브·그로우 / Bad-News : 무기형의 완부를 사용하는 4각형 AC. 기동력을 살린 교란전을 전문으로 적극적으로 ECM 메이커를 사용한다.  

  * 23위 - 탕크스 아루아 / 킬러 비글 : 무기형의 완부를 사용하는 탱크형 AC. 실탄방어를 중시한 중장타입. 화력이 높은 그레네이드에 주의를 필요로 한다.  

  * 24위 - 楊 / 라·루`[10]` : 보조 부스터를 장착한 경량2각형 AC. 강력한 하이 레이저 라이플을 구사하나, 기체 부하가 높고 종합적인 성능은 낮다.  

  * 25위 - 지니 / 시노비 : 근거리전을 전문으로 하는 경량2각형 AC. EO와 블레이드를 주력무장으로 하여 적극적인 접근으로부터 고속전투를 전개한다.
