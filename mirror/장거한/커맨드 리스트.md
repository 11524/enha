# 12. [CVS2](CVS2.md) ¶

  

★ 잡기  

  * 파안격 : 근접해서 ← or → + 강P 
  * 휙 던지기 : 근접해서 ← or → + 강K 
  * 휙 던지기 & 하혈 찌르기 : 근접해서 ← or → + 강K (최번개가 근처에 있을 때만 사용 가능)   

★ 특수기  

  * 뺑소니 : ↘ + 강P  

★ 필살기  

  * 철구대회전 : P 연타 (PP로 강제 캔슬 가능)
  * 철구분쇄격 : ← 모으고 → + P
  * 대파괴던지기 : →↘↓↙←→ + P (잡기)
  * 비상공열참 (최번개) : ←↙↓↘→ + K 
  * 용권질풍참 (최번개) : →↘↓↙← + K 
  * 질주비상참 (최번개) : →↓↘ + K   

★ 슈퍼 콤보 or 초필살기  

  * 철구대폭주 : ↓↘→↘↓↙← + P
  * 진! 초절용권진공참 (최번개) : →↘↓↙←→↘↓↙← + K
