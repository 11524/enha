여러 [운영체제](%EC%9A%B4%EC%98%81%EC%B2%B4%EC%A0%9C.md)에서
[키보드](%ED%82%A4%EB%B3%B4%EB%93%9C.md)로 사용 가능한 단축키는 [키보드단축키](%ED%82%A4%EB%B3%B4%EB%93%9C%20%EB%8B%A8%EC%B6%95%ED%82%A4.md)로, 단축키는
한/영 키로 영어 설정이 된 상태에서 쓸 수 있다. 안드로이드 스마트폰에서는 키보드가 있는 폰과 강제호출 기능을 사용하면 사용할 수 있다.
메뉴 버튼을 길게 누르면 키보드가 호출되며, 문자입력시 검색창이 뜨지만 닫으면 단축키가 인식되어 작동하는 것을 볼 수 있다.

# 1. 리그베다 위키 단축키 목록 ¶

  * 참고: [엔하위키 미러](%EC%97%94%ED%95%98%EC%9C%84%ED%82%A4%20%EB%AF%B8%EB%9F%AC.md)에서는 A와 Backspace 말고는 쓸 수 없다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%8B%A8%EC%B6%95%ED%82%A4?action=
edit&section=2)]

## 1.1. 탐색 관련 기능(shortcut for Browsing) ¶

  * **F**ront page - 처음화면(H로도 가능) 
  * **Q**uery, **S**earch - 검색
  * r**A**ndom - 랜덤 페이지
  * recent**C**hanges - 최근 바뀐 글 페이지 표시
  * **K**eyword - 키워드 검색
  * **L**ike - 비슷한 페이지 검색
  * **R**eload - 새로고침  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%8B%A8%EC%B6%95%ED%82%A4?action=
edit&section=3)]

## 1.2. 편집 관련 기능(shortcut for Editing) ¶

  * **W**rite / **E**dit - 수정
  * **D**ifference - 이전 수정대비 비교
  * h**I**story - 과거 수정로그
  * **P**rint - 프린트용 페이지  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%8B%A8%EC%B6%95%ED%82%A4?action=
edit&section=4)]

## 1.3. 기타 단축키(other shortcuts) ¶

  * **U**ser - 유저 정보(로그인, 사용자 등록)
  * **B**ookmark - 즐겨찾기
  * **Z**oom - 검색창 입력`[1]`  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EB%8B%A8%EC%B6%95%ED%82%A4?action=
edit&section=5)]

## 1.4. [모질라 파이어폭스](%EB%AA%A8%EC%A7%88%EB%9D%BC%20%ED%8C%8C%EC%9D%B4%EC%96%B4%ED%8F%AD%EC%8A%A4.md), [오페라](%EC%98%A4%ED%8E%98%EB%9D%BC%28%EC%9B%B9%20%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80%29.md)에서 작동하는 단축키`[2]` ¶

알파벳 단축키 외에도 F1 등의 단축키를 이용할 수 있다. 아래에 없는 단축키(F3, F5 등)들은 파폭에서 고유기능이 있는 단축키이므로
넣지 않았다. <del>사실 아래 단축키들의 효율은(...)</del>

  

  * F1 - 프린트용 페이지. 단축키 P와 같다.
  * F2,F4 - 검색. 단축키 Q/S와 같다.
  * F6 - 유저 정보(로그인, 사용자 등록). 단축키 U와 같다.
  * F8 - 수정. 단축키 W/E와 같다.
