※ 버튼 (4버튼 플레이 게임 기준.`[1]`)  

  * 약 펀치 : A
  * 약 킥 : B
  * 강 펀치 : C
  * 강 킥 : D
  * KOF XI 날리기 공격 : E

# 1. [KOF 97](KOF%2097.md) ¶

★ 기본잡기  

  * 팔철 : 근접해서 ← or → + C
  * 일찰 업어치기 불완전 : 근접해서 ← or → + D  

★ 특수기  

  * 외식 굉부 모양만 : → + B  

★ 필살기  

  * 백식 귀신태우기 미완성 : →↓↘ + A or C
  * 백십사식 황물기 미완성 : ↓↘→ + A
  * 백십오식 독물기 미완성 : ↓↘→ + C
  * 백일식 농차 미완성 : ↓↙← + B or D
  * 신고 킥 : ←↙↓↘→ + B or D  

★ 초필살기  

  * 버닝 신고 : ↓↙←↙↓↘→ + A or C
  * 외식 봉린달리기 : ↓↘→↓↘→ + A or C
