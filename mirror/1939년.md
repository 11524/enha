# 1. 개요 ¶

전 세계가 그토록 두려워하던 또 다른 대규모 전면전쟁, [제2차세계대전](%EC%A0%9C2%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)이 결국
발발하였다. 그러나 1939년 [전쟁](%EC%A0%84%EC%9F%81.md)이 터질 당시만 해도 대다수의 유럽인들은 이 전쟁이
세계대전이 될 거라고는 미처 생각지 못하고 **적당히 싸우다 멈출 거**라고 생각했다.

## 2.1. 실제 ¶

  * [2월 10일](2%EC%9B%94%2010%EC%9D%BC.md) : [중일전쟁](%EC%A4%91%EC%9D%BC%EC%A0%84%EC%9F%81.md) 중 [일본](%EC%9D%BC%EB%B3%B8.md)이 하이난 섬을 침공했다.
  * [3월 2일](3%EC%9B%94%202%EC%9D%BC.md) : [비오 12세](%EB%B9%84%EC%98%A4%2012%EC%84%B8.md)가 로마 교황으로 취임했다.
  * [3월 15일](3%EC%9B%94%2015%EC%9D%BC.md) : [독일](%EB%8F%85%EC%9D%BC.md)이 뮌헨 협정을 파기하고 [체코슬로바키아](%EC%B2%B4%EC%BD%94%EC%8A%AC%EB%A1%9C%EB%B0%94%ED%82%A4%EC%95%84.md)를 강제 병합했다.
  * [4월 1일](4%EC%9B%94%201%EC%9D%BC.md) : [스페인 내전](%EC%8A%A4%ED%8E%98%EC%9D%B8%20%EB%82%B4%EC%A0%84.md)이 종식되었다.
  * [4월 7일](4%EC%9B%94%207%EC%9D%BC.md) : [이탈리아](%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84.md)가 [알바니아](%EC%95%8C%EB%B0%94%EB%8B%88%EC%95%84.md)를 침공, 병합했다.
  * [6월 24일](6%EC%9B%94%2024%EC%9D%BC.md) : [태국](%ED%83%9C%EA%B5%AD.md)이 국호를 시암에서 타일랜드(Thailand)로 변경했다.
  * [7월 25일](7%EC%9B%94%2025%EC%9D%BC.md) : [한국](%ED%95%9C%EA%B5%AD.md)에서 경춘철도, 즉 [경춘선](%EA%B2%BD%EC%B6%98%EC%84%A0.md)이 개통되었다.
  * [8월 2일](8%EC%9B%94%202%EC%9D%BC.md) : [알베르트 아인슈타인](%EC%95%8C%EB%B2%A0%EB%A5%B4%ED%8A%B8%20%EC%95%84%EC%9D%B8%EC%8A%88%ED%83%80%EC%9D%B8.md)이 [프랭클린 D. 루스벨트](%ED%94%84%EB%9E%AD%ED%81%B4%EB%A6%B0%20D.%20%EB%A3%A8%EC%8A%A4%EB%B2%A8%ED%8A%B8.md)에 핵개발 연구를 제안하는 편지를 썼다.
  * [8월 23일](8%EC%9B%94%2023%EC%9D%BC.md) : 독일과 [소련](%EC%86%8C%EB%A0%A8.md)이 독소 불가침조약인 몰로트프-리벤트로프 조약을 체결했다. 겉보기엔 불가침조약인 이 조약의 숨겨진 이면은 동유럽을 어떤 식으로 분할할지 결정한 것이었다.
  * [9월 1일](9%EC%9B%94%201%EC%9D%BC.md) : 독일군이 [폴란드 침공](%ED%8F%B4%EB%9E%80%EB%93%9C%20%EC%B9%A8%EA%B3%B5.md) 작전인 백색 작전을 시작했다. **[제2차 세계대전](%EC%A0%9C2%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)의 시작**이었다.
  * [9월 3일](9%EC%9B%94%203%EC%9D%BC.md) : 영국과 프랑스가 독일에 선전포고했다.
  * [9월 16일](9%EC%9B%94%2016%EC%9D%BC.md) : [할힌골 전투](%ED%95%A0%ED%9E%8C%EA%B3%A8%20%EC%A0%84%ED%88%AC.md)가 종결되었다.
  * [11월 10일](11%EC%9B%94%2010%EC%9D%BC.md) : [일본](%EC%9D%BC%EB%B3%B8.md) [조선총독부](%EC%A1%B0%EC%84%A0%EC%B4%9D%EB%8F%85%EB%B6%80.md)가 조선민사령을 발표하고 창씨개명을 구체화하였다.
  * [11월 30일](11%EC%9B%94%2030%EC%9D%BC.md) : 소련이 [핀란드](%ED%95%80%EB%9E%80%EB%93%9C.md) 침공을 시작했다. [겨울전쟁](%EA%B2%A8%EC%9A%B8%EC%A0%84%EC%9F%81.md)의 발발.

[[edit](http://rigvedawiki.net/r1/wiki.php/1939%EB%85%84?action=edit&section=4

## 2.2. 가상 ¶

  * [8월 18일](8%EC%9B%94%2018%EC%9D%BC.md) : [미야후지 요시카](%EB%AF%B8%EC%95%BC%ED%9B%84%EC%A7%80%20%EC%9A%94%EC%8B%9C%EC%B9%B4.md)의 생일에 아버지 [미야후지 이치로](%EB%AF%B8%EC%95%BC%ED%9B%84%EC%A7%80%20%EC%9D%B4%EC%B9%98%EB%A1%9C.md)의 사망소식이 브리타니아로부터 전해졌다. - [스트라이크 위치스](%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9D%B4%ED%81%AC%20%EC%9C%84%EC%B9%98%EC%8A%A4.md)  

    * [네우로이](%EB%84%A4%EC%9A%B0%EB%A1%9C%EC%9D%B4.md)들이 [유럽](%EC%9C%A0%EB%9F%BD.md)에 쳐들어오면서 [제2차 세계대전](%EC%A0%9C2%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md)이 시작되었다. - [스트라이크 위치스](%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9D%B4%ED%81%AC%20%EC%9C%84%EC%B9%98%EC%8A%A4.md)
