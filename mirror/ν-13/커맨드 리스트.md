### 2. [Blazblue : Continuum Shift](BlazBlue%20%3A%20Continuum%20Shift.md)
¶

![CA012_Nu.png](//rv.wkcdn.net/http://rigvedawiki.net/r1/pds/CA012_Nu.png)

[PNG image (459.44 KB)]

  
컨티뉴엄 시프트 공식 일러스트

  

[Λ-11](%CE%9B-11.md)의 언리미티드 버전으로 출현. 자세한 건 [Λ-11/커맨드 리스트](%CE%9B-11/%EC%BB%A4%EB%A7%A8%EB%93%9C%20%EB%A6%AC%EC%8A%A4%ED%8A%B8.md)를 참고하길 바라며 이 항목은 변경점만
서술한다.

  

  * 소드 서머너가 한번에 2개 소환
  * ↓C의 발동 속도가 전작과 마찬가지로 빠름.
  * 그래비티 실드가 전작의 것으로 변경. 남발이 가능하다.
  * 시클 스톰이 전작의 성능으로 변경.
  * 스파이크 체이서가 C 입력 없이도 화면 끝까지 나감.
  * 전작의 액트 펄서가 부활.
  * 액트 펄서 쯔바이(A 액트 펄서)가 다가와서 기본잡기를 사용하는 커맨드 잡기 기술로 변화
  * 액트 펄서 쯔바이 블레이드(B 액트 펄서) 후 액트 펄서 쯔바이 기병대(C 액트 펄서)가 연속 히트
  * C 액트 펄서 히트 시의 벽 바운드 효과가 사라짐.
  * 히트 게이지가 자동으로 상승.
  * 레거시 엣지의 공격횟수가 증가.
  * 캘러미티 소드 4연발.  

★ 이 항목은 블레이블루 컨티니엄 쉬프트 익스텐드에 한함 ★

  

★ 숨겨진 필살기  
기술명 불명 : ↓↘→↓↘→ + B

크로노판타즈마 공식 일러스트

  

★ 드라이브 : 소드 서머너  

  * D
  * → + D
  * ← + D
  * ↓ + D
  * 공중에서 D
  * 공중에서 ↓ + D  

★ 오버 드라이브 : 언리미티드 드라이브  

  * 드라이브 사용시 출력되는 검의 수가 증가한다. 이 드라이브를 사용하면 언리미티드 뉴처럼 소드 서머너가 2개씩 나가거나 디스토션 드라이브 레거시 엣지와 캘러미티 소드의 검 출력횟수가 늘게된다.  

★ 특수기  

  * → + A
  * → + B
  * ← + B
  * → + C
  * ↘ + C
  * 공중에서 ↓ + C  

★ 필살기 - CP에서 뉴는 디아포르마, 루나포르마 두가지 모드로 나뉘어졌으며 기술및 성능도 두가지 모드에 따라 달라졌다.  

  * 공통  

    * **페이스 리프트** : ↓↓ + A or 히트시 ↓↓ + A`[2]`
    * **스프라 레이지** : →↓↘ + A
  * 스타일에 따라 변화  

    * 스파이크 체이서 : ↓↙← + D (디아포르마일 때만 C로 변화)
    * 시클 스톰 : ↓↘→ + D (디아포르마일 때만 C로 변화)
    * 크레센트 세이버 : 공중에서 ↓↙← + D (디아포르마일 때만 C로 변화)
    * 그래비티 실드 : ↓↙← + A or B or C
  * 디아포르마 한정  

    * 액트 펄서 : 특정 기술 사용 후 →→ or ←←
  * 루나포르마 한정  

    * 액트 펄서-쯔바이 : ↓↘→ + A
    * 액트 펄서-쯔바이 블레이드 : ↓↘→ + B
    * 액트 펄서-쯔바이 카발리에 : ↓↘→ + C  

★ 디스토션 드라이브  

  * 레거시 엣지 : ↓↘→↓↘→ + D (디아포르마 한정)
  * 캘러미티 소드 : →↘↓↙←→ + D (루나포르마 한정/공중 가능)  

★ 아스트랄 히트  

  * 멸망의 검 : ↓↙←↓↙← + D
