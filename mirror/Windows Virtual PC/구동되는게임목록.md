## 분류 ¶

Windows Virtual PC 또는 Microsoft Virtual PC 2007에서 어느 정도 구동가능한 게임을 적어야 하며
Windows Virtual PC에서 구동가능한 것은 'WVPC'라고 적고 Microsoft Virtual PC 2007에서 구동 가능한
게임은 'MSVPC2007'로 표기하고, [Hyper-V](Hyper-V.md)에서 구동 가능한 게임은 'HV'로 표기하며, 전부 구동
가능한 게임은 'all'로 표기해야 한다.
