# 1. [KOF MI](KOF%20MI.md) ¶

  

★ 기본잡기  

  * 철화: 근접해서 ← or → + C
  * 열화: 근접해서 ← or → + D  

★ 특수기  

  * 골파: → + A  

★ 필살기  

  * 참광(斬光): ↓↙← + A or C
  * 풍방권: ↓↘→ + A or C
  * 뇌방권: ↓↘→ + B or D
  * 괵룡: →↓↘ + A or C
  * 박룡산타: →↘↓↙← + D (잡기)  

★ 초필살기(☆은 기게이지 소모 개수 표시)  

  * 비오의 개문뇌신권(☆): ↓↘→↓↘→ + A
  * 궁극비오의 패왕뇌광권(☆☆): ↓↘→↓↘→ + C  

★ 스타일리쉬 아트  

  * 01: A ▶ A ▶ C ▶ → + C ▶ → + C
  * 02: A ▶ A ▶ C ▶ → + C ▶ → + D
  * 03: B ▶ D
  * 04: B ▶ B ▶ B or D ▶ B or D ▶ B or D
  * 05: → + B ▶ → + D
  * 06: → + B ▶ ↓ + D
  * 07: ↘ + B ▶ B ▶ D
  * 08: ↘ + B ▶ B ▶ ↓ + D
  * 09: ↘ + B ▶ ↓ + B ▶ D
  * 10: ↘ + B ▶ C ▶ C
  * 11: C ▶ C ▶ C
  * 12: C ▶ C ▶ ← + A ▶ → + C
  * 13: C ▶ C ▶ ← + B ▶ → + D
  * 14: C ▶ C ▶ ← + D (페인트)
  * 15: C ▶ ↘ + D
  * 16: C ▶ ↘ + B ▶ ↘ + D
  * 17: C ▶ → + C ▶ → + C
  * 18: → + C ▶ C ▶ → + C
