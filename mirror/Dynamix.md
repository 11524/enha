## 2. 특징 ¶

  

게임 플레이는 [Deemo](Deemo.md)와 비슷하나, 다른 모바일
[노트](%ED%83%AD%EC%86%8C%EB%8B%89.md) [낙하형](O2JAM.md)
[리듬게임](Deemo.md)과 달리 노트가 **좌,우**에서도 나온다.

  

[DJMAX](DJMAX.md)시리즈 처럼 게임내 랭크가 있어서 랭크로 곡이 해금된다.

  

초기에 핸드폰간 연동이 안되어서 게임 데이터에 불편한 점이 많았다. 그래서 안드로이드 폴더
내의sdcard/Android/data/com.c4cat.  
dynamix/files/Save에 있는 save.save와 save_backup.save 두 파일을 옮기는 형태로 세이브를 해야 했다.

  

2014년 12월 30일, 2.0 업데이트로 클라우드 저장기능이 생겼다. 옵션에서 Syn으로 되어있는 부분에 체크를 하면 클라우드와
동기화되어서, 더 이상 위에서 서술된 방법을 쓰지 않아도 다른 핸드폰으로 같은 데이터를 쓸 수 있다.

  * C : 699999~600000점.

### 2.4. 이펙터 ¶

  

일종의 게임머니인 300Bit로 프리미엄 팩을 구매하면 사용이 가능하다. 현재는 4가지의 이펙터가 구현.  

  * Bleed : 화면 중앙에 게이지가 생긴다. 미스가 나면 게이지가 줄어들며, 미스 없이 이어지다보면 게이지가 차오른다.게이지가 다 떨어지면 게임오버. 20랭크가 되어서 메가난이도 플레이 시 프리미엄팩을 사지 않았다면 이 이펙트가 고정되어있다(...).
  * Mirror : 좌우 화면이 반전된다.
  * Narrow : 노트가 얇게 나온다. 75, 50, 25%로 나누어져 있다.
  * Auto : 지정 방향의 노트를 자동으로 플레이 할 수 있다. 좌, 우, 좌우, 모든 방향으로 나누어져 있다.  

[[edit](http://rigvedawiki.net/r1/wiki.php/Dynamix?action=edit&section=7)]

### 2.5. 난이도 체계 ¶

  

  * BASIC : 1~4.
  * NORMAL : 4~7. 본격적으로 좌우노트가 나온다.
  * HARD : 7~10. <del>그래도 여기까진 엄지플이 가능하다</del>
  * MEGA : 11~14. 2.0버전에서 Extra 항목을 선택하면 나오는 가장 어려운 난이도.
  * <del>GIGA</del> : 추후에 업데이트로 추가될 난이도.[영상](http://www.youtube.com/watch?v=E5aWfaUdFzI)
