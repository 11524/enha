# 1. [아랑 MOW](%EC%95%84%EB%9E%91%20MOW.md) ¶

  

★ 기본잡기  

  * 바이바이 부 : 근접해서 ← or → + C
  * 폴링 크러쉬 : 공중 근접해서 ↑ 이외 방향 + C  

★ 특수기  

  * 슈퍼점프 : ↓↑  

★ 컴비네이션  

  * 근접해서 A - A - A
  * 원거리 C - C
  * 수직 점프 D - A  

★ 페인트기  

  * 메니메니 토피도 페인트 : → + AC
  * 오로라 페인트 : ↓ + AC  

★ TOP기  

  * 롤링 썬더 : CD  

★ 필살기  

  * 배플스 : ↓↘→ + A or C
  * 크레이지 이반 : ↓↙← + A or C
  * 걸프 토마호크 : ↓↙← + B or D
  * D.하인드 : ↓↘→ + B or D
  * 해리어.B : 공중에서 ↓ + B or D 연타  

★ 초필살기  

  * 메니메니 토피도 : ↓↘→↓↘→ + A
  * 오로라 : ↓↘→↓↘→ + B  

★ 잠재능력  

  * 메니메니 토피도 : ↓↘→↓↘→ + C
  * 오로라 : ↓↘→↓↘→ + D
  * 앙누이 마드모아젤 : 저스트 디펜스 중 A, B, C
