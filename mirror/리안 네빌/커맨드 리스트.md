# 1. [KOF MI](KOF%20MI.md) ¶

  

★ 기본잡기  

  * 카프: 근접해서 ← or → + C
  * 주르: 근접해서 ← or → + D  

★ 필살기  

  * 어설트 타입ι【시르마】: ↓↘→ + A or C
  * 어설트 타입η【자이아】: →↓↘ + A or C (모으기 가능)
  * 어설트 타입γ【사다크비아】: ↓↙← + A or B or C or D
  * 어설트 타입δ【예드 프리어】: ← 모으고 → + A
  * 어설트 타입ε【예드 포스테리어】: ← 모으고 → + C
  * 어설트 타입σ【알 니야트】: →↘↓↙← + A or C (잡기)
  * 어설트 타입π【아젤파파지】: 공중에서 ↓↙← + B or D  

★ 초필살기 (☆은 기게이지 소모 개수 표시)  

  * 어설트 타입λ【샤울라】(☆): ↓↙←↙→ + D
  * 어설트 타입β【알 타르프】(☆☆): ↓↙←↙→ + C  

★ 스타일리쉬 아트  

  * 01 : A ▶ A ▶ A ▶ C
  * 02 : A ▶ A ▶ A ▶ D ▶ D
  * 03 : A ▶ A ▶ A ▶ ↓ + D
  * 04 : →+ A ▶ D
  * 05 : →+ A ▶ ↓ + D
  * 06 : ↓ + A ▶ ↓ + C ▶ C ▶ → + C
  * 07 : → + B ▶ B ▶ D ▶ D
  * 08 : ↓ + B ▶ B ▶ C ▶ → + C
  * 09 : C ▶ C ▶ D ▶ D
  * 10 : C ▶ C ▶ D ▶ ↓ + D
  * 11 : → + C ▶ C ▶ C ▶ C
  * 12 : → + C ▶ C ▶ ← + C (페인트)
  * 13 : ← + C ▶ ← + C
  * 14 : ↘ + C ▶ → + C ▶ ← + C
  * 15 : ↘ + C ▶ ← + C (페인트)
  * 16 : D ▶ D ▶ D
  * 17 : D ▶ D ▶ ↓ + D
  * 18 : → + D ▶ C ▶ → + C ▶ → + C
  * 19 : → + D ▶ D ▶ D
  * 20 : → + D ▶ ↓ + D
  * 21 : 공중에서 A ▶ C
