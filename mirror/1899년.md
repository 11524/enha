# 1. 개요 ¶

[대한제국](%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD.md)에
[경인선](%EA%B2%BD%EC%9D%B8%EC%84%A0.md) [철도](%EC%B2%A0%EB%8F%84.md)가 최초로
부설되었다.

## 2.1. 실제 ¶

  * [1월 23일](1%EC%9B%94%2023%EC%9D%BC.md) : [필리핀](%ED%95%84%EB%A6%AC%ED%95%80.md), 말로로스 [헌법](%ED%97%8C%EB%B2%95.md)을 제정하며 [아시아](%EC%95%84%EC%8B%9C%EC%95%84.md) 최초의 [공화국](%EA%B3%B5%ED%99%94%EA%B5%AD.md)인 필리핀 공화국을 선포.
  * [9월 18일](9%EC%9B%94%2018%EC%9D%BC.md) : [대한제국](%EB%8C%80%ED%95%9C%EC%A0%9C%EA%B5%AD.md)에서 [노량진역](%EB%85%B8%EB%9F%89%EC%A7%84%EC%97%AD%28%EA%B2%BD%EB%B6%80%EC%84%A0%29.md)과 [제물포역](%EC%A0%9C%EB%AC%BC%ED%8F%AC%EC%97%AD.md)`[2]` 사이에 [경인선](%EA%B2%BD%EC%9D%B8%EC%84%A0.md) [철도](%EC%B2%A0%EB%8F%84.md) 33.2㎞가 개통되었다.
  * [10월 11일](10%EC%9B%94%2011%EC%9D%BC.md) : 2차 [보어 전쟁](%EB%B3%B4%EC%96%B4%20%EC%A0%84%EC%9F%81.md) 발발.
  * [11월 29일](11%EC%9B%94%2029%EC%9D%BC.md) : [스페인](%EC%8A%A4%ED%8E%98%EC%9D%B8.md) [축구](%EC%B6%95%EA%B5%AC.md) 클럽 [FC 바르셀로나](FC%20%EB%B0%94%EB%A5%B4%EC%85%80%EB%A1%9C%EB%82%98.md) 창단.
  * [12월 16일](12%EC%9B%94%2016%EC%9D%BC.md) :  

    * [이탈리아](%EC%9D%B4%ED%83%88%EB%A6%AC%EC%95%84.md) [축구](%EC%B6%95%EA%B5%AC.md) 클럽 [AC 밀란](AC%20%EB%B0%80%EB%9E%80.md) 창단.

## 3.1. 실존인물 ¶

  * [1월 4일](1%EC%9B%94%204%EC%9D%BC.md) : [제2차 세계대전](%EC%A0%9C2%EC%B0%A8%20%EC%84%B8%EA%B3%84%EB%8C%80%EC%A0%84.md) 당시 [나치 독일](%EB%82%98%EC%B9%98%20%EB%8F%85%EC%9D%BC.md) 군인 [프리츠 바이에를라인](%ED%94%84%EB%A6%AC%EC%B8%A0%20%EB%B0%94%EC%9D%B4%EC%97%90%EB%A5%BC%EB%9D%BC%EC%9D%B8.md)
  * [2월 27일](2%EC%9B%94%2027%EC%9D%BC.md) : 대한민국의 여성운동가 [김활란](%EA%B9%80%ED%99%9C%EB%9E%80.md) <del>근데 이인간 [친일파](%EC%B9%9C%EC%9D%BC%ED%8C%8C.md)아니던가</del>
  * [5월 8일](5%EC%9B%94%208%EC%9D%BC.md) : 오스트리아의 [경제학](%EA%B2%BD%EC%A0%9C%ED%95%99.md)자 [프리드리히 하이에크](%ED%94%84%EB%A6%AC%EB%93%9C%EB%A6%AC%ED%9E%88%20%ED%95%98%EC%9D%B4%EC%97%90%ED%81%AC.md)`[3]`
  * [6월 14일](6%EC%9B%94%2014%EC%9D%BC.md) : 일본 [소설가](%EC%86%8C%EC%84%A4%EA%B0%80.md) [가와바타 야스나리](%EA%B0%80%EC%99%80%EB%B0%94%ED%83%80%20%EC%95%BC%EC%8A%A4%EB%82%98%EB%A6%AC.md)
  * [7월 21일](7%EC%9B%94%2021%EC%9D%BC.md) : 미국의 [소설가](%EC%86%8C%EC%84%A4%EA%B0%80.md) [어니스트 헤밍웨이](%EC%96%B4%EB%8B%88%EC%8A%A4%ED%8A%B8%20%ED%97%A4%EB%B0%8D%EC%9B%A8%EC%9D%B4.md)
  * [8월 13일](8%EC%9B%94%2013%EC%9D%BC.md) : 미국의 [영화](%EC%98%81%ED%99%94.md)감독 [앨프리드 히치콕](%EC%95%A8%ED%94%84%EB%A6%AC%EB%93%9C%20%ED%9E%88%EC%B9%98%EC%BD%95.md)
  * [8월 23일](8%EC%9B%94%2023%EC%9D%BC.md) : 한국의 독립운동가 [김법린](%EA%B9%80%EB%B2%95%EB%A6%B0.md)
  * [8월 24일](8%EC%9B%94%2024%EC%9D%BC.md) : 아르헨티나의 작가 [호르헤 루이스 보르헤스](%ED%98%B8%EB%A5%B4%ED%97%A4%20%EB%A3%A8%EC%9D%B4%EC%8A%A4%20%EB%B3%B4%EB%A5%B4%ED%97%A4%EC%8A%A4.md)
  * [8월 28일](8%EC%9B%94%2028%EC%9D%BC.md) : 대한민국의 [정치인](%EC%A0%95%EC%B9%98%EC%9D%B8.md) [장면](%EC%9E%A5%EB%A9%B4.md)
  * [9월 20일](9%EC%9B%94%2020%EC%9D%BC.md) : 미국의 [정치](%EC%A0%95%EC%B9%98.md) [철학자](%EC%B2%A0%ED%95%99%EC%9E%90.md) [레오 스트라우스](%EB%A0%88%EC%98%A4%20%EC%8A%A4%ED%8A%B8%EB%9D%BC%EC%9A%B0%EC%8A%A4.md)
  * [11월 9일](11%EC%9B%94%209%EC%9D%BC.md) : 한국의 아동문학가 [방정환](%EB%B0%A9%EC%A0%95%ED%99%98.md)
  * [12월 2일](12%EC%9B%94%202%EC%9D%BC.md) : 영국의 지휘자 [존 바비롤리](%EC%A1%B4%20%EB%B0%94%EB%B9%84%EB%A1%A4%EB%A6%AC.md)
