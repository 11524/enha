# 2. 추가/수정 규칙 ¶

  * 맵을 위한 문서를 새로 작성하고자 하는 경우, 동음이의 문서가 있는지 먼저 검색을 해보고 없다면 그대로 개설해도 되지만 있다면 '맵이름(스타크래프트)'로 분리하는 것을 권장한다.  
예: 로스트 템플 -> 동음이의 문서가 없으니 [로스트템플](%EB%A1%9C%EC%8A%A4%ED%8A%B8%20%ED%85%9C%ED%94%8C.md)로 항목 생성, 티아매트 ->
[티아매트](%ED%8B%B0%EC%95%84%EB%A7%A4%ED%8A%B8.md) 문서가 있으니 [티아매트(스타크래프트)](/wik
i/%ED%8B%B0%EC%95%84%EB%A7%A4%ED%8A%B8%28%EC%8A%A4%ED%83%80%ED%81%AC%EB%9E%98%
ED%94%84%ED%8A%B8%29)로 항목 생성.  

  * 본 항목에 맵을 추가 할 경우 되도록 기존에 존재하는 항목 목록들과 가나다 -> ABC -> 123 순서를 맞추어서 추가한다.
