## 개요 ¶

[윈도우](Microsoft%20Windows.md)에 내장된 [가상머신](Windows%20Virtual%20PC.md).

  

[Windows Server 2008](Windows%20NT.md)의 특정 버전부터 사용할 수 있었으며, 이전에는 **Windows
Server Virtualization**라는 이름으로 불렸다. 말그대로 서버 가상화를 위해 나온 가상머신이다. 그렇지만, 서버용이 아닌
[Windows 8](Windows%208.md) 프로에도 탑재됬다.

  

[Windows 7](Windows%207.md)의 [XP](Windows%20XP.md)모드와 달리, Guest OS로 윈도우
뿐만 아니라 [리눅스](%EB%A6%AC%EB%88%85%EC%8A%A4.md)도 지원한다.`[1]` 대신 사운드를 에뮬레이션하지
않아서 게임용으로 쓰기에는 [영 좋지않다](%EC%98%81%20%EC%A2%8B%EC%A7%80%20%EC%95%8A%EB%8B%A4.md). 게임하려면
[VMware](VMware.md)나 [VirtualBox](VirtualBox.md)를 쓰자. 더구나
[CPU](CPU.md)가 [SLAT(Second Level Address
Translation)](http://en.wikipedia.org/wiki/Second-
level_address_translation)`[2]`이라는 기능을 지원해야 하기 때문에 더더욱 쓰기 힘든 편. <del>i7을 [강요받고있는 거다!](%EA%B0%95%EC%9A%94%EB%B0%9B%EA%B3%A0%20%EC%9E%88%EB%8A%94%20%EA%B1%B0%EB%8B%A4%21.md)</del>

  

경쟁제품은 널리 알려진 [VMware](VMware.md)사의 VMware ESX다.
