# 1. [용호의 권외전](%EC%9A%A9%ED%98%B8%EC%9D%98%20%EA%B6%8C%20%EC%99%B8%EC%A0%84.md) ¶

  

★ 기본잡기  

  * 잡아돌려 앞떨구기 : 근접해서 → + C  

★ 특수기  

  * 굉섬 : AB
  * 안개떨치기 : 회피 중 A
  * 백파 : ← + B
  * 쌍장탄 : →→ + A  
┠ 쌍장창돌각 : 쌍장탄 후 ← + B  
┗ 쿠사나기 : 쌍장탄 후 ↓ + B

  * 비창 : →→ + B
  * 살축음타 : 상대의 중단 펀치와 동시에 ← + A
  * 살장음축 : 상대의 상단 펀치와 동시에 ← + A
  * 갑옷깨기 : 상대방 다운시 ↘ + A or B  

★ 필살기  

  * 토도류 겹쳐치기 : ↓↘→ + A
  * 토도류 뇌포공 : ↓↙← + A  

★ 초필살기  

  * 토도류 비전 초겹쳐치기 : ←→↘↓↙← + A
