## 1. 수 ¶

위에서 적었듯, 7999보다 크고 8001보다는 작은 수이다. 자연수에서 이 수를 기점으로 천 단위 자리수가 7에서 8로 바뀐다.

  

[[edit](http://rigvedawiki.net/r1/wiki.php/8000?action=edit&section=2)]

### 1.1. 8000이상 9000미만의 수 중 특수한 수 ¶

※이 부분은 영어 위키백과 [8000과](http://en.wikipedia.org/wiki/8000_\(number\)) 한국어 위키백과
[8000](http://ko.wikipedia.org/wiki/8000)을 참고했습니다.  

  * 8001 : 삼각수
  * 8100 : 90의 제곱수
  * 8125 : [완전수](%EC%99%84%EC%A0%84%EC%88%98.md), 조화수, 172번째 삼각수, 64번째 육각수, 8번째 292각수, 4번째 1356각수
  * 8128 : 4번째 완전수
  * 8177 : 74번째 오각수
  * 8191 : 5번째 메르센 소수
  * 8192 : 2의 13제곱수
  * 8208 : 각 자릿수의 4제곱의 합
  * 8281 : 91의 제곱수
  * 8400 : 75번째 오각수
  * 8464 : 92의 제곱수
  * 8626 : 76번째 오각수
  * 8649 : 93의 제곱수
  * 8833 : 88의 제곱 더하기 33의 제곱이 이 수다. (88^2+33^2=8833)
  * 8836 : 94의 제곱수
  * 8855: 77번째 오각수
