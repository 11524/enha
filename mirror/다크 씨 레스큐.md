[유희왕](%EC%9C%A0%ED%9D%AC%EC%99%95.md)에 등장하는 카드. 원작 사용자는
[보머](%EB%B3%B4%EB%A8%B8.md).

**ANPR-JP013 / 다크 씨 레스큐 / 레벨 1 / 어둠 속성 / [기계족](%EA%B8%B0%EA%B3%84%EC%A1%B1.md) / 공격력 : 0 / 수비력 : 0**  
이 카드가 싱크로 몬스터의 싱크로 소환에 사용되어 묘지로 보내졌을 경우, 자신의 덱에서 카드를 1장 드로우 한다.

단순한 능력만 보자면 [튜닝 서포터](%ED%8A%9C%EB%8B%9D%20%EC%84%9C%ED%8F%AC%ED%84%B0.md)의
하위호환. 일단 이쪽은 어둠속성에 공격력 0이라 [악몽의재림](%EC%95%85%EB%AA%BD%EC%9D%98%20%EC%9E%AC%EB%A6%BC.md)등의 어둠속성 서포트를 받을 수
있는 이점을 살려야 한다.

여담이지만 원작에서 [다크 플랫톱](%EB%8B%A4%ED%81%AC%20%ED%94%8C%EB%9E%AB%20%ED%86%B1.md)의 제물이 될 때의 얼빵한
표정이 실로 압권. 거기다가 이때는 **덱에서 카드를 2장 드로우**하는 어이상실의 효과였다.

몇턴 후에 2장째의 다크 씨 레스큐가 등장해 함정 카드 바사라의 제물이 되었다.

**ANPR-JP014 / 다크 씨 플로트 / 레벨 1 / 어둠 속성 / [기계족](%EA%B8%B0%EA%B3%84%EC%A1%B1.md) / 공격력 : 0 / 수비력 : 300**  
필드 위에 존재하는 이 카드가 효과에 의해서 파괴되어 묘지로 보내졌을 때, 자신의 덱에서 카드를 1장 드로우 한다.

원작에서는 [자이언트 폭탄 에어레이드](%EC%9E%90%EC%9D%B4%EC%96%B8%ED%8A%B8%20%ED%8F%AD%ED%83%84%20%EC%97%90%EC%96%B4%EB%A0%88%EC%9D%B4%EB%93%9C.md)의 효과로 **그냥 패에서 슥
버린것뿐인데, 카드를 드로우하는 사기를 보여주었다.**

또한 북미판에서는 씨 레스큐와 마찬가지로 좀비가 사람으로 탈피했다.
