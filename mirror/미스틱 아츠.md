### 2.1. 기술표 ¶

  * 기본 조작  

Z = 공격(펀치 -> 펀치 -> 돌려차기 -> 날리기)  
X = 점프  
C = 공격(Z와 같음)

  * 기술  

→→ + Z = 대시공격(날려보냄)

↑↑ + Z = 올려차기(띄움)  
↓ + Z = 다리걸기(살짝 뜸)  
↘ + Z = 슬라이딩(다리걸기와 같은데 살짝 미끄러짐)  
↓→ + Z = 이단차기  
↓↓ + Z = 전신 실드(몸 전체에 보호막을 쳐 상대를 튕겨냄.)  
↓↑ + Z = 페이크 전신 실드(모션은 같은데 보호막은 안 나가고 타격음도 주먹과 같음.)  
→←→ + Z = 영거리 기공포(세이니의 기술 가운데 콤보수 대비 가장 강력한 기술)  
[↑↑↓↓←→←→ +C](%EC%BD%94%EB%82%98%EB%AF%B8%20%EC%BB%A4%EB%A7%A8%EB%93%9C.md) =
**자살**(진짜로 있다)`[4]`
