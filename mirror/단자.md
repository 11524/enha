端子  
[Terminal](%ED%84%B0%EB%AF%B8%EB%84%90.md) / Socket / Interface / Connector
/ Plug / Jack

[전기](%EC%A0%84%EA%B8%B0.md)회로나 전기기기 등에서 전류의 입력이나 출력부분에 전극을 접속시키기 위해 붙이는
쇠붙이.(by [네이버](%EB%84%A4%EC%9D%B4%EB%B2%84.md) 백과사전)

영상 신호나 음성 신호, 데이터 입출력 신호 등도 모두
[전자기학](%EC%A0%84%EC%9E%90%EA%B8%B0%ED%95%99.md)적 신호`[1]`이기 때문에, [리그베다위키](%EB%A6%AC%EA%B7%B8%EB%B2%A0%EB%8B%A4%20%EC%9C%84%ED%82%A4.md)에서는 주로 이러한
신호를 전송하는 PC나 여러 가전제품을 연결할 때, 어떠한 단자가 있고 어떠한 케이블을 연결해야 하는지 등의 정보를 제공한다.

  * [단자/영상](%EB%8B%A8%EC%9E%90/%EC%98%81%EC%83%81.md)
  * [단자/음성](%EB%8B%A8%EC%9E%90/%EC%9D%8C%EC%84%B1.md)
  * [단자/데이터 입출력](%EB%8B%A8%EC%9E%90/%EB%8D%B0%EC%9D%B4%ED%84%B0%20%EC%9E%85%EC%B6%9C%EB%A0%A5.md)
  * [단자/전원](%EB%8B%A8%EC%9E%90/%EC%A0%84%EC%9B%90.md)
  * [단자/기타](%EB%8B%A8%EC%9E%90/%EA%B8%B0%ED%83%80.md)
  * [단자/휴대전화](%EB%8B%A8%EC%9E%90/%ED%9C%B4%EB%8C%80%EC%A0%84%ED%99%94.md)  

한편, 하나의 단자를 다른 단자로 바꾸는 기기도 있는데 이를 [젠더](%EC%A0%A0%EB%8D%94.md)(또는
[컨버터](%EC%BB%A8%EB%B2%84%ED%84%B0.md))라고 한다.

`\----`

  * `[1]` 대부분은 전류를 쓰지만, 일부 **광신호**를 쓰는 단자(TOSLINK 등)도 있기 때문에 이들을 포괄하는 개념인 '전자기학적 신호'라는 용어를 쓴다.
