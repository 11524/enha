# 2. 상세 ¶

[진 마사토](%EC%A7%84%20%EB%A7%88%EC%82%AC%ED%86%A0.md)가 개발한 초기형의 버스터 머신으로 잠수함
형태의 노멀모드와 개구리형태의 버스터 애니멀 모드로 변형이 가능하다. [CB-01치타](CB-01%20%EC%B9%98%ED%83%80.md),[GT-02고릴라](GT-02%20%EA%B3%A0%EB%A6%B4%EB%9D%BC.md),[RH-03래빗](RH-03%20%EB%9E%98%EB%B9%97.md)과 마찬가지로 개구리형 버디로이드
[에네땅](%EC%97%90%EB%84%A4%EB%95%85.md)이 버스터 머신의 코어유닛을 담당한다.

# 4. 작중 활약 ¶

만들어진후 오래동안 대기상태로 있었으나, 극장판에서 테스트 운전을 겸해 최초로 출동하게 된다. 개구리답게 수중전이 주특기. 치타, 고릴라와
합체하여 고버스터 케로오가 된다.

  

극장판 등장 이후 TV판 44화에서 재등장.

  

[특명전대 고버스터즈 VS 해적전대 고카이저 THE MOVIE](%ED%8A%B9%EB%AA%85%EC%A0%84%EB%8C%80%20%EA%B3%A0%EB%B2%84%EC%8A%A4%ED%84%B0%EC%A6%88%20VS%20%ED%95%B4%EC%A0%81%EC%A0%84%EB%8C%80%20%EA%B3%A0%EC%B9%B4%EC%9D%B4%EC%A0%80%20THE%20MOVIE.md)에서는 [이와사키 류우지](%EC%9D%B4%EC%99%80%EC%82%AC%ED%82%A4%20%EB%A5%98%EC%9A%B0%EC%A7%80.md)
가 탑승하여 1557년 지중해로 날아간다.
