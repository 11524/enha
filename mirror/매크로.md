## 1. 접두어 ¶

「거대한」이라는 의미를 나타내는 접두어로 「아주 작은」이란 뜻의 접두어
[마이크로](%EB%A7%88%EC%9D%B4%ED%81%AC%EB%A1%9C.md)(micro)와는 비교된다. 예를 들어
macrophysics(거시물리학)이 그것. 경제학에서는 microeconomics, 즉 미시경제학과 반대되는 macroeconomics,
즉 거시경제학의 용례가 있다. 최근에는 매크로가「전체, 마이크로가「개별이라는 의미가 되어 매크로는 GNP, 마이크로는 기업의 대명사처럼
쓰이는 경우도 있다.

## 2. 컴퓨터 용어 ¶

자주 사용하는 여러 개의 명령어를 묶어서 하나의 키 입력 동작으로 만든 것. 어셈블리 언어에서 주로 많이 사용되며, 관련 프로그램으로는
[스프레드시트](%EC%8A%A4%ED%94%84%EB%A0%88%EB%93%9C%EC%8B%9C%ED%8A%B8.md)와

여러번 해야 하는 일을 간단하게 수행하기 위하여 사용하기도 하지만, 문서 안의 같은 문자열을 한꺼번에 변경 할 때도 사용된다.

### 2.3. [리그베다위키](%EB%A6%AC%EA%B7%B8%EB%B2%A0%EB%8B%A4%20%EC%9C%84%ED%82%A4.md)의 매크로 페이지
¶

[모니위키](%EB%AA%A8%EB%8B%88%EC%9C%84%ED%82%A4.md) 설치시 자동으로 인스톨되는 문서들. 각각의 매크로
기능을 설명하는 것이 목적이며 [역링크](%EC%97%AD%EB%A7%81%ED%81%AC.md)도
[존재감](%EC%A1%B4%EC%9E%AC%EA%B0%90.md)(..)도 거의 없고 [편집도움말](%ED%8E%B8%EC%A7%91%20%EB%8F%84%EC%9B%80%EB%A7%90.md)항목이 있는 지금의 리그베다
위키엔 그다지 쓸모가 없는 [페이지](%ED%8E%98%EC%9D%B4%EC%A7%80.md)인데다 해당 기능을 관리자가 꺼놓은
경우들도 있지만 [관리자](%EC%B2%AD%EB%8F%99#s-2.md)는 이 페이지의
[삭제](%EC%82%AD%EC%A0%9C.md)를 금하고 있다.

  

해당 [항목](%ED%95%AD%EB%AA%A9.md)의 [목록](%EB%AA%A9%EB%A1%9D.md).

  

  * [갤러리 매크로](%EA%B0%A4%EB%9F%AC%EB%A6%AC%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : [갤러리](%EA%B0%A4%EB%9F%AC%EB%A6%AC.md)기능을 사용하는 방법을 설명하는 페이지.
  * [랜덤페이지 매크로](%EB%9E%9C%EB%8D%A4%ED%8E%98%EC%9D%B4%EC%A7%80%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : 본 위키의 다른 문서로 통하는 링크롤 무작위로 배열한 목록을 띄우는 페이지.
  * [목차 매크로](%EB%AA%A9%EC%B0%A8%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : 문단을 분리하는 방법을 설명하는 페이지.
  * [바뀐글 매크로](%EB%B0%94%EB%80%90%EA%B8%80%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : [최근 변경내역](RecentChanges.md)과 비슷.
  * [바벨피쉬 매크로](%EB%B0%94%EB%B2%A8%ED%94%BC%EC%89%AC%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : [야후](%EC%95%BC%ED%9B%84.md)의 [바벨피쉬](%EB%B0%94%EB%B2%A8%ED%94%BC%EC%89%AC.md)라는 [번역기](%EB%B2%88%EC%97%AD%EA%B8%B0.md)를 사용해서 위키의 어떤 항목을 [번역](%EB%B2%88%EC%97%AD.md)하는 방법을 설명하는 페이지.
  * [플레이 매크로](%ED%94%8C%EB%A0%88%EC%9D%B4%20%EB%A7%A4%ED%81%AC%EB%A1%9C.md) : **[BGM](BGM.md)**을 올리는 방법을 설명하는 페이지.
