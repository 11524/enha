# 5 [네오지오 배틀 컬리시엄](%EB%84%A4%EC%98%A4%EC%A7%80%EC%98%A4%20%EB%B0%B0%ED%8B%80%20%EC%BB%AC%EB%A6%AC%EC%8B%9C%EC%97%84.md) ¶

  

★ 기본잡기  

  * 발목용: 근접해서 CD  

★ 특수기  

  * 광륜살: → + A  

  * 용회두: ↘ + C  

  * 컴비네이션 A: 근접 C ▶ C ▶ →← + C  

  * 컴비네이션 B: 앉아 A ▶ 앉아 B ▶ 앉아 C  

  * 컴비네이션 C: 앉아 A ▶ 앉아 C  

  * 컴비네이션 D: 앉아 B ▶ 앉아 C  

★ 필살기  

  * 제왕신족권: →→ + A or C  

  * 제왕천안권: ↓↘→ + A or C  

  * 제왕천이권: →↓↘ + A or C  

  * 제왕신안권: →↘↓↙← + A or B or C  
┗ 제왕신안권 강락: 제왕신안권(B 버튼) 도중 공격 버튼

  * 공살신안권: 공중에서 →↘↓↙← + D  
┗ 제왕신안권 강락: 공살신안권 도중 공격 버튼

  * 용등장: ↓↘→ + B or D  
┗ 용등장 환멸: →↘↓↙← B or D  

★ 초필살기  

  * 제왕루진권: ↓↙←→ + A or C  

  * 제왕공살루진권: 공중에서 ↓↙←→ + A or C  

  * 제왕숙명권: ↓↘→↓↘→ + A or C  

  * 해룡조림: ↓↙←↙↓↘→ + A or C(게이지 2줄 소모)
