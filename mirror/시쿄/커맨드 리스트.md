## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 광행 ・ 독 삼키기 : 근접해서 ← or → + B  

★ 특수기  

  * 광행 ・ 내장 찌르기 : 상대 다운시 ↑ + B   
┗ 광행 ・ 내장 뜯기 : 내장 찌르기 중 레버 회전 + 버튼 연타  

★ 필살기  

  * 땅 핥으며 미끄러지기 : →←→ + C 
  * 독취 : ↓↙← + A or B  
┗ 쪼아먹기 : 독취 중 ↑ + B

  * 회전 내장베기 : →↓↘ + A or B   
┗ 때려 부수기 : 회전 내장베기 중 A or B 연타

  * 참육 카마이타치 : ←↙↓↘→ + A or B
  * 참육 큰가위(●) : 공중에서 ↓↙← + B 
  * 무자비 : 근접해서 →↘↓↙← + C  

★ 초오의  

  * 미흉사쇠 ・ 광희 : ↓↘→↓↘→ + AB  
┗ 홍소 : 미흉사쇠 ・ 광희 중 스타트 버튼`[1]`  

┗ 미흉사쇠 ・ 규귀 : 홍소 중 →↘↓↙← + AB  

★ 잠재오의  

  * 미흉사쇠 ・ 흉기 : ↓↘→↓↘→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 광행 ・ 독 삼키기 : 근접해서 ← or → + CD  

★ 특수기  

  * 광행 ・ 내장 찌르기 : 상대 다운시 ↑ + B   
┗ 광행 ・ 내장 뜯기 : 내장 찌르기 중 레버 회전 + 버튼 연타  

★ 필살기  

  * 땅 핥으며 미끄러지기 : →←→ + C 
  * 독취 : ↓↙← + A or B  
┗ 쪼아먹기 : 독취 중 ↑ + B

  * 회전 내장베기 : →↓↘ + A or B   
┗ 회전 내장베기 추가 : 회전 내장베기 중 A or B 연타

  * 참육 카마이타치 : ←↙↓↘→ + A or B
  * 참육 큰가위(●) : 공중에서 ↓↙← + A or B 
  * 무자비 : 근접해서 →↘↓↙← + C  

★ 초오의  

  * 미흉사쇠 ・ 흉기(凶器) : ↓↘→↓↘→ + AB (모으기 가능)  

★ 잠재오의  

  * 미흉사쇠 ・ 흉기(凶飢) : ↓↘→↓↘→ + B  

★ 난무오의  

  * 광희 ・ 먼지 썰어내기 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, ↓↘→ + B  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C,↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, BC,↓↘→ + B
