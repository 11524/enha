종류:GPMG(다목적 기관총)  
구경 : 7.62mm(HK11)/5.56 mm(HK13)  
탄약 : 7.62 x 51mm NATO(HK11)/5.56 x 45mm NATO(HK13)  
작동방식 : 지연 블로우백  
총열길이 : 450mm  
전장 : 1,030mm  
중량 : 7.70kg/8.15kg/8kg(HK11A1/HK11E/HK13E)  
발사속도 : 800발/분  
유효사거리 : 1200m

[H&K](H%26K.md)에서 개발한 [기관총](%EA%B8%B0%EA%B4%80%EC%B4%9D.md)이다.
