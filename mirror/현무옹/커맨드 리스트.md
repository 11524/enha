※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 낚싯대 퍼올리기 : 근접해서 ← or → + B  

★ 특수기  

  * 망태 내리기 : 상대 다운시 ↘ + B  

★ 필살기  

  * 귀서 : ↓↘→ + A or B or C
  * 귀무 ・ 지(●) : →↘↓↙← + A or B
  * 귀무 ・ 천 : →↘↓↙← + C
  * 조과대량 : →↓↘ + A or B or C 
  * 무공용 : ↓↓ + C  

★ 초오의  

  * 현무의 포효 : →←↙↓↘→ + AB   

★ 잠재오의  

  * 현무의 분노 : →←↙↓↘→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 낚싯대 퍼올리기 : 근접해서 ← or → + CD  

★ 특수기  

  * 망태 내리기 : 상대 다운시 ↘ + B  

★ 필살기  

  * 귀서 : ↓↘→ + A or B or C
  * 귀무 ・ 지(●) : →↘↓↙← + A or B
  * 귀무 ・ 천 : →↘↓↙← + C
  * 조과대량 : →↓↘ + A or B or C 
  * 무공용 ・인 : ←↓↙ + A
  * 무공용 ・지 : ←↓↙ + B
  * 무공용 ・천 : ←↓↙ + C  

★ 초오의  

  * 현무의 포효 : →←↙↓↘→ + AB   

★ 잠재오의  

  * 현무의 분노 : →←↙↓↘→ + B   

★ 난무오의  

  * 현무의 춤 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, ↓↘→ + A  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, BC, A, ↓↘→ + B
