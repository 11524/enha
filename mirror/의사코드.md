  * 상위 항목: [프로그래밍](%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%98%EB%B0%8D.md)  

pseudo code.

프로그램 코드 중 하나.

프로그램 코드와 비슷하지만 조금 더(...) 알아보기 쉽게 문법을 약간 무시, 생략, 단순화하여 사람의 말에 가깝게 만들어 놓은 코드를
말한다.

쉽게 말해서 진짜 코드인 것처럼 [코스프레](%EC%BD%94%EC%8A%A4%ED%94%84%EB%A0%88.md)하고 있는 코드라고
생각하면 된다. 당연히 실제 작성된 코드처럼 실행할 수 없다. <del>일반인이 스파이더맨 코스프레를 한다고 해서 손에서 거미줄이 나오게 할
수는 없는 것과 같은 이치.</del> 어차피 실제로 돌아가지 않기 때문에 굳이 문법을 엄격하게 지킬 필요도 없고, 오히려 사람이 알아보기
쉽게 서술하는 게 장땡이다. 특정한 프로그래밍 언어에 대한 지식이 없이도 해석할 수 있는 게 좋은 의사 코드다.

이러한 특성 때문에 주로 [알고리즘](%EC%95%8C%EA%B3%A0%EB%A6%AC%EC%A6%98.md)을 설명하거나 코딩을 하기
전 미리 프로그램 코드를 구상할 때 쓰인다.

다음은 의사코드의 예제이다.  

    
    
    program Name and short description ReadDataStructure() ReadData() ... if Data is not perfect then exit CalculateStatistics Write Result in HTML-formatend program Name
