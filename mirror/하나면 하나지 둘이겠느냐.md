## 1. 설명 ¶

<del>[미란이](%EB%AA%A8%EB%A6%AC%20%EB%9E%80.md)의 애창곡</del>`[1]`  
  
(이 동영상의 32초부터 나오는 노래. 영상 시작부터 나오는 노래가 왕경태의 주제곡, '영심아 난 너만 보면 그래'이다.)

  

[한국](%ED%95%9C%EA%B5%AD.md)산 애니메이션
[영심이](%EC%98%81%EC%8B%AC%EC%9D%B4.md)에 나오는 노래. 만화 내에서 캠핑이라든가 극기훈련, 또는 소풍같이
'집단생활'을 하는 부분에서 꼭 나오는 악마의 노래이다`[2]`.

  

문제는 이 노래를 '타잔이 10원짜리 팬티를 입고'`[3]` 식으로 계속 숫자를 높여가면서 끝없이 부를 수 있다는 것. <del>숫자가
높아지면 가사가 길어져서 박자가 안 맞을것 같기도 하지만 그냥 그러려니 하자</del> 물론 부르는 사람도 지치고 듣는 사람도 지친다.
원작서 경태는 친구의 생일잔치에서 이 노래를 한 **1억 5천** 정도까지 불렀다.

  

사람들이 '영심이'라는 만화에 대해서는 주제가는 기억이 안 나도 대부분 이 노래는 기억한다.
[후크송](%ED%9B%84%ED%81%AC%EC%86%A1.md)의 원조라는 말도 있다. 한때 리그베다 위키에선 1000(!)까지
셌으나 용량문제와 트래픽 문제로 결국 지워졌다.(…)<del>정말 보고 싶다면 과거문서 항목으로 가면 된다.</del>

  

그리고 [2011년](2011%EB%85%84.md) [12월](12%EC%9B%94.md) 3일에 방영된
[무한도전](%EB%AC%B4%ED%95%9C%EB%8F%84%EC%A0%84.md) 명수는 12살에서 이 노래가 다시 등장

#### 2.5.1. 수정버전 ¶

#include <stdio.h>  
#include <math.h>

  

#define FINISH 10 //끝내고 싶은 횟수 지정  
int main(void)  
{

  

int n=1;  
for(n=1; n<=FINISH -1; n++)  
{  

switch((int)ceil(n / 2.0) % 4)  
{  
case 1:  

printf("%d(이)면 %d(이)지 %d이겠느냐 \n", n,n,n+1);  
break;

case 0:  
case 2:  

printf("%d(이)면 %d(이)지 %d 아니야 \n", n,n,n+1);  
if((n % 4) ==0)  
{  

printf("랄라랄라 라라라라 랄라랄라라 \n");  
printf("랄라랄라 라라라라 랄라랄라라 \n");

}  
break;

case 3:  

printf("%d(이)면 %d(이)지 %d (이)될까 \n", n,n,n+1);  
break;

}

}  
return 0;

}
