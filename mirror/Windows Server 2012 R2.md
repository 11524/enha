## 1. 개요 ¶

[윈도우 8.1](%EC%9C%88%EB%8F%84%EC%9A%B0%208.1.md)의 서버판이다. 윈도우 서버 2012의 업그레이드
판이기도 하다.

[[edit](http://rigvedawiki.net/r1/wiki.php/Windows%20Server%202012%20R2?action
=edit&section=2)]

## 2. 주요 개선점 ¶

  * 시작버튼 추가됨
  * 작업폴더 추가됨
  * 원하는 상태 환경설정 쉬워짐
  * 스토리지 티어링 추가됨
  * 스토리지 피닝 추가됨
  * 라이트 백 캐시 추가됨

[[edit](http://rigvedawiki.net/r1/wiki.php/Windows%20Server%202012%20R2?action
=edit&section=3)]
