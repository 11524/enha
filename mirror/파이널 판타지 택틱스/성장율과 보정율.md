## 1. 성장율 ¶

성장율은 값이 작을수록 성장율이 높다.  
레벨이 낮으면 성장율이 높아진다.

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%8C%8C%EC%9D%B4%EB%84%90%20%ED%8
C%90%ED%83%80%EC%A7%80%20%ED%83%9D%ED%8B%B1%EC%8A%A4/%EC%84%B1%EC%9E%A5%EC%9C%
A8%EA%B3%BC%20%EB%B3%B4%EC%A0%95%EC%9C%A8?action=edit&section=2)]

### 1.1. HP, MP 성장 공식 ¶

  * 레벨업 시  
레벨업 후의 기본수치 = 기본수치 X (레벨 + 성장수치 + 1 / 레벨 + 성장수치)

  * 레벨다운 시  
레벨다운 후의 기본수치 = 기본수치 X (레벨 + 성장수치 - 2 / 레벨 + 성장수치 - 1)

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%8C%8C%EC%9D%B4%EB%84%90%20%ED%8
C%90%ED%83%80%EC%A7%80%20%ED%83%9D%ED%8B%B1%EC%8A%A4/%EC%84%B1%EC%9E%A5%EC%9C%
A8%EA%B3%BC%20%EB%B3%B4%EC%A0%95%EC%9C%A8?action=edit&section=3)]

### 1.2. 물리AT, 마법AT, Speed 성장 공식 ¶

  * 전투 시작시의 레벨 < 전투 종료시의 레벨  
신 기본수치 = 기본수치 X (종료시의 레벨 + 성장수치 / 시작시의 레벨 + 성장수치)

  * 전투 시작시의 레벨 > 전투 종료시의 레벨  
신 기본수치 = 기본수치 X (종료시의 레벨 + 성장수치 -1 / 시작시의 레벨 + 성장수치 - 1)

[[edit](http://rigvedawiki.net/r1/wiki.php/%ED%8C%8C%EC%9D%B4%EB%84%90%20%ED%8
C%90%ED%83%80%EC%A7%80%20%ED%83%9D%ED%8B%B1%EC%8A%A4/%EC%84%B1%EC%9E%A5%EC%9C%
A8%EA%B3%BC%20%EB%B3%B4%EC%A0%95%EC%9C%A8?action=edit&section=4)]

## 2. 보정율 ¶

  * 보정율은 높을수록 수치에 높게 반영된다.
  * 실제 반영되는 현재수치 = 기본수치 X 보정수치 / 50
  * 간이식  
현재수치 X (최종레벨 + 성장수치) /(초기레벨 + 성장치)
