## 2. 심리학개론 ¶

심리학개론 교재에서는 주로 다음을 다룬다.  

  * 개관  

    * 정의, 목적, 연구대상
    * 역사, 상호보완적 접근방법
    * 분야 소개, 심리학 연구방법론, 심리통계(자료분석 및 정리방법)
  * 생리심리학 및 영양  

    * 신경생리학 : 개요, 신경계, 말초신경계, 중추신경계, 신경계 연구방법론
    * 영양심리학 : 개요, 영양분의 기능
  * 감각과 지각  

    * 감각 : 시각, 청각, 그 외 감각
    * 지각 : 지각이론, 지각의 차이, 착각과 착시, 초감각지각
  * 인지 심리학 (의식, 사고, 언어)  

    * 의식 : 의식변화, 약물에 의한 의식변화, 의식의 구분, 양식, 시간적 차원
    * 사고 : 개념형성, 사회추론과 추단율, 문제해결의 원리와 전문성
    * 언어 : 언어 발달, 언어와 의사소통
  * 학습과 기억  

    * 학습 : 정의, 범위, 접근법
    * 연합주의적 학습이론 : 고전적 조건형성([파블로프](%ED%8C%8C%EB%B8%94%EB%A1%9C%ED%94%84.md)), 조작적 조건형성 (강화이론)
    * 인지적 학습이론 : Khler의 통찰학습, 인지도 학습, 개념학습, Bandura의 관찰학습
    * 기억의 연구와 모형 : 기억연구의 접근법, 기억의 모형, 망각, 거짓 기억, 기억에 작용하는 요인, 기억의 발달
  * 동기와 정서  

    * 동기
    * 정서 : 정서의 원인, 정서가 인지과정과 행동에 미치는 영향
  * 발달심리학  

    * 발달의 정의, 발달의 과학적 설명
    * 발달의 구분 : 지각의 발달, Piaget의 인지 발달, 애착과 [사회성](%EC%82%AC%ED%9A%8C%EC%84%B1.md) 발달, 발달에서 사회화의 영향, 도덕성 발달, 성인기 이후의 문제
  * 성격심리학  

    * 성격의 기초, 요인
    * 성격이론의 구분 : 특성이론, 정신분석 이론, 기타 이론들
    * 심리검사
  * 건강심리학, 이상심리학, 심리치료  

    * 건강 : 스트레스, 성격과 건강
    * 이상심리학 : 개념과 구체적 사례들
    * 심리치료 : 정신분석적 치료, 행동치료, 인지치료, 인간중심치료, 게슈탈트 치료, 그룹치료, 새로운 심리치료 방법들, 생의학 치료
  * 사회심리학  

    * 타인 및 사회적 사건의 이해 : 사회심리학의 정의, 사물지각과 대인지각, 인상형성, 사회인지, 귀인
    * 대인관계 : 친교행동, 공격행동, 사회영향, 집단에서의 행동

## 3. 교육과정 ¶

한 마디로, 심리학개론 교재의 대단원 하나하나가 심리학과의 교과목 하나하나를 설명한다.

  

심리통계, 심리학 연구방법론, 생리심리학, 감각과 지각 심리학, 인지 심리학, 학습과 기억 심리학, 동기와 정서 심리학, 발달심리학,
성격심리학, 사회심리학 같은 과목들이 심리학과 학부 과정의 기초를 이룬다는 것.

  

이런 기초 심리학 과목들을 통해 학부 고학년에서는 세부적인 과목들을 배운다.  

  * 임상심리학, 상담심리학, 이상심리학, 건강심리학, 심리검사 : 심리적인 문제를 찾고 해결하는 방법.
  * 산업·조직심리학, 소비자 심리학, 광고심리학 : 심리학의 산업에서의 사용.  

그 외에도 여러 다른 주제들도 배운다.  

  * 학교심리학, 청소년 심리학
  * 노인 심리학  

심리학 대학원에 가려면 입학시험을 치러야 한다. [기출문제](http://cafe.daum.net/counseling0511/R7cv/22)
같은 것을 보면 유리하다.
