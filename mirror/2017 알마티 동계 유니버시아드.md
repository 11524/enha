# 개요 ¶

  * [2017년](2017%EB%85%84.md)에 [카자흐스탄](%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84.md) [알마티](%EC%95%8C%EB%A7%88%ED%8B%B0.md)에서 개최되는 제 28회 동계 [유니버시아드](%EC%9C%A0%EB%8B%88%EB%B2%84%EC%8B%9C%EC%95%84%EB%93%9C.md) 대회.  

# 유치과정 ¶

[알마티](%EC%95%8C%EB%A7%88%ED%8B%B0.md) 단독후보였기 때문에 [2017 타이베이 하계 유니버시아드](/wik
i/2017%20%ED%83%80%EC%9D%B4%EB%B2%A0%EC%9D%B4%20%ED%95%98%EA%B3%84%20%EC%9C%A0
%EB%8B%88%EB%B2%84%EC%8B%9C%EC%95%84%EB%93%9C)가 결정되는 날, 같이 개최지로 선정되었다.

# 이모저모 ¶

  * [카자흐스탄](%EC%B9%B4%EC%9E%90%ED%9D%90%EC%8A%A4%ED%83%84.md)은 이전에 [2011 아스타나·알마티 동계 아시안 게임](2011%20%EC%95%84%EC%8A%A4%ED%83%80%EB%82%98%C2%B7%EC%95%8C%EB%A7%88%ED%8B%B0%20%EB%8F%99%EA%B3%84%20%EC%95%84%EC%8B%9C%EC%95%88%20%EA%B2%8C%EC%9E%84.md)을 개최했었기 때문에 이 경험을 바탕으로 [유니버시아드](%EC%9C%A0%EB%8B%88%EB%B2%84%EC%8B%9C%EC%95%84%EB%93%9C.md)를 개최하게 된다.
  * 자세한 내용은 [2017년](2017%EB%85%84.md) 대회가 개최되면 [추가바람](%EC%B6%94%EA%B0%80%EB%B0%94%EB%9E%8C.md).
