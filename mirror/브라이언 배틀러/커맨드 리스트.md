# 1. [KOF 94](KOF%2094.md) ¶

★ 기본잡기  

  * 아토믹 드롭 : 근접해서 ← or → + C
  * 락 버스터 : 근접해서 ← or → + D  

★ 특수기  

  * 버스터 토마호크 : 공중에서 ↓ + C  

★ 필살기  

  * 브라이언 토네이도 : →↓↘ + A or C
  * 하이퍼 태클 : →↘↓↙← + A or C
  * 스크류 보디 프레스 : 공중에서 ↓↘→ + B or D  

★ 초필살기  

  * 빅 뱅 태클 : ↓↙←→↘↓↙← + C

# 2. [KOF 98](KOF%2098.md) ¶

★ 기본잡기  

  * 아토믹 드롭 : 근접해서 ← or → + C
  * 락 버스터 : 근접해서 ← or → + D  

★ 특수기  

  * 버스터 토마호크 : 공중에서 ↓ + C (공중 히트시 다운)  

★ 필살기  

  * 브라이언 토네이도 : ←↙↓↘→ + A or C
  * 하이퍼 태클 : ←↙↓↘→ + B or D (가드포인트 존재)
  * 스크류 보디 프레스 : 공중에서 ↓↘→ + A or C
  * 브라이언 해머 : ↓↙← + A or C  
┣ 더블 해머 : ↓↙← + A or C  
┃　┗ DDT : ↓↙← + A or C  
┗ 타이거 드라이버 : ←↙↓↘→ + A or C (사용후 사무라이 봄 이나 숄더 백 브레이커로 이행가능)  

┣ 사무라이 봄 : ↓↓ + A or C  
┗ 숄더 백 브레이커 : ↓↓↓ + B or D

　　　┗ 버스터 & 봄 : ↓↑ + A or C

  * 로켓 태클 : →↓↘ + B or D (대공기)  

★ 초필살기  

  * 아메리칸 슈퍼노바 : ↓↘→↓↘→ + A or C (대공 잡기)
  * 빅 뱅 태클 : ↓↘→↓↘→ + B or D

# 3. [KOF 98 UM](KOF%2098%20UM.md) ¶

★ 기본잡기  

  * 아토믹 드롭 : 근접해서 ← or → + C
  * 락 버스터 : 근접해서 ← or → + D  

★ 특수기  

  * 버스터 토마호크 : 공중에서 ↓ + C  

★ 필살기  

  * 브라이언 토네이도 : ←↙↓↘→ + A or C
  * 하이퍼 태클 : ←↙↓↘→ + B or D
  * 스크류 보디 프레스 : 공중에서 ↓↘→ + A or C
  * 브라이언 해머 : ↓↙← + A or C  
┣ 더블 해머 : ↓↙← + A or C  
┃　┗ DDT : ↓↙← + A or C  
┗ 타이거 드라이버 : ←↙↓↘→ + A or C  
　　┣ 사무라이 봄 : ↓↓ + A or C  
　　┗ 숄더 백 브레이커 : ↓↓↓ + B or D  
　　　　┗ 버스터 & 봄 : ↓↑ + A or C

  * 로켓 태클 : →↓↘ + B or D  

★ 초필살기

  * 빅 뱅 태클 : ↓↘→↓↘→ + B or D
  * 갤럭시 사이클론 : ↓↙←↙↓↘→ + A or C
