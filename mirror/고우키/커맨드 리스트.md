## 4. [스트리트 파이터 X 철권](%EC%8A%A4%ED%8A%B8%EB%A6%AC%ED%8A%B8%20%ED%8C%8C%EC%9D%B4%ED%84%B0%20X%20%EC%B2%A0%EA%B6%8C.md) ¶

★ 기본잡기  

  * 호충파 : PK
  * 주열인 : ← + PK
★ 특수기  

  * 두개파살 : → + 중P
  * 천마공인각 : 전방 점프중 정점에서 ↓ + 중K
★ 필살기  

  * 호[파동권](%ED%8C%8C%EB%8F%99%EA%B6%8C.md) (EX) : ↓↘→ + P
  * 참공파동권 (EX) : 공중에서 ↓↘→ + P
  * 작열파동권 (EX) : →↘↓↙← + P
  * 호[승룡권](%EC%8A%B9%EB%A3%A1%EA%B6%8C.md) (EX) : →↓↘ + P
  * [용권참공각](%EC%9A%A9%EA%B6%8C%EC%84%A0%ED%92%8D%EA%B0%81.md) (EX) : ↓↙← + K (공중 가능)
  * [아수라섬공](%EC%95%84%EC%88%98%EB%9D%BC%EC%84%AC%EA%B3%B5.md) : →↓↘ 또는 ←↓↙ + PPP or KKK
  * **백귀습**`[3]` (EX) : →↓↘ + K  

    * 백귀호참 : 백귀습 시전 후 미입력
    * 백귀호충 : 백귀습 시전 중 P
    * 백귀호인 : 백귀습 시전 중 K
    * 백귀호쇄 : 백귀습 시전 중 상대방 근접 상태에서 약P + 약K
    * EX 참공 파동권 : EX 백귀습 시전 중 ↓↘→ + PP
★ 슈퍼 콤보  

  * 순옥살 : 약P, 약P, →, 약K, 강P
  * 재계：↓↓↓ + KKK

### 8.1. 고우키 ¶

★ 잡기  

  * 업어치기： (접근해서) AB 
  * 배대 되치기： (접근해서) CD 
★ 특수기  

  * 천마공인각： (전방 점프중 정점에서) ↓ + B 
  * 두개파살： → + C 
  * 무릎 차기： → + D 
  * 복사뼈 킥： ↙ + D 
★ 필살기  

  * 호파동권： ↓↘→ + A or C 
  * 참공파동권： (공중에서) ↓↘→ + A or C 
  * 작열파동권： →↘↓↙← + A or C 
  * 호승룡권： →↓↘ + A or C 
  * 용권참공각： ↓↙← + B or D 
  * 공중 용권참공각： (공중에서) ↓↙← + B or D 
  * 아수라섬공： →↓↘ or ←↓↙ + AC or BD 
  * 백귀습： →↓↘ + B or D   

    * 백귀호참： (백귀습 중) 추가 입력 없음 
    * 백귀호쇄： (백귀습 중 접근해서) AB 
    * 백귀호충： (백귀습 중) A or C 
    * 백귀호인： (백귀습 중) B or D 
★ 초필살기  

  * 멸살호파동： →↘↓↙←→↘↓↙← + A or C 
  * 멸살호승룡： ↓↘→↓↘→ + A or C 
  * 천마호참공： (공중에서) ↓↘→↓↘→ + A or C 
★ 익시드  

  * 순옥살： A, A, →, B, C
