## 1. 상세 ¶

2013년 9월 20일 "전격문고 가을제전 2013"에서 첫 공개되었다. 2013년 10월 11~13일에 첫 로케테스트가 일본 현지에서
시행중이며 Ustream으로 중계중이다.

  

실제 제작사는 [멜티 블러드](%EB%A9%9C%ED%8B%B0%20%EB%B8%94%EB%9F%AC%EB%93%9C.md) 시리즈 및
[UNDER NIGHT IN-BIRTH](UNDER%20NIGHT%20IN-BIRTH.md)를 제작한
[프랑스빵](%ED%94%84%EB%9E%91%EC%8A%A4%EB%B9%B5.md)과
[에콜](%EC%97%90%EC%BD%9C.md)로 밝혀졌다.  
초기에는 샤나, 미사카 미코토, 코우사카 키리노와 아스나 4명만 플레이어블로 공개되었으나, 추후 액셀 월드의 흑설공주와 듀라라라의 헤이와지마
시즈오가 추가로 공개되었다.  
[ALL.Net P-ras MULTI](ALL.Net.md)에 대응하며, 역시 RINGEDGE2 기판을 사용한다.  
2014년 3월 18일 정식가동 개시. 5월 10일, 추가 캐릭터가 발표되었다. 2014년 7월 [PS3](PS3.md)과 [PSVITA](PS%20VITA.md)로 이식이 발표되었다. PS4도 이식해주지... 세가 코리아에서 정발을 고려해보기도 했으나 [특정 문제](%EB%A7%88%EB%B2%95%EA%B3%BC%EA%B3%A0%EA%B5%90%EC%9D%98%20%EC%97%B4%EB%93%B1%EC%83%9D.md)로 [불가능하다고 밝혔다](http://bbs2.ruliweb.daum.net/gaia/do/ruliweb/de
fault/ps/93/read?articleId=1545166&objCate1=&bbsId=G003). 그 덕분에 다른 작품 팬들의 불만이
커지고 있다.`[1]` [중국어판은 정상적으로 로컬라이징이 되었다.](http://bbs2.ruliweb.daum.net/gaia/do/ru

## 2. 시스템 ¶

  * 블라스트 - 일정시간 경과 후 회복
  * 서포트 - 일정시간 경과 후 회복
  * 비장의 카드 - 2개 기본. 질 경우 하나 회복.
  * 클라이맥스 게이지 - 최대 5 축적 가능  

  * 기본키 - 약 중 강 A B C 서포트 S
  * 대쉬 - 빠르게 →→
  * 낙법 - 날려지는 중 공격버튼
  * 잡기 - ← 또는 → + C
  * 잡기 회피 - 잡히는 순간 ← 또는 → + C
  * 빠른 점프 - 빠르게 ↙↗  

  * 콤비네이션 - A > B > C 로 연결
  * 퀵 콤비네이션 - A 연타
  * 임팩트 스킬 - A + B / ↓+ A + B 
  * 임팩트 브레이크 - ← + A + B 
  * 익스텐드 액션 - 통상공격/필살기 후 버튼 지속(모으기)  

  * 클라이맥스 아츠 - 특정 커맨드 + B + C
  * 리플렉션 가드 - 가드 중 ABC 가운데 2개 동시
  * EX필살기 - 필살기 커맨드 + A + B
  * 캔슬 서포트 - 공격중 S  

  * 블래스트 - A + B + C  

    * 파워업 블래스트 - 통상 사용. 패러미터가 일정 시간 상승하며 체력이 서서히 회복 & 클라이맥스 게이지가 서서히 증가한다. 비장의 수단 지속시간이 남아있을 경우에 발동시엔 비장의 수단 지속시간도 어느정도 증가 시켜준다. 발동시 나오는 충격파를 맞췄을 경우 클라이맥스 게이지 2칸을 채워준다.
    * 콤보 블래스트 - 공격 중 사용. 충격파를 맞췄을 경우 상대가 바닥에 닿을 때 까지 다운 복귀 불가 상태로 띄워지므로 강제 콤보 연결이 가능하다.
    * 이스케이프 블래스트 - 방어, 피격 중 사용. 모든 피격 딜레이를 취소하고 충격파에 맞을 경우 그대로 상대를 날려보낸다. 두 블래스트와 다르게 충격파를 맞추더라도 클라이맥스 게이지를 회복시켜 주지 않는다.  

  * 비장의 수단 - A + C를 입력시 전격 마크 1개를 소모하여 캐릭터마다 다른 효과를 발휘한다. 가장 큰 부류로는 버프형과 공격형으로 나뉜다. 공격형인 아키라의 경우 강력한 데미지를 입히는 공격인 철산고를 날리고, 미코토라면 원작의 자판기 킥을 날린다. 버프형인 키리토의 경우 엑스트라 스킬 이도류로 자신의 공격 판정을 크게 올리거나, 렌타로의 터미널 호라이즌으로 기동력 상승과 모으기 단축 효과를 얻는다. 전격 마크는 게임 시작시 2개가 주어지고 패배할 때마다 1개 회복하지만 최대 한도는 2개이므로 첫 라운드에서 쓰지 않고 패배할 경우 추가로 차오르지 않는다. 공통적인 효과로 지속시간 중에는 캔슬 서포트를 게이지 소모 없이 사용할 수 있다는 이점이 있다. 또한 지속시간이 다 지나지 않아도 발동 자체는 가능하며 이 경우 지속시간이 초기화된다. 피격 당했을시엔 지속시간이 일정량 깍인다.  

  * 포텐셜 - 캐릭터마다 발동 조건과 효과가 다르다.
