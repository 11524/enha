카드 일람. 물론 리그베다 위키에 항목으로 등록된 카드는 매직 더 개더링 전체 카드 중 빙산의 일각이다. 2012년 **20,000장**을
넘었고 매년 1000장 가량 늘어난다. 다만 이중에서 타입2에서 쓸 수 있는 카드는 적을땐 1000여장, 많을 땐 2000여장 **밖에**
안된다.`[1]`

카드 검색 사이트 - [개더러](http://gatherer.wizards.com)`[2]`,
[매직카드인포](http://magiccards.info/),
[카드킹덤](http://www.cardkingdom.com/search/mtg)`[3]`,
[스타시티](http://sales.starcitygames.com/spoiler/spoiler.php)`[4]`, , [미식
스포일러](http://mythicspoiler.com) `[5]`

  * 카드 이름은 영문판 [알파벳](%EC%95%8C%ED%8C%8C%EB%B2%B3.md)순으로 정렬합니다. 
  * 한국어판이 존재했을 때 나온 카드(기본판 4~5판, 확장판 Visions / Weatherlight / Tempest / Stronghold / Exodus / Urza's Saga, 기본판 Magic 2012 / 확장판 Innistrad 이후에 나온 카드)는 한국어판 이름도 옆에 표기합니다. 단, DotP에서만 나온 번역은 이탤릭체로 표기합니다. [카드명 합의안](http://wikibbs.net/agreement/959161) 참고
  * 카드 이름에 Æther가 들어가는 것은 검색의 편의상 Aether라는 표기를 사용합니다.
  * 스플릿 카드는 '왼쪽 카드 이름 // 오른쪽 카드 이름'으로 항목을 작성합니다.
  * 플립 카드나 양면 카드는 변신 전 이름으로 항목을 작성하되, 목록에 변신 후 이름도 표시합니다.
