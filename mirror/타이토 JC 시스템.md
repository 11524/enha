**타이토 JC 시스템**이란, 타이토에서 개발한 3-D 채감형 [기판](%EA%B8%B0%ED%8C%90.md)이다. 

## 사양 ¶

  

  * 메인 CPU : 68040, TMS32025  

  * 보조 CPU : MC68HC11   

  * 사운드 제어 CPU : 68EC000   

  * 음원 : ES5505   

플리곤의 본격적인 사용을 할 수 있으며, 플리곤의 색감이나 사용 수는 떨어지지만 텍스쳐화 된 플리곤이 가능하다.
