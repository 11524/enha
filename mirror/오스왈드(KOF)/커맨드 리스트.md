# 1. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * ２ : 근접해서 ← or → + C or D  

★ 특수기  

  * Ｊ : → + A
  * １０ : ↘ + A  

★ 필살기  

  * Ｑ : →↓↘ + A or B or C or D
  * Ｋ : ←↓↙ + B or D
  * ♠ : ↓↙← + A or C (연속 2회 입력)
  * ◆ : ↓↘→ + A or C (연속 2회 입력)
  * ♥ : ↓↘→ + B or D  
┗ Ａ : 슈트 캔슬 버전 ♥ 사용 중 ↓↘→ + E or ↓↙← + E

  * ♣ : ↓↙← + B or D  

★ 초필살기  

  * ♠♥◆♣ : ↓↘→↘↓↙← + A or C
  * １０·Ｊ·Ｑ·Ｋ·Ａ : →↘↓↙←→↘↓↙← + A or C  

★ 리더 초필살기  

  * Ｊ·Ｏ·Ｋ·Ｅ·Ｒ : ← 모으고 →←→ + E
