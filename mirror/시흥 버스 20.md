# 1. 노선 정보 ¶

  * 기점 : 시흥시청후문.종합일자리센터
  * 종점 : 중앙역앞
  * 첫차 : 시흥시청 05:20 / 중앙역 06:10
  * 막차 : 시흥시청 22:00 / 중앙역 23:00
  * 배차간격 : 25~40분  

[시흥교통](%EC%8B%9C%ED%9D%A5%EA%B5%90%ED%86%B5.md)에서 운행중인
[시내버스](%EC%8B%9C%EB%82%B4%EB%B2%84%EC%8A%A4.md) 노선으로 시흥시청을 출발해 능곡동, 경일고,
동명아파트, 강서고, 올림픽기념관, 안산시청을 경유해 중앙역까지를 잇는 노선이다.

  

**배차간격이 상당히 긴 편이므로 시간표를 미리 확인하는 것이 좋다.** 시간표는 [시흥교통 홈페이지](http://www.shbus.net/)에서 확인 가능하다.
