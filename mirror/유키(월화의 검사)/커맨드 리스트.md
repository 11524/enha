## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 무상신명 : 근접해서 ← or → + B  

★ 특수기  

  * 일창 ・ 빙주 찌르기 : 상대 다운시 ↘ + B (모으기 가능)  

★ 필살기  

  * 빙인 : ↓↘→ + A or B
  * 상화 : →↓↘ + A or B 
  * 순설참(●) : ↓↙← + A or B
  * 빙경 : ←↙↓↘→ + C
  * 수참 : 근접해서 →↘↓↙← + C  

★ 초오의  

  * 설풍권 : ↓↙←→ + AB  
┣ 동자 : 설풍권 중 ↓↘→ + B  
┗ 동참 : 설풍권 중 ↓↘→ + C  

★ 잠재오의  

  * 진 ・ 설풍권 : ↓↙←→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 무상신명 : 근접해서 ← or → + CD  

★ 특수기  

  * 일창 ・ 빙주 찌르기 : 상대 다운시 ↘ + B (모으기 가능)  

★ 필살기  

  * 빙인 : ↓↘→ + A or B
  * 상화 : →↓↘ + A or B 
  * 순설참(●) : ↓↙← + A or B
  * 빙경 : ←↙↓↘→ + C
  * 수참 : 근접해서 →↘↓↙← + C  

★ 초오의  

  * 수빙 : ↓↘→↓↘→ + AB (히트 시 A버튼 연타로 위력 증가)  

★ 잠재오의  

  * 진 ・ 설풍권 : ↓↙←→ + B  

★ 난무오의  

  * 심 ・ 설풍권 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, ↓↘→ + A  
┣ 띄우기 루트 : 히트 후 A, B, C,↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C,↓ + C, A, BC, A,↓↘→ + B
