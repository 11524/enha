## 1. 내용 ¶

한국의 AD Chips라는 회사에서 개발한 [아케이드](%EC%95%84%EC%BC%80%EC%9D%B4%EB%93%9C.md)
[게임](%EA%B2%8C%EC%9E%84.md) [기판](%EA%B8%B0%ED%8C%90.md)이다.  
라인업은 아케이드 붐이 일어난 2000년대 초 개발된 국산게임들이 주류를 이루지만, 해외에서 제작된 게임도 여럿 있다.  
이 기판의 바리에이션 중 하나로 브렛자소프트에서 개발한 크리스탈 시스템이 존재하며, 사양 자체는 비슷하다.  
현재 이 기판으로 제작, 출시된 많은 게임이 에뮬레이팅되었지만, 상태가 불안정한 경우도 많다.(그래픽이 깨진다거나, 게임이 도중에
멈춘다거나...)

## 2. 사양`[1]` ¶

Feature

Processor(CPU+2D Graphic+Sound)

32Bit EISC CPU Core(Cache 4Kbyte), 208 LQFP Pin, Wavetable Synthesizer
(32Channel, 16bit)  
High Performance 2D Graphic Based on 3D  
Controller, Other Peripherals

Operating Frequency

CPU : 40MHz/Graphic [Core:80MHz](Core%3A80MHz.md)

Memory

Texture Memory : 8Mbyte  
Frame Memory : 8Mbyte  
Data Memory : 16Mbyte

Display

RGB Monitor  
VGA Monitor  
Horizontal Freq. : 15Khz, 24Khz, 30Khz

Copy protection

built in copy protection

LAN interface

one Ethernet interface

Interface

JAMMA, Extension Slot, External Serial Port, 3/4P Expansion Slot

Sound Format

44.1Khz, 16bit Stereo, I2S Format

Back-up System

128Kbyte SRAM, RTC

Graphic Effect (2D Graphic based on 3D)

Zoom In/Out, Rotation, Alpha Blending, Shading, Line-scrolling

AMP

22Watt, Stereo  
Power Supply  
DC 5V(+/-10%) and 12V(+/-10%)

Dimension

215mm*185mm(Main board only)
