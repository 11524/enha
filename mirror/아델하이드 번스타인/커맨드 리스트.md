# 2. [KOF XI](KOF%20XI.md) ¶

  

★ 기본잡기  

  * 드뤼켄 : 근접해서 ← or → + C or D  

★ 특수기  

  * A.쿤스트 아인 : ↘ + A
  * A.쿤스트 쯔바이 : → + B
  * A.쿤스트 바이어 : ← + B  

★ 필살기  

  * G.반트 : ↓↘→ + A or C
  * G.키켄 : ↓↘→ + B or D
  * G.드뤼켄 : →↘↓↙← + A or C
  * G.프린체신 : ↓↘→ + 도발  

★ 초필살기  

  * G.크론프린츠 : →←↙↓↘→ + A or C (모으기 가능)
  * G.크라이스 : ↓↘→↓↘→ + B or D  

★ 리더 초필살기  

  * G.슐라겐 : ↓↙←↙↓↘→ or ↓↘→↘↓↙← + E
