## 2. 설명 ¶

방위청에서는 본래 [육해공](%EC%9C%A1%ED%95%B4%EA%B3%B5.md) 각 막료감부마다 따로 정보부서를 뒀고 방위청도
따로 정보부서를 둬서 업무 중복이 많았고 [부서이기주의](%EB%B6%80%EC%84%9C%20%EC%9D%B4%EA%B8%B0%EC%A3%BC%EC%9D%98.md)로 인해 정보의
유통도 잘 되지를 않았다. 그탓에
[통합막료회의의장](%ED%95%A9%EB%8F%99%EC%B0%B8%EB%AA%A8%EC%9D%98%EC%9E%A5.md)이었던
[이시이 마사오](%EC%9D%B4%EC%8B%9C%EC%9D%B4%20%EB%A7%88%EC%82%AC%EC%98%A4.md)를
으뜸으로 한 프로젝트를 만들었다. 이들은 미국의 [국가정보장](ODNI.md)`[1]`을 본따 1997년 1월 20일에 창설되었다.
전신은 통합막료회의사무국제2막료실(統合幕僚会議事務局第2幕僚室)`[2]`과 육상막료감부조사부제2과별실(陸上幕僚監部調査部第2課別室)이었다.

[도쿄도](%EB%8F%84%EC%BF%84%EB%8F%84.md)
[신주쿠](%EC%8B%A0%EC%A3%BC%EC%BF%A0.md)구 이치가야혼무라초(市谷本村町) 5번 1호에 위치한다.

  

창설 당시 규모는 1700명이고 [2010년대](2010%EB%85%84%EB%8C%80.md) 들어서 규모는 2400명 수준이다.
2400명 가운데 [자위관](%EC%9E%90%EC%9C%84%EA%B4%80.md)이 1900명, 사무관(문관)이 500명 수준이다.

  

본부장은 [중장](%EC%A4%91%EC%9E%A5.md)이 맡고 부본부장은
[문민](%EB%AC%B8%EB%AF%BC.md)`[3]`이 맡는다.

## 3. [직제](%EC%A7%81%EC%A0%9C.md) ¶

  

  * 정보본부장 : 육해공 중장  

    * 부본부장 : 사무관
    * 정보관 : 자위관 3인`[4]`, 사무관 1인
    * 정보평가관 : 사무관
    * 정보보전관 : 사무관
  * 총무부장 : 1종 1좌(준장급)
  * 계획부장 : 1종 1좌(준장급)
  * 분석부장 : 사무관
  * 통합정보부장 : 1종 1좌(준장급)
  * 화상/지리부장 : 1종 1좌(준장급)
  * 전파부장 : 사무관
  * 통신소 : 여섯 곳의 통신소를 두고 있다. 히가시치토세(東千歳)`[5]`, 쇼쿠도(小舟渡)`[6]`, 오이(大井)`[7]`, 미호(美保)`[8]`, 다치아라이(太刀洗)`[9]`, 기카이지마(喜界島)`[10]` 여섯 곳. 각 통신소장은 대령, 쇼쿠도만 중령이 맡는다.
