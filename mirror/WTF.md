# 2. World of warcraft Text File ¶

[확장자](%ED%99%95%EC%9E%A5%EC%9E%90.md). [월드 오브 워크래프트](%EC%9B%94%EB%93%9C%20%EC%98%A4%EB%B8%8C%20%EC%9B%8C%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8.md)에서
사용하는 [게임](%EA%B2%8C%EC%9E%84.md) 내 커스텀 [파일](%ED%8C%8C%EC%9D%BC.md) 형식이자,
이 같은 파일이 모여있는 [WOW](WOW.md) 내의 폴더 [이름](%EC%9D%B4%EB%A6%84.md)이기도 하다. 주로
[애드온](%EC%95%A0%EB%93%9C%EC%98%A8.md)이라든가,
[매크로](%EB%A7%A4%ED%81%AC%EB%A1%9C.md)라든가, 대화창 위치 및
[설정](%EC%84%A4%EC%A0%95.md) 같은 설정이 저장된다.

  

[블리자드](%EB%B8%94%EB%A6%AC%EC%9E%90%EB%93%9C.md)인 만큼 1번 항목을 연상시키고자 폴더명을 지었을
확률이 매우 농후하다.
