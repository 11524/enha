# 3. [리얼바웃](%EB%A6%AC%EC%96%BC%EB%B0%94%EC%9B%83.md) 아랑전설 스페셜 ¶

★ 기본잡기  

  * 니 스매셔 : 근접해서 ← or → + C  

★ 특수 컴비네이션 아츠  

  * 근접 C - C - ↓↘→ + C  

★ 특수기  

  * 데스 해머 : → + BC
  * 다이빙 엘보 : 상대 다운 중 ↓ + C
  * 카이저 바디 프레스 : 노멀 점프 중 ↓ + C  

★ 페인트기  

  * 카이저 웨이브 페인트 : ↓ + BC  

★ 필살기  

  * 블릿츠 볼 상단 : ↓↙← + A
  * 블릿츠 볼 하단 : ↓↙← + C
  * 레그 토마호크 : ↓↘→ + B
  * 데인저러스 스루 : 근접해서 ←↙↓↘→ + A
  * 리프트 업 블로 : 근접해서 →↘↓↙← + B
  * 피닉스 스루 : ←↙↓↘→ + C  

★ 초필살기  

  * 카이저 웨이브 : ← 모으고 ↗ + BC (모으기 가능)  

★ 잠재능력  

  * 기간틱 사이클론 : 근접해서 →↘↓↙←→↘↓ + C
  * 언리미티드 디자이어 : →↘↓↙←→＋A - A - B - C - B - C - A - B - C - 피니시 입력  
┠ 노멀 피니시 : C  
┗ 기간틱 사이클론 피니시 : ←↓↙＋AC

  * 블릿츠 볼 하단 : ↓↙← + C
  * 레그 토마호크 : ↓↘→ + B
  * 카이저 클로 : →↓↘ + C

★ 초필살기

★ 잠재능력

  * 언리미티드 디자이어 : →↘↓↙←→＋A - A - B - C - B - C - A - B - C - 피니시 입력  
┠ 노멀 피니시 : C
