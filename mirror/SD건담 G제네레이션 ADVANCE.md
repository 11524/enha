## 1. 개요 ¶

2003년 [게임보이 어드밴스](%EA%B2%8C%EC%9E%84%EB%B3%B4%EC%9D%B4%20%EC%96%B4%EB%93%9C%EB%B0%B4%EC%8A%A4.md)용으로 발매 된 SD건담 G제네레이션.

  

사실 게임의 내용은 [SD건담 G제네레이션 개더비트](SD%EA%B1%B4%EB%8B%B4%20G%EC%A0%9C%EB%84%A4%EB%A0%88%EC%9D%B4%EC%85%98%20%EA%B0%9C%EB%8D%94%EB%B9%84%ED%8A%B8.md)의 리메이크에
가깝다. 이전까지 좌우로 늘어선 소대가 쿼터뷰 방식으로 바뀌었고, 그래픽도 입체적이고 부드러운 **PS1급 3D 그래픽**과 강렬하고
스무스한 연출로 바뀌어 상당히 좋아졌다. 사실 부드러운 3D는 개더비트2 이후의 강점이기도 했다.

  

특히 필살기 연출들은 GBA라는게 믿기 힘들 정도의 그래픽이 많다.. 인기기체 누락, 쓸데없는 기체 삽입 등과 병맛나는 시나리오, 루즈한
진행으로 혹평을 받지만 연출은 **초기 GBA 슈로대를 압도한다.** 심지어 [슈퍼로봇대전J](%EC%8A%88%ED%8D%BC%EB%A1%9C%EB%B4%87%EB%8C%80%EC%A0%84%20J.md)에서는, 이 게임의
샤이닝 건담 연출을 그대로 가져왔을 정도로 이 게임을 다소 의식한 모습을 보였다.

  

슈로대의 정신기 개념인 ID 커멘드의 존재로 알고보면 **상당히 쉽다.** 특히 ID 커멘드는 **공격력과 명중률 +30% 이상**의 효과가
많은데다, 슈로대의 **필중 + 열혈**을 거의 매턴 쓸 수 있다고 보면 된다. 전함까지 붙여놓고 ID 커멘드를 쓰면 회피보정도 걸린다.
<del>치트 게임인가!</del> 하지만 인기 기체가 너무 나오질 않고 쓸데없는 플레이타임 먹기가 많아 초반에 도태되기 십상인게 매우 큰
단점.

### 2.1. 시나리오 ¶

휴대용 최초로 시드가 정식 참전했는데, 퍼건이 동시간대라 스토리가 좀 이상하다. 게다가 참전한 건 좋은데 제대로 다뤄주질 않아서 크루제가
**프리덤 건담(…)**을 타고 나오고,`[1]` 시드계 MS들은 개발 루트조차 없는 등 전체적으로 상당히 푸대접을 받고 있다.

  

게다가 시나리오 자체도 안드로메다인 환상적인 전개가 특징. 붉은 혜성의 초상이 참전작으로 등록 되어 있고, 이 게임의 최종 보스는 [기렌자비](%EA%B8%B0%EB%A0%8C%20%EC%9E%90%EB%B9%84.md)가 아니라 **[기니어스 사하린](%EA%B8%B0%EB%8B%88%EC%96%B4%EC%8A%A4%20%EC%82%AC%ED%95%98%EB%A6%B0.md)(?!)이다**. 게다가
중간에 아군끼리 두 패로 나뉘어서 쌈박질하는 스테이지도 있다. [카미유비단](%EC%B9%B4%EB%AF%B8%EC%9C%A0%20%EB%B9%84%EB%8B%A8.md)이 [샤아아즈나블](%EC%83%A4%EC%95%84%20%EC%95%84%EC%A6%88%EB%82%98%EB%B8%94.md)이 아닌
[아무로 레이](%EC%95%84%EB%AC%B4%EB%A1%9C%20%EB%A0%88%EC%9D%B4.md)한테 그 유명한
[명대사](%EB%AA%85%EB%8C%80%EC%82%AC.md)를 작렬시키는 장면을 보고 싶다면 꼭 한번 해보도록.

  

반면 본 작품으로 출세한 캐릭터는 [쿠클스도안](%EC%BF%A0%ED%81%B4%EC%8A%A4%20%EB%8F%84%EC%95%88.md). 특정 조건에서 하이퍼화하게 되며
정권찌르기가 없다는 걸 빼면 원작에서의 초활약을 잘 재현했다. 심지어
[명경지수](%EB%AA%85%EA%B2%BD%EC%A7%80%EC%88%98.md)까지 깨우치며 **전용 테마까지 준비되어 있다.**

  

덤으로 **14세**의 [하만 칸](%ED%95%98%EB%A7%8C%20%EC%B9%B8.md)과 [캐스발 렘다이쿤](%EC%83%A4%EC%95%84%20%EC%95%84%EC%A6%88%EB%82%98%EB%B8%94.md) 버전을 아군으로
쓸 수 있는 거의 유일한 게임이다. 그래서 [게이머즈](%EA%B2%8C%EC%9D%B4%EB%A8%B8%EC%A6%88.md)에서는
이런 전개를 두고 **제목은 G제네, 내용은
[슈퍼로봇대전](%EC%8A%88%ED%8D%BC%EB%A1%9C%EB%B4%87%EB%8C%80%EC%A0%84.md), 히로인은
시마 누님**이라는 평가를 내렸다. 그 평가가 아주 딱 들어맞는다는 점이 베스트.

## 3. 시스템 ¶

**◎ 파일럿 능력치**  
＃ SP : 정신기 사용 포인트  
＃ 전용 (持) : 전용기 표시  
＃ 근접 (近接) : 격투무기 명중률 영향  
＃ 사격 (射擊) : 사격무기 명중률 영향  
＃ 반응 (反応) : 선제공격 확률, 크리티컬 확률, 추가공격, 회피율 영향  
＃ 조종계 (操縦系) : 조종기체의 계열  
＃ 경험치 (EXP) : 현재 경험치  
＃ 다음까지 (NEXT) : 레벨업까지 남은 경험치

  

**◎ 기체 능력치**  
＃ HP : 기체의 체력. 0이 되면 폭발.  
＃ 범용 (汎用) : 지형 만능  
＃ 운동 (運動) : 회피율, 명중률 영향  
＃ 한계 (限界) : 운동성 제한 수치  
＃ 장갑 (裝甲) : 수치 1당 무기 공격력의 수치를 25 감소시킨다. (EX> 210 x 5 -> 195 x 5)  
＃ 이동 (移動) : 기체의 이동수

  

**◎ 무기 관련**  
＃ 내용 (実) : 무기 이름  
＃ 공격력 (功) : 무기 공격력. 데미지 X 횟수가 나와있다.  
＃ 명중률 (命) : 무기의 기본 명중률  
＃ 탄 수 (彈) : 무기의 기본 탄 수  
＃ 사식 (射単) : 사격 형식  
＃ 근식 (近単) : 근접 형식

  

**◎ ID 효과 (效果), 기체 특수기능**  
＃ 공격, 명중, 회피 보정  
＃ 전용 기능 (持) : 기체 특수기능  
＃ 남은 횟수 (残り回數) : 특수기능 사용가능 횟수

  

**◎ 커멘드**  
＃ 이동 (移動) : 이동능력만큼 이동  
＃ 공격 (功擊) : 인접한 유닛을 공격  
＃ ID 커멘드 (ID) : ID 커멘드 사용  
＃ 포획 (捕獲) : 근처에 전함이 있고 유닛이 3인 이상 둘러싸이면 커멘드가 발동된다.  
＃ 간접 (間接) : 원거리 포격 커멘드

  

**◎ ID 커맨드**  
＃ 주요 인물들의 명대사 목록으로 이루어져 있고 사용시 관련 대사를 외친다.  
＃ 사용하면 SP 포인트가 소모되며 매 턴 소량 회복된다. (슈퍼로봇대전의 정신기)  
＃ 전투시 사용하는 전투 ID와 필드에서 사용하는 필드 ID로 나뉜다.  
＃ 필드 ID 는 유닛의 행동 종료 후에도 커서로 선택하여 사용할 수 있다.

  

**◎ 전투 커맨드 : 포격(包擊)**  
＃ 일정거리 이상 떨어진 적에게 공격하는 커맨드이며 상대방의 반격을 받지 않는다. (주로 전함이 보유)  
＃ 포격당하는 쪽의 선택지는 ' 산개 ' 와 ' 대기 ' 로 나뉜다.  
＃ 산개는 완전회피인 대신 팀이 흩어지게 되고 대기는 팀이 뭉친 채로 회피율에 의지하여 피하게 된다.  
＃ 포격 커멘드로 공격을 하면 ' 대기 ' 커맨드시의 적의 회피율이 뜬다.  
＃ 전함이 공격당하거나 주변의 유닛들로 인해 이동경로가 막히는 경우 ' 대기 ' 밖에 사용할 수 없다.

  

**◎ 전투 커멘드 : 공격(功擊)**  
＃ 아군과 적이 번갈아 공격하는 커멘드이며 이벤트 대사가 발생하면 전투 캔슬이 불가능하다.  
＃ 공격 순서는 사격 -> 격투로 이루어지며, 같은 속성의 공격은 동시에 진행된다.  
＃ 반응(反応) 능력치가 높으면 일정 확률로 선제 공격이 발동하여 먼저 공격할 수 있다.  
＃ 방어측은 무기 선택 전에 공격측의 사용 무기와 최종 명중률을 확인할 수 있다. (아군의 경우엔 적군턴에 확인 가능)  
＃ 전투시 유닛 수가 같으면 서로 해당 위치의 상대끼리 전투한다.  
＃ 기체 3기가 기체 2기를 공격할 시엔, A B A 식으로 순환 타겟 설정이 된다.  
＃ 기체 1기를 상대로 여럿이 공격할 땐 1기에 집중포화하는 형식으로 이루어진다.

  

**◎ EXP 획득 구조**  
＃ EXP는 전투를 실시하여 데미지를 입히거나 이벤트를 발생시키면 얻게 된다.  
＃ 전투로 얻는 EXP는 적에게 입힌 HP의 비율과 비례하여 획득한다.  
＃ EV EXP (이벤트 경험치) 는 이벤트를 발생시키면 얻는 경험치다.
