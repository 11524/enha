## 1. 개요 ¶

아우디 80/90 모델의 후속으로 데뷔한 [아우디](%EC%95%84%EC%9A%B0%EB%94%94.md)의 준중형차. [메르세데스-벤츠 C클래스](%EB%A9%94%EB%A5%B4%EC%84%B8%EB%8D%B0%EC%8A%A4-%EB%B2%A4%EC%B8%A0%20C%ED%81%B4%EB%9E%98%EC%8A%A4.md), [BMW3시리즈](BMW%203%EC%8B%9C%EB%A6%AC%EC%A6%88.md),
[렉서스](%EB%A0%89%EC%84%9C%EC%8A%A4.md) IS 등과 경쟁한다.  
세단과 스테이션 왜건 모델이 출시중이며, 쿠페, 소프트톱 컨버터블 모델도 나왔지만, 최근에 A5로 분리됐다. 왜건형은
<del>당연히</del> 국내에 안 들어온다.

## 2. 특징 ¶

원래는 C세그먼트에서 그저그런 모델이여서 한 세대 이전인 B7까지는 [BMW3시리즈](BMW%203%EC%8B%9C%EB%A6%AC%EC%A6%88.md)에 밀려 존재감이 약했으나, 2007년에 풀 모델
체인지된 B8형이 **초대박**을 치면서`[1]` D세그먼트의 영원한 강자였던 [BMW](BMW.md) 3시리즈보다도 많이 팔렸다.
특유의 LED 램프를 비롯한 아우디만의 멋진 디자인과 더불어 체급 대비 상당한 크기,
[아우디](%EC%95%84%EC%9A%B0%EB%94%94.md)의 특징 중 하나인 세로배치 FF 플랫폼, 후륜기반의 4:6의 비대칭
콰트로 시스템(FF에는 [CVT](CVT.md)) + 2.0리터 엔진 주제에 강력한 211마력 터보의 조합의 승리인듯 하다.

  

특히 B8의 경우 전장이 4.7미터로 거의 중형차에 맞먹는다. 경쟁 모델인 BMW 3시리즈와 20cm 가까이 차이나는 형편이니 이건 반칙
수준. 특히 2.0리터 가솔린 터보 엔진이 달린 A4 2.0T 콰트로가 메인 모델인데, 벤츠 C200이라든가 BMW 320i 등을 폭풍 관광
시켰다. 이후 3시리즈에 대비하기 위해 2.0리터 TDI도 나왔다. 디젤도 콰트로가 있으며, FF의 경우 디젤 엔진을 장착한 주제에
[CVT](CVT.md)의 일종인 멀티트로닉을 달고 나온다. 물론 지금은 BMW 3시리즈도 모델 체인지가 되었기 때문에 A4가 이긴다고
속단할 수는 없다.  

B8형이 나오고 2012년, 한 번의 페이스리프트를 거친 후
[대한민국](%EB%8C%80%ED%95%9C%EB%AF%BC%EA%B5%AD.md)에도 [BMW3시리즈](BMW%203%EC%8B%9C%EB%A6%AC%EC%A6%88.md)를 겨냥해 TDI가 출시됐다.

  

A4의 플랫폼은 SUV인 아우디 Q5와 [포르쉐마칸](%ED%8F%AC%EB%A5%B4%EC%89%90%20%EB%A7%88%EC%B9%B8.md)이 공용하고 있다. 세로배치
전륜구동 방식의 플랫폼이기 때문에 이 플랫폼은 아우디에서만 사용하는 플랫폼이다.

  

고성능 버전으로 일반형인 S4와 고마력의 RS4가 있는데 S4는 B7까지 V8 엔진을 썼으나, B8부터는 A6에도 달린 적이 있는 333마력
V6 3.0리터 슈퍼차저 엔진으로 다운사이징했다. <del>B8형부터는 RS4를 더 이상 생산하지 않는다고 한다.</del> B8형 RS4는
RS5에 달리는 4.2L V8 엔진을 달아 450마력을 낸다.

## 3. 라인업 ¶

  * 휘발유  

    * **A4 왜건 1.8** : 1.8L 직렬 4기통 엔진 / 170마력 / 전륜구동(FF)
    * **A4 2.0 Quattro** : 2.0L 직렬 4기통 터보 엔진 / 220마력 / 풀타임 4륜구동(AWD)
    * **S4 3.0 TFSI Quattro** : 3.0L V6 TFSI 엔진 / 333마력 / 풀타임 4륜구동(AWD) / A4의 고성능 모델
    * **RS4 왜건 4.2 Quattro** : 4.2L V8 엔진 / 450마력 / 풀타임 4륜구동(AWD) / A4의 초고성능 모델 / 현재 단종.  

  * 경유  

    * **A4 2.0 TDI** : 2.0L 직렬 4기통 TDI 엔진 / 143마력 / 전륜구동(FF) 
    * **A4 2.0 TDI Quattro** : 2.0L 직렬 4기통 TDI 엔진 / 177마력 / 풀타임 4륜구동(AWD)
