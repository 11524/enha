HP Proliant Server (휴렛팩커드 프로라이언트 서버)

[HP](HP.md)에서 제작한 서버 브랜드 명칭, DL, ML, BL 시리즈로 모델이 나누어진다.  
시리즈  
DL : 랙 타입 서버 (일반적으로 사용하는 표준 19인치 규격 랙마운트 서버)  
ML : 타워 타입 서버 (일부 모델 ML150, 350, 370, 570.. 등은 랙타입으로 변경이 가능한 컨버젼 킷이 존재한다.)  
BL : 블레이드 타입 서버  

제너레이션  
현재 8세대 장비까지 나와 있다. (컴팩시절 장비는 제외)  
G는 세대를 뜻하며, 뒤의 숫자는 몇번째 제품을 의미함.  
  
G1 : 베이지 색을 주로 사용하며, 전면부가 철로 되어있다. (G2 까지 일부는 컴팩이 제작함)  
G2 : 베이지 외의 쥐색도 추가되었다! (플라스틱 역시 사용되었다..),([iLO](iLO.md)를 지원하는 세대)  
G3 : 컴팩이 서버쪽에서 손을 떼기 시작하였다. (이 때 부터 쥐색계열로 쭉 가는듯..)  
G4 : 랙 고정쇠가 보강되었다.. (나머지는 여태까지 그래왔듯이 디자인은 그대로..)  
G5 : 페이스오프!!!!!!!!!!!!!!!!!!!! (우선 추가된 순서로 DL360, 380 에는 상태등이 들어간다, 771소켓의 지원,
SCSI를 버  

렸는지 [SAS](SAS/%EB%8F%99%EC%9D%8C%EC%9D%B4%EC%9D%98%EC%96%B4#s-7.md),[SATA](
/wiki/SATA)를 사용하며, 여태까지 그래왔듯 모친출타하신 트레이 값은 덤..

G6 : 풀체인지!!!!!!!!!!!!!!!!!!!!!! (이 때부터 검은색을 사용한다, 1366소켓으로 업그레이드, 가격이 더 싸졌다..)  
G7 : 6세대와 다를 것이 없어 패쓰. 다만, 일부 버튼의 위치가 사라지거나 달라졌다.  
Gen8 : 페이스 오프!!! (페이스오프는 2세대에 걸쳐하는지 이제는 별것도 아니다.. 트레이가 달라지고, 2011소켓을 사용  

이 놈들은 델을 따라하는지 이 때부터 전면커버를 옵션으로 판매한다.)  

마이크로 서버 : 쪼끄만한 가정용 서버, Gen8이 나오면서 iLO가 지원 된다.  
문샷 서버 : 서버안에 PCI카드 처럼 생긴 서버 카트리지가 꽂히면서..[자세한 설명은 생략한다](%EC%9E%90%EC%84%B8%ED%95%9C%20%EC%84%A4%EB%AA%85%EC%9D%80%20%EC%83%9D%EB%9E%B5%ED%95%9C%EB%8B%A4.md)
