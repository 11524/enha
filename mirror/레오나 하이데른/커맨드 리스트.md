# 14. [KOF MI2](KOF%20MI2.md),[KOF MIA](KOF%20MIA.md) ¶

  * 오델 버클러 : 근접해서 ← or → + D
  * 하이데른 인페르노 : 공중 근접해서 ← or → + C or D  

★ 특수기  

  * 플립 플롭 : 공중에서 ↓ + B
  * 블링킹 덩크 : 공중에서 ↓ + C
  * 스팅거 : 공중에서 ↓ + D  

★ 필살기  

  * 문 슬래셔 : ↓ 모으고 ↑ + A or C  
┗ 리버스 슬래셔 : 강 문 슬래셔 사용 중 ↓↑ + C

  * 그랜드 세이버 : ← 모으고 → + B or D  
┗ 글라이딩 버스터 : 강 그랜드 세이버 중 → + D

  * X 캘리버 : 공중에서 ↓↙← + A or C
  * 볼텍 런쳐 : ← 모으고 → + A or C  
┗ 디스차지 : 강 볼텍 런쳐 사용 중 ↓↘→ + C

  * 이어링 폭탄 1 : ↓↙← + B or D
  * 이어링 폭탄 2 (하트어택) : ←↓↙ + B or D  
┗ 하트어택 폭파 : 일정 시간 후 폭파 또는 ←↓↙ + B or D

  * 그래비티 스톰(☆☆☆) : ↓↘→↓↘→ + B or D
