## 1. 포켓몬스터의 용어 ¶

종족치 또는 종족값. [포켓몬스터](%ED%8F%AC%EC%BC%93%EB%AA%AC%EC%8A%A4%ED%84%B0.md)의 종류별로
정해지는 각 스테이터스의 성장률을 말한다. 이 종족치에 바탕을 두고 레벨이 오름에 따라 스테이터스가 정해지고, 여기에
[개체치](%EA%B0%9C%EC%B2%B4%EC%B9%98.md), [포켓몬의성격](%ED%8F%AC%EC%BC%93%EB%AA%AC%EC%9D%98%20%EC%84%B1%EA%B2%A9.md)에 의한 보정,
[노력치](%EB%85%B8%EB%A0%A5%EC%B9%98.md) 등이 가산되어 포켓몬의 능력치가 최종적으로 결정된다. 참고로
플레이어들은 각각의 능력치를 **H**it point(체력) ,**A**ttack(공격),**B**lock(방어)
,**C**ontact(특공),**D**efence(특방),**S**peed(스피드)의 머리글자를 따서 흔히들**HABCDS**라는 약어를
쓴다.

  

쓸데없는 곳에 종족치가 많이 배분되어 있거나, 스테이터스 별로 똑같이 배분되어 특색 없이 밋밋하다거나 하는 식으로 차이가 나기 때문에
절대적인 기준이라 할 수는 없지만, 그래도 보통은 종족치가 높을수록 더 강한 포켓몬이라 할 수 있다. 일반적으로 각 수치별 종족치의 합계를
통해 해당 포켓몬의 능력을 가늠하며, 전설계를 비롯하여 특히 종족치가 높은 포켓몬들을 대전에서 제약하는 기준이 되기도 한다.`[1]`

  

참고로 비전설 포켓몬들은 게을킹이라는 예외를 제외하면 총합 스텟 600이 상한선으로 이
[대기만성](%EB%8C%80%EA%B8%B0%EB%A7%8C%EC%84%B1.md)형`[2]` 600족 포켓몬이 매세대마다 한마리씩
나온다는 전통이 있다.`[3]` 본래 특수가 특공/특방으로 분리된 2세대를 제외하면 **포켓몬의 종족치가 변하는 일은 절대 없었지만**,
6세대에선 사상 최초로 **[포켓](%ED%94%BC%EC%B9%B4%EC%B8%84.md)[몬의](%ED%8E%9C%EB%93%9C%EB%9D%BC.md) [종족치](%EB%94%B1%EA%B5%AC%EB%A6%AC.md)
[자체가](%EB%A7%88%EB%A6%B4%EB%A6%AC.md) 변경되는** 일이 벌어졌다.

### 1.4. 포켓몬의 능력치 계산법 ¶

소수점 이하는 나올 때 마다 자른다.  
레벨 50 기준으로 많이 필요하기 때문에 보통 아래 쪽의 계산식이 더 유용하다.

  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%A2%85%EC%A1%B1%EC%B9%98?action=
edit&section=6)]

#### 1.4.1. HP ¶

능력치 = `[ { (종족값 x 2) + 개체값 + (노력치/4) } x LV/100]` \+ 10 + LV `[8]`

  

레벨 50 개체 V 기준: <종족값 + 75.5 + 노력치/8`[9]`> 계산 후 소수점 이하 버림.

  
  

[[edit](http://rigvedawiki.net/r1/wiki.php/%EC%A2%85%EC%A1%B1%EC%B9%98?action=
edit&section=7)]

#### 1.4.2. 공격,방어,특수공격,특수방어,스피드 ¶

능력치 = `[ { (종족값 x 2) + 개체값 + (노력치/4) } x LV/100 + 5]` x 성격보정

  

레벨 50 개체 V 기준: <종족값 + 20.5 + 노력치/8`[10]`> 계산 후 소수점 이하 버림. 이후 성격보정(1.1배, 1배,
0.9배)를 곱해주고 다시 소수점 이하 버림.
