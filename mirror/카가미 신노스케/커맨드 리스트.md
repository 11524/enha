## 1 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) ¶

※ 기술명 옆의 ●표는 力모드(POWER)에서의 승화 적용기를 뜻합니다.

  

(각성 전)  
★ 기본잡기  

  * 초완 ・ 폭염 : 근접해서 ← or → + B  

★ 특수기  

  * 초권 ・ 재염 : 상대 다운시 ↘ + B 
  * 초각 ・ 분여 : 공중에서 ↓ + C
  * 각성 : 기력 MAX or 체력 게이지 점멸시 BCD`[1]`  

★ 필살기  

  * 시라누이 : ↓↘→ + A or B
  * 초포후(●) : →↓↘ + A or B
  * 염포후 : 근접해서 →↘↓↙← + C  

★ 초오의  

  * 홍련주작 : 공중에서 ←↙↓↘→ + AB  

★ 잠재오의  

  * 분서갱유 : ↓↙←→ + B  

(각성 후)  
★ 기본잡기  

  * 초완 ・ 폭염 : 근접해서 ← or → + B  

★ 특수기  

  * 초권 ・ 재염 : 상대 다운시 ↘ + B 
  * 초각 ・ 강염 : 공중에서 ↓ + C  

★ 필살기  

  * 지옥의 겁화 : ↓↘→ + A or B
  * 황천으로의 날개(●) : →↓↘ + A or B
  * 어리석은 자의 탄식 : 근접해서 →↘↓↙← + C
  * 명부로의 인도 : ↓↙← + A or B
  * 기만의 거울 : ←↙↓↘→ + C  

★ 초오의  

  * 패배의 심판 : 공중에서 ←↙↓↘→ + AB
  * 패배의 세례 : →←↙↓↘→ + AB ([가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))  

★ 잠재오의  

  * 검푸른 맹금 : ↓↙←→ + B

## 2 [월화의 검사](%EC%9B%94%ED%99%94%EC%9D%98%20%EA%B2%80%EC%82%AC.md) 2 ¶

※ 기술명 옆의 ●표는 力모드(POWER) 및 極모드(EX)에서의 승화 적용기를 뜻합니다.

  

★ 기본잡기  

  * 초완 ・ 폭염 : 근접해서 ← or → + CD  

★ 특수기  

  * 초권 ・ 재염 : 상대 다운시 ↘ + B 
  * 강염습 : 공중에서 ↓ + C
  * 조용한 고동 : ↓↙← + C (지속 가능)  

★ 필살기  

  * 비연익 : ↓↘→ + A
  * 겁염조 : ↓↘→ + B  
┗ 시라누이 : 겁염조 히트시 ↓↘→ + B

  * 초포후(●) : →↓↘ + A or B
  * 염포후 : 근접해서 →↘↓↙← + B  

★ 초오의  

  * 홍련주작 : 공중에서 ←↙↓↘→ + AB
  * 성화요원 : →←↙↓↘→ + AB ([가드 불능](%EA%B0%80%EB%93%9C%20%EB%B6%88%EB%8A%A5.md))  

★ 잠재오의  

  * 봉황승천 : 공중에서 ←↙↓↘→ + B  

★ 난무오의  

  * 도 ・ 남붕익 : ↓↓ + AB  
┣ 상단 루트 : 히트 후 A, B, C, A, B, C, A, B, ↓↘→ + C  
┣ 하단 루트 : 히트 후 A, B, C, A, B, C, C, B, ↓↘→ + A  
┣ 띄우기 루트 : 히트 후 A, B, C, ↓ + C, A, ↓ + C, → + BC  
┗ 특수 루트 : 히트 후 A, B, C, ↓ + C, A, BC, A, ↓↘→ + B
